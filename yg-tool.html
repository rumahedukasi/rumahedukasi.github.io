24. Template apa sajakah yg ada pada tool new slide?


Beberapa template yang ada pada tool new slide di Powerpoint adalah

- Title Slide. Digunakan untuk menampilkanjudul dan sub judul dari persentasi

- Title and Content. Digunakan untuk menampilkan judul beserta isi dari konten. Isi konten dapat berupa table, chart, smartart graphic, picture, clip art, media clip.

- Section Header. Digunakan untuk menampilkan judul header dari tiap bagian persentasi.

- Two Content. Digunakan untuk menampilkan 2 isi konten yang berbeda tetapi masih memiliki kemiripan.

- Comparison. Digunakan untuk menampilkan 2 isi konten yang berbeda untuk perbandingan.

- Title Only. Digunakan untuk menampilkan judul.

- Blank. Digunakan untuk menampilkan slide kosong.

- Content with Caption. Digunakan untuk menampilkan isi konten beserta keterangannya.

- Picture with Caption. Digunakan untuk menampilkan gambar beserta keterangannya.

26. Yg merupakan atribut Magic Wand Tool adalah?


Magic wand tool merupakan salah satu tool yang ada pada program edit gambar seperti Adobe Photoshop atau Adobe Illustrator. Magic wand tool adalah tool yang digunakan untuk menyeleksi piksel berdasarkan warnanya. Cara menggunakan magic wand tool sangat mudah. Kita hanya perlu klik ikon magic wand tool kemudian klik di area yang ingin diseleksi.

Pembahasan

Ikon artibut magic wand tool dapat dilihat seperti pada gambar di lampiran berikut ini. Pada tiap versi Photoshop mungkin bentuknya ada sedikit perubahan-perubahan, namun inti gambarnya tetap sama. Magic wand tool dilambangkan dengan bentuk tongkat sihir yang diujungnya terdapat bintang atau sinar. Untuk lebih jelasnya, berikut ini saya uraikan contoh penggunaan magic wand tool dan langkah-langkah teknisnya:

Buka Adobe Photoshop.Masukkan gambar yang akan diedit.Klik ikon magic wand tool yang ada pada toolbox.Klik pada bagian gambar yang memiliki warna sama.Bagian yang memiliki warna yang sama tersebut akan terseleksi dan siap diedit sesuai keinginan kita.

Pada magic wand tool terdapat pilihan tolerance. Pengaturan angka tolerance akan menunjukkan toleransi perbedaan warna yang hendak diseleksi. Semakin besar angka tolerance maka semakin besar pula kemungkinan perbedaan warna yang ikut terseleksi. Begitu pula sebaliknya semakin kecil angka tolerance maka semakin kecil pula kemungkinan warna yang berbeda akan terseleksi.

36. tool yang bersifat sebagai penetral perintah adalah..a.zoom toolb.knife toolc.pick tool d.freehand toole.rectangle tool


Tool yang bersifat sebagai penetral perintah adalah pick tool.


Pembahasan

Toolbox adalah kumpulan dari tool - tool atau perintah yang ada pada coreldraw. Pada setiap Toolbox terdapat tanda segiempat berlipat berwarna hitam yang menandakan ada tool lain yg tersembunyi. Tool yg tersembunyi ini dikenal sebagai “Flyout”.


Berikut ini beberapa flyout yang ada di coreldraw:

1. Pick Tool berfungsi untuk memilih objek dan mengatur objek.

2. Flyout Shape Tool berfungsi untuk memodifikasi objek yang telah dibuat, terdiri atas:

Shape : Mengubah bentuk garis, objek.Knife : Memotong objek.Eraser : Menghapus objek.Smudge Brush : Menghaluskan garis.Roughen Brush : Membuat garis kasar.Free Transform : Memutar objek dengan bebas.

3. Flyout Zoom berfungsi untuk mengadakan perubahan tampilan gambar dilayar, terdiri atas:

Zoom : Memperbesar/memperkecil tampilan.Hand : Menggerakkan objek dengan mendorong.

4. Flyout Freehand, terdiri atas:

Freehand : Menggambar garis bebas.Bezier : Membuat garis dengan titik-titik hubung.Artistic Media : Membuat garis artistik.Pen : Membuat garis lengkung, kurva dengan hubungan titik-titik.Polyline : Menggambar objek pola tertentu.3 Point Curve : Menggambar kurva.Interactive Connector : Menggambar garis hubung.Dimension : Membuat garis keterangan lengkap dengan keterangan panjang garis.

5. Flyout Rectangle digunakan untuk membuat gambar berupa kotak, terdiri atas:

Rectangle :Menggambar persegi panjang.3 Point Rectangle : Menggambar persegi panjang dengan 3 titik.

39. Jelaskan perbedaan Pick Tool, Shape tool, Freehand Tool, Zoom Tool dan Text Tool !


Perbedaan tersebut dapat kita lihat melalui fungsinya.

1. Pick Tool
Pick Tool ini memiliki ikon seperti cursor mouse pada setting defaultnya. Berfungsi untuk menyeleksi atau memilih objek.

2. Shape Tool
Shape Tool ini berfungsi untuk mengedit garis atau objek dengan manipulasi titik.

3. Freehand Tool
Freehand Tool berfungsi untuk menggambar garis kurva atau garis lengkung dan juga garis lurus.

4. Zoom Tool
Zoom Tool ini berfungsi untuk mengubah tingkat pembesaran window document Corel yang sedang kita buka. Kita bisa mengedit objek yang terlalu besar atau objek yang terlalu kecil sehingga nantinya tammpilan tersebut akan sesuai di mata kita.

5. Text Tool
Text Tool ini berfungsi untuk menambahkan tulisan pada canvas Corel kita. Kita juga bisa mengatur jenis font, besar font, dan juga memperbesar atau memperkecil tulisan yang ada.
