10. 3 Bilangan oksidasi atom S yang paling tinggi terdapat pada A. SO₂ B. S₂02 C. S406 D. SO E. H₂S​


Jawaban:

D

Penjelasan:

Bilangan oksidasi S tertinggi pada pilihan jawaban terdapat pada senyawa SO₄⁻² ( pilihan jawbaan D) . Bilangan oksidasi S pada senyawa adalah + 6. Bilangan oksidasi S terendah pada pilihan jawaban terdapat pada senyawa H₂S. Bilangan oksidasi S pada senyawa adalah -2.

Pembahasan

∴ Bilangan oksidasi S pada SO₂ adalah +4.

Diperoleh dari atom yang sudah diketahui bilangan oksidasinya pada SO₂ untuk memperoleh bilangan oksidasi S

∵Bilangan oksidasi total SO₂ adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

SO₂ = 0

(jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = 0

(1 x biloks S) + (2x (-2)) = 0

Biloks S - 4 = 0

Biloks S= 0+4

Biloks S = +4

∴ Bilangan oksidasi S pada S₂O₃⁻² adalah +2.

Diperoleh dari atom yang sudah diketahui bilangan oksidasinya pada S₂O₃⁻² untuk memperoleh biloks S

∵Biloks total S₂O₃⁻² adalah -2 ( karena bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

S₂O₃⁻²= -2

(jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = -2

(2 x biloks S) + (3x (-2)) = -2

2 Biloks S - 6 = -2

2 Biloks S= 6-2

2 Biloks S = +4

Biloks S = \frac{4}{2}

2

4

Biloks S = +2

∴ Bilangan oksidasi S pada S₄O₆⁻² adalah +2.5.

Diperoleh dari atom yang sudah diketahui bilangan oksidasinya pada S₄O₆⁻² untuk memperoleh biloks S

∵Biloks total S₄O₆⁻² adalah -2 ( karena bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

S₄O₆⁻²= -2

(jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = -2

(4 x biloks S) + (6x (-2)) = -2

4 Biloks S - 12 = -2

4 Biloks S= 12-2

4 Biloks S = +10

Biloks S = \frac{10}{4}

4

10

Biloks S = +2.5

∴ Bilangan oksidasi S pada SO₄⁻² adalah +6.

Diperoleh dari atom yang sudah diketahui bilangan oksidasinya pada SO₄⁻² untuk memperoleh biloks S

∵Biloks total SO₄⁻² adalah -2 ( karena bilang oksidasi suatu ion poliatomik adalah sesuai dengan jumlah muatannya ) dan bilangan oksidasi O adalah -2 ( karena bilangan oksidasi untuk Oksigen ( O) pada umumnya adalah -2 )

SO₄⁻²= -2

(jumlah atom S x biloks atom S) + (jumlah atom O x biloks atom O) = -2

(1 x biloks S) + (4x (-2)) = -2

Biloks S - 8 = -2

Biloks S= 8-2

Biloks S = +6

∴ Bilangan oksidasi S pada H₂S adalah -2.

Diperoleh dari atom yang sudah diketahui bilangan oksidasinya pada H₂S untuk memperoleh biloks S

∵Biloks total H₂S adalah 0 ( karena bilangan oksidasi total untuk senyawa diatomik atau poliatom yang netral adalah (0) ), dan biloks H adalah +1 ( karena Bilangan oksidasi untuk Hidrogen ( H) adalah +1 )

Biloks H₂S = 0

(jumlah atom S x biloks atom S) + (jumlah atom H x biloks atom H) = 0

(1 x biloks S) + ( 2 x 1) = 0

Biloks S + 2 = 0

Biloks S = -2

18. Quiz20/166............​


Jawaban:

sistem organ pernapasan

Organ :

-hidung: tempat masuk dan keluarnya udara

-laring: tempat dimana suara dihasilkan

-trakea: tempat dimana udara mengalir ke paru paru

-paru paru: tempat pertukaran o2 dan co2

semoga membantu

Jawaban:

Pendahuluan

semua makhluk hidup perlu bernapas agar dapat bertahan hidup. bernapas merupakan proses memasukkan oksigen ke tubuh serta mengeluarkan karbondioksida dan uap air dari tubuh.

mekanisme pernapasan dapat berlangsung dengan baik karena manusia dianugerahi sistem pernapasan. sistem pernapasan manusia meliputi organ-organ pernapasan dan mekanisme kerja organ-organ pernapasan tersebut. organ-organ apa saja yang berperan dalam mekanisme pernapasan tersebut. Kita akan membahasnya

Pembahasan

a. Rongga Hidung

• Untuk menghidup udara

• selaput lendir untuk melembabkan udara pernapasan agar sesuai dengan kondisi suhu tubuh

• rambut-rambut di dalam rongga nya yang berfungsi untuk menyaring udara pernapasan

b. Faring

• faring tersusun atas otot lurik yang dilapisi membran mukosa

• faring berfungsi sebagai saluran udara dan saluran makanan.

c. Laring

• terdapat katup epiglotis yang berfungsi mengatur jalannya makanan dan udara agar dapat melewati saluran nya masing-masing

• terdapat pita suara sebagai penghasil suara pada manusia

d. trakea

• berfungsi menghubungkan laring dengan bronkus

• dindingnya berfungsi mengeluarkan kotoran yang masuk bersama udara pernapasan

e. bronkus

• membawa udara ke paru paru

f. Paru paru

• menerima udara dari bronkus

g. alveolus

• tempat pertukaran antara gas oksigen dan karbondioksida pada proses pernafasan

Kesimpulan

Maka itulah organ penyusun utama sistem pernapasan pada manusia
