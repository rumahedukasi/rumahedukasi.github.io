10. Energi ionisasi atom hidrogen dari lintasan ketiga adalah . . . . a. 1,51 eV b. 4,53 eV c. 10,6 eV d. 16,3 eV e. 40,8 eV


TEORI ATOM

• tingkat energi

n = 3

Eᵢ = __?

Eₙ = -13,6 / n²

untuk n = 3

E₃ = -13,6 / 3²

E₃ = - 1,51 eV

untuk n = ∞

E∞ = -13,6 / ∞

E∞ = 0

Energi untuk mengioniasi elektron dari n = 3 ke n = ∞ sebesar

Eᵢ = E∞ - E₃

Eᵢ = 0 - (1,51)

Eᵢ = 1,51 eV ✔️
