2. pliiis jawab dong pentiing


x² - 5x - 2 = 0

a = 1
b= -5
c = -2

p + q = -b/a = -(-5)/1 = 5
p.q = c/a = -2/1 = -2

persamaan kuadrat baru yg akar²nya 3p dan 3q adalah
(x - 3p)(x - 3q) = 0

x² - 3px - 3qx + 9pq = 0

x² - (3p + 3q)x + 9.(-2) = 0

x² - (3 (p+q)) x - 18 = 0

x² - (3(5))x - 18 = 0

x² - 15x - 18 = 0

************************************************
Mapel : Matematika
Kelas : 10
Materi : Persamaan dan Fungsi Kuadrat
Kata kunci : Persamaan kuadrat baru
Kode Kategorisasi : 10.2.5Matematika
Bab: Persamaan Kuadrat
Kelas: X SMA
Semester 1 KTSP 2006
===============
Diketahui:
a = 1
b = -5
c = -2

Untuk menyusun persamaan kuadrat baru apabila diketahui faktor faktor nya, bisa menggunakan 2 cara, yakni:
1. (x-x1) (x-x2) = 0
2. x2 - (x1+x2)x + (x1.x2) = 0

Sifat akar:
x1 + x2 = -b/a
x1 + x2 = -(-5)/1
x1 + x2 = 5

x1.x2 = c/a
x1.x2 = -2/1
x1.x2 = -2

Sekarang, masukkan sifat akar tersebut.
(x-3p) (x-3q)
= x2 -3px - 3qx + 9pq
= x2 - 3p+q(x) + 9 (pq)
= x2 - 3(5) x + 9(-2)
= x2 - 15x - 18 persamaan kuadrat barunya.
