Storage: 64GB

Firmware: UEFI

23. Cara mematikan windows update windows 10 permanen adalah .....


Jawaban:

Cara-Cara

1. Buka Control Panel dengan cara mengetikkan “Control Panel” di kolom pencarian di samping Start Menu.

2. Kemudian, pilih System and Security dan tekan menu Administrative Tools.

3. Akan muncul halaman baru yang berisikan berbagam macam fitur dengan fungsi yang berbeda-beda. Di sini, Anda pilih Services saja.

4. Scroll ke bawah sampai Anda menemukan Windows Update, dan klik dua kali. Muncul halaman Windows Update Properties.

5. Sekarang, ganti opsi pada Startup Type menjadi Disabled, dan klik tombol Stop pada Service Status. Kemudian tekan Ok.

25. Bagaimana Cara Mengaktifkan Windows Defender Kembali Pada Windows 8 


Pertama, kepala ke layar Mulai bergaya modern dan ketik "Windows Defender" untuk memiliki Windows mencari program, kemudian klik pada ikon Windows Defender ketika muncul dalam hasil.Sebuah jendela Windows Defender akan muncul di desktop klasik.Jika perangkat lunak keamanan Microsoft dinonaktifkan, Anda akan melihat banyak warna merah menakutkan bersama-sama dengan "Beresiko" peringatan dan gambar dari layar komputer dengan X besar di atasnya.Halus, eh? Selanjutnya, klik pada Pengaturan tab di bagian atas jendela.Pastikan "perlindungan real-time" dipilih di panel kiri, kemudian centang kotak di samping "Hidupkan perlindungan real-time (dianjurkan)."Terakhir, klik Simpan Perubahan di bagian bawah Window. Anda akan tahu itu bekerja ketika menakutkan merah "Beresiko" bar di bagian atas Windows berubah warna lebih menenangkan hijau dan beralih ke "Status PC:. Dilindungi" Periksa kebocoran Kau tidak cukup belum dilakukan.Sekarang saatnya untuk memastikan PC Anda sebenarnya adalah malware-free!Klik Memperbarui tab, kemudian klik pada besar Memperbarui tombol di tengah Jendela untuk men-download definisi malware terbaru Microsoft memiliki file. Selanjutnya, buka Rumah tab dan pilih "Full" tombol radio dalam daftar Pilihan Scan.Yang harus Anda lakukan sekarang adalah klik Scan Now, Kemudian duduk kembali dan menunggu sementara Windows Defender memeriksa sudut dan celah dari PC Anda untuk setiap penjahat tersembunyi.Ambil secangkir kopi, mungkin diperlukan beberapa waktu.Sementara Anda menunggu, kami sarankan Anda memeriksa Windows 8 pilihan antivirus.
