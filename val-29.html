Jawaban & Penjelasan:

Fungsi Val hanya mengubah nilai numerik yang ada dalam string menjadi Double. Tetapi tidak dapat mengubah dari nilai bertipe data numerik. sedangkan cdec CDec untuk mengonversi nilai numerik menjadi desimal

contoh Val :

Dim MyDouble, MyString

MyString = "10000000.0587"

MyDouble = val(MyString)

contoh CDec :  

Dim MyDecimal, MyCurr

MyCurr = 10000000.0587

MyDecimal = CDec(MyCurr)

15. Jelaskan kegunaan dari fungsi berikut!a. Val()b. CStr() c. Ucase()d. StrReverse(string)mohon bantuannya ​


Jawaban:

a. Val(): Fungsi Val() digunakan untuk mengonversi sebuah string menjadi bilangan pecahan (floating-point) atau bilangan bulat (integer) di dalam bahasa pemrograman. Fungsi ini akan mengabaikan karakter-karakter non-numerik dalam string dan mengambil angka-angka yang valid untuk diubah menjadi bilangan.

b. CStr(): Fungsi CStr() digunakan untuk mengonversi sebuah nilai atau ekspresi menjadi tipe data String di dalam bahasa pemrograman. Fungsi ini akan mengubah tipe data apapun menjadi representasi string.

c. Ucase(): Fungsi Ucase() digunakan untuk mengubah semua karakter dalam sebuah string menjadi huruf kapital (uppercase) di dalam bahasa pemrograman. Fungsi ini berguna ketika kita ingin memanipulasi teks dengan memastikan bahwa semua karakternya dalam huruf besar.

d. StrReverse(string): Fungsi StrReverse() digunakan untuk membalikkan urutan karakter dalam sebuah string di dalam bahasa pemrograman. Fungsi ini akan menghasilkan string baru yang merupakan kebalikan dari string asli. Misalnya, jika string awal adalah "Hello", maka fungsi StrReverse() akan menghasilkan string "olleH".

Val(txtpanjang.text) bertujuan untuk a. Membuat nilai yang ada di txtpanjang agar bersifat numerik/angka sehingga bisa dimasukkan dalam perhitungan aritmatika.

Penjelasan:

Fungsi val() adalah fungsi value untuk mengubah tipe data suatu isi variabel menjadi numerik/angka. Tipe numerik dapat dioperasikan menggunakan operator aritmatika, ini berguna untuk mengubah tipe data suatu variabel yang ingin isinya dijumlahkan atau dioperasikan secara aritmatika. Contoh:

panjangTeks.text = "123"

val(panjangTeks.text)

Maka panjangTeks akan bernilai 123 bukan "123" lagi.
