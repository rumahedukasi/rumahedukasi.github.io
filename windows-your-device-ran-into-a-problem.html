1. Buka New Task
2. ketik "msconfig"
3. Masuk ke Tab Services
4. Centang Hide All Microsoft Settings -> lalu Disable All
5. Lalu Ke Startup...(Biarkan Jendela msconfig terbuka) 
6. Open Task manager
7. Lalu pada Task Manager Buka Tab Startup
8. Disable Apps yg Bermasalah 
9. Tutup Taskbar
10. Kembali ke Jendela msconfig
11. Klik OK

26. Dalam windows dan linux bagaimana manajemen Proses, memori,device& berkas​


Penjelasan:

file management linux

Data pada file bisa berupa numerik, alpha numerik binary atau text. Setiap File memiliki nama dan pengacuan terhadap suatu file menggunakan nama file tersebut. Pada Linux penamaan File bersifat case sensitif yaitu membedakan antara lower case dan upper case letters sehingga file-file Tugas, tugas, TUGAS, TUGas merupakan file-file yang berbeda. Sebagai perbandingan, pada MS-DOS, file-file tadi dianggap sama.

Pemberian nama dan extention pada Linux tidak dibatasi jumlah karakternya dan suatu file dapat memiliki lebih dari satu extention. Contohnya: prog.c.z yairu prog.c yang sudah dikompres. Selain nama, file memiliki atribut seperti tipe, size, time, date, dan user identification, protection dll. Tipe dari file dikenal dari extentionnya. Dengan tipe inilah OS dapat merespon file secara tepat.

File biasanya disimpan dalam media disk (floppy disk, harddisk, atau CD).

Operasi-operasi file yang biasa dilakukan antara lain: OPEN, CLOSE, CREATE, DELETE, COPY, RENAME, READ, WRITE, UPDATE, INSERT, APPEND.

Manajemen File di Windows

Dalam sistem operasi windows kelompok file ekstensi memainkan peran penting. Semua file yang ada di sistem operasi windows memiliki beberapa ekstensi. Kebanyakan dari mereka yang akan Anda ingat. Sys,. Dll dll Ini semua sistem file. Mengerti sistem operasi Windows file dengan ekstensi dan menggunakan program yang tepat untuk membuka file bagi pengguna. Misalnya ketika Anda membuka Dukungan Microsoft, Windows membantu memiliki ekstensi tertentu dan ada sebagai sebuah file dan kemudian menggunakan sistem operasi Microsoft, explorer untuk membuka file untuk Anda..

28. On my way to the campus, I ... a friend whom I had not met for me a. ran into b. ran over c. ran down d. ran out


On my way to the campus, I ran into a friend whom I had not met for me.

Bila diartikan: Dalam perjalanan ke kampus, saya bertemu seorang teman yang saya tidak bertemu.

Pembahasan

The meaning of the phrasal verb:

A. Ran into = cost (harga), meet by accident (bertemu dengan tidak sengaja)

B. Ran over = explain quickly (menerangkan dengan cepat), hit with a vehicle (tertabrak kendaraan)

C. Ran down =

Hit a pedestrian with a vehicle (menabrak seorang pejalan kaki dengan kendaraan)Lose energy or power (kehabisan tenaga)Criticise, disparage (mengkritik)Find the source or origin of something (mencari sumber atau asal sesuatu)

D. Ran out = Have none left (habis)
