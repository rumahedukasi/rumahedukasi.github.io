9. Persamaan polinomial x^3-5x^2+8x+n=0 mempunyai dua akar kembar. Jika akar-akar polinomial tersebut adalah bilangan bulat, nilai n = ....


Misalkan x³-5x²+8x+n=0 memiliki akar-akar a, b, dan c

Karena akar kembar, a = b, maka
a + b + c = -(-5)        ⇔ 2a + c = 5        .......(1)
a.b + a.c + b.c = 8    ⇔ a² + 2ac = 8     .......(2)
a.b.c = -(n)               ⇔ a²c = -n           ........(3)

Dari (1) dan (2)
c = 5-2a
a² + 2ac = 8
a² + 2a(5-2a) = 8
a² - 4a² + 10a = 8
-3a² + 10a = 8
-3a² + 10a - 8 = 0
(-3a + 4) (a - 2) = 0
a = 4/3 atau a = 2

Karena akar polinomial bilangan bulat, dipilih a = 2
c = 5 - 2a = 5 - 2(2) = 1
-n = a²c = 2².1 = 4
n = -4

12. tentukan akar akar persamaan polinomial 3x³+2x²_19x+6=0 adalah​


Penjelasan dengan langkah-langkah:

Untuk menentukan akar-akar persamaan polinomial tersebut, kita dapat menggunakan metode faktorisasi atau metode lain seperti metode numerik. Namun, karena persamaan tersebut memiliki derajat 3, maka kita dapat menggunakan rumus Cardano-Tartaglia untuk menentukan akar-akarnya.

Rumus Cardano-Tartaglia untuk persamaan polinomial berderajat 3 adalah:

x = (-b ± √(b² - 4ac + 2a√(b² - 4ac))) / 2a

di mana a, b, dan c adalah koefisien persamaan polinomial.

Menerapkan rumus tersebut pada persamaan 3x³+2x²-19x+6=0, maka kita dapatkan:

a = 3, b = 2, dan c = -19

x = (-2 ± √(2² - 4(3)(-19) + 2(3)√(2² - 4(3)(-19)))) / 2(3)

x = (-2 ± √(4 + 228 + 12√247)) / 6

x = (-2 ± √(4 + 228 + 12√247)) / 6

x = (-2 ± √(4 + 228 + 12√247)) / 6

x1 = (-2 + √(4 + 228 + 12√247)) / 6

x2 = (-2 - √(4 + 228 + 12√247)) / 6

Sehingga akar-akar persamaan polinomial 3x³+2x²-19x+6=0 adalah x1 dan x2 yang telah dinyatakan di atas.

20. Diketahui x1, x2, x3 merupakan akar akar suatu persamaan polinomial yang memenuhi persamaan berikut


Persamaan polinomial berderajat tiga adalah

ax³ + bx² + cx + d = 0

Rumus jumlah dan hasil kali akar-akar polinomial berderajat tiga adalah
x₁ + x₂ + x₃ = [tex]-\frac{b}{a} [/tex]
x₁ . x₂ + x₁ . x₃ + x₂ . x₃ = [tex] \frac{c}{a} [/tex]
x₁ . x₂ . x₃ = [tex]- \frac{d}{a} [/tex]

Mari kita lihat soal tersebut.
Diketahui:
Rumus jumlah dan hasil kali akar-akar polinomial berderajat tiga adalah
x₁ + x₂ + x₃ = 2
x₁ . x₂ + x₁ . x₃ + x₂ . x₃ = -5
x₁ . x₂ . x₃ = -6

Ditanyakan:
Persamaan polinomial dan akar-akarnya

Jawab:
x₁ + x₂ + x₃ = 2
⇔ [tex]-\frac{b}{a} [/tex] = 2
⇔ b = -2a ... (1)

x₁ . x₂ + x₁ . x₃ + x₂ . x₃ = -5
⇔ [tex] \frac{c}{a} [/tex] = -5
⇔ c = -5a ... (2)

x₁ . x₂ . x₃ = -6
⇔ [tex]- \frac{d}{a} [/tex] = -6
⇔ d = 6a ... (3)

Persamaan polinomial
ax³ + bx² + cx + d = 0
⇔ ax³ -2ax² -5ax + 6a = 0
⇔ a(x³ - 2x² - 5x + 6) = 0
⇔ x³ - 2x² - 5x + 6 = 0

Jadi, persamaan polinomialnya adalah x³ - 2x² - 5x + 6 = 0.

Kemudian, 
x³ - 2x² - 5x + 6 = 0
⇔ (x - 3)(x² + x - 2) = 0
⇔ (x - 3)(x + 2)(x - 1) = 0
⇔ x = 3 V x = -2 V x = 1

Jadi, akar-akar persamaan polinomialnya adalah -2, 1, dan 3.
