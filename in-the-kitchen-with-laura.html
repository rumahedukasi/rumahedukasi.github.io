PEMBAHASAN Present Continuous

Digunakan untuk menyatakan Kegiatan yg terjadi sekarang atau sedang berlangsung saat ini.

Rumus Present Continuous(+) S + is/am/are + Verb-ing(-) S + is/am/are + not + Verb-ing(?) is/am/are + S + Verb-ing?

Catatan

Subjek + Tobe

I + amIt/He/She + is ✔️You/we/they + are

~

My mother...rice in the kitchen (cook)​​

Answer (Jawaban):

(Ibuku memasak nasi di dapur.)

Explanation (Penjelasan):

"My mother cooks rice in the kitchen."

=> Subject + verb(s) + ...

SIMPLE PRESENT TENSEUsage (Kegunaan):

Simple present tense digunakan ketika suatu kejadian berlangsung berulang kali (kebiasaan/rutinitas).

Rumus Kalimat Positive/Affirmative (+)

S + Verb (1) / Verb dasar / to be (is, am, are) + O

Rumus Kalimat Negative (-)

S + Don’t / Doesn’t + Verb (1) + O

Jika kata kerja(verb) nya berupa To Be, maka rumusnya adalah...

S + (is, am, are) + not + O

Rumus Kalimat Tanya/Interrogative (?)

Do / Does + S + Verb 1 + O + ?

She, He, It, [singular] on Simple present tense = Verb + s/es
