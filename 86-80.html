29. Rumus suku ke-n dari barisan 98, 92, 86, 80, … adalah


Jawaban:

Un = -6n + 104

Penjelasan:

98, 92, 86, 80...

a = 98

b = n2 - n1 = 92 - 98 = -6

n = n

RUMUS BARISAN ARITMATIKA :

Un = a + (n - 1) × b

maka, rumus barisan tersebut adalah...

Un = 98 + (n - 1) × -6

Un = 98 - 6n + 6

Un = -6n + 104

bukti :

U1 = -6(1) + 104

U1 = -6 + 104 = 98

U2 = -6(2) + 104

U2 = -12 + 104 = 92

U3 = -6(3) + 104

U3 = -18 + 104 = 86

Jadi, rumus suku ke - n pada barisan tersebut adalah Un = -6n + 104
