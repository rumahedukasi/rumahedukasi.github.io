Arti : " Itu Saudaraku, Dia Insinyur yang Rajin (Aktif). "

___________________________________

PembahasanArti Kosakata

ذَلِكَ = Itu ( Muzakkar )

اَخِيْ = Saudaraku

هُوَ = Dia ( Muzakkar )

مُهَنْدِسُ = Insinyur

نَشِيْط = Giat/Rajin/Aktif

-

Kata Tunjuk ( Isim Isyarah )

Isim Isyarah merupakan isim yang menunjukkan pada sesuatu dengan mengisyaratkan kepadanya. Isim Isyarah adalah kata tunjuk, seperti " ini " dan itu ".

.

Isim Isyarah jika dilihat dari jenisnya terbagi menjadi 2, yaitu muzakkar dan mu'annas.

A.) Muzakkar

Muzakkar adalah kata tunjuk untuk subjek yang tidak feminim. Ciri-cirinya adalah di subjeknya tidak terdapat Ta Marbutoh ( ة ) di akhir katanya.

B.) Mu'annas

Mu'annas adalah kata tunjuk untuk subjek yang feminim. Ciri-cirinya adalah di subjeknya terdapat Ta Marbutoh ( ة ) di akhir katanya.

.

Isim Isyarah dilihat dari sisi bilangannya terbagi menjadi 3, yaitu isim isyarah mufrad (tunggal), isim isyarah musanna (menunjukkan 2), dan isim isyarah jama' (Jamak/Plural).

Isim Isyarah Mufrad ( Tunggal )

Dalam kalimat tersebut hanya dikenalkan isim isyarah mufrad (tunggal). Berikut yang tergolong isim isyarah mufrad adalah :

هَذَا = Ini ( muzakkar dekat )ذَالِكَ = Itu ( muzakkar jauh )هَذِهِ = Ini ( mu'annas dekat )تِلْكَ = Itu ( mu'annas jauh )

-

Dhamir

Dhamir adalah kata ganti orang. Contohnya saya, kamu, dia, kami.

Berikut yang tergolong Dhamir :

اَنَا = Sayaاَنْتَ = Kamu ( Muzakkar )اَنْتِ = Kamu ( Mu'annas )هُوَ = Dia ( Muzakkar )هِيَ = Dia ( Mu'annas )نَحْنُ = Kami
