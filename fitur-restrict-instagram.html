3. Perbandingan fitur apa saja yang ada di media sosial Instagram dan Twitter​


Perbandingan fitur yang ada di media sosial Instagram dan Twitter​ yakni Twitter mempunyai fitur retweeting, quoting, dan rantai balasan bertingkat, sedangkan Instagram hanya memiliki rantai balasan tingkat tunggal. Twitter dan Instagram sendiri ialah sebuah platform media sosial yang dirancang untuk berbagi media dan konten. Media sosial Instagram berfokus pada konten media, sementara media sosial Twitter juga memungkinkan posting dan juga jajak pendapat teks.

Pembahasan

Pengguna dapat mengelompokkan pesan hingga 50 orang di media sosial Twitter sedangkan media sosial instagram hingga 15 orang. Instagram ini menawarkan Cerita yang hilang setelah diposting 24 jam. Kedua platform ini memungkinkan iklan serta tautan afiliasi, pada media sosial Instagram juga menyediakan tombol Beli Sekarang. Aplikasi iOS maupun Android berfitur lengkap untuk sosial media Twitter serta Instagram. Situs web Twitter mempunyai hampir semua opsi yang sama dengan aplikasi sedangkan versi desktop Instagram terbatas untuk posting foto.

Media sosial Twitter dan Instagram ini mungkin melayani audiens dan demografi yang serupa tetapi fungsinya sangat berbeda. Instagram sendiri ialah alat yang hebat untuk berbagi foto video dan mendorong konten asli dan membangun merek. Sedangkan Twitter ialah alat yang ideal untuk mendistribusikan konten dan terlibat dengan audiens. Perbedaan utama antara kedua media sosial Instagram dan Twitter yakni terletak pada jenis kontennya.

10. Tuliskan fitur-fitur fitur-fitur yang ada dalam peramban​


Berdasarkan apa yang saya tahu, fitur pada peramban itu bermacam-macam tergantung dengan peramban apa yang kita pakai. Namun secara umum peramban itu pasti ada beberapa fitur seperti:

1. Dockable Tab

Simplenya itu adalah fitur yang memperbolehkan kita untuk membuka dan mengakses banyak web sekaligus dalam waktu yang bersamaan. Contohnya jika pada PC kita bisa melihat susunan tab dari web-web yang kita buka di bagian atasnya.

2. Bookmarks

Seperti namanya "penanda buku", kita bisa menandai dan menyimpan sebuah alamat web di peramban kita. Dan jika suatu saat kita ingin mengakses kembali web tersebut, kita hanya perlu mencarinya di bookmark.

Fitur ini sangat berguna terlebih kepada pengguna yang bahkan tidak ingat dengan nama web yang ingin mereka ingin kunjungi kembali.

3. History

Hampir sama seperti bookmark, namun history akan menyimpan traffic kegiatan pengguna yaitu apapun yang ia lakukan di peramban. Seperti menyimpan informasi apa yang sudah dia search, atau web apa yang sudah ia sudah kunjungi.

4. Search Bar

Tentu ini adalah hal yang paling jelas, semua peramban harus memiliki fitur dimana itu dapat membantu pengguna untuk mencari sesuatu.

22. Jelaskan fitur fitur pada bios​


Jawaban:

Ada 5 fitur pada BIOS ini, yaitu Information, Main, Security, Boot, dan Exit

Penjelasan:

1. Information

Dari fitur Information ini, kita dapat mengetahui beberapa komponen yang terpasang pada PC ini, disitu tertulis beberapa komponen, yaitu CPU (Processor), Harddisk, VGA, dan jenis PC nya.

2. Main

Dan dari fitur Main ini, kita bisa mengetahui waktu berdasarkan system di PC kita, dan kita juga bisa mengetahui jumlah kapasitas RAM dan juga VRAM di PC kita.

Disitu juga tertulis Quiet Boot, Network Boot, F12 Boot Menu, D2D Recovey, SATA Mode

Quiet Boot berfungsi untuk menentukan apakah pada saat PC kita booting, akan mengeluarkan suara atau tidak.

Network Boot berfungsi untuk menentukan apakah kita akan booting melalui internet, atau tidak.

F12 Boot Menu pada fitur ini, kita dapat menentukan Boot Priority Order (urutan boot) tanpa harus masuk ke sistem BIOS.

D2D Recovery fitur ini dapat mempermudah kita untuk melakukan Recovery OS tanpa perlu install ulang.

SATA Mode di fitur ini, kita dapat menentukan mode Harddisk, ACHI atau IDE.

3. Securtity

Pada fitur Security ini, kita dapat membuat Supervisor Password, User Password, atau Harddisk Password.

4. Boot

Pada fitur Boot ini, kita dapat menentukan urutan Boot melalui Hardware, misal kita ingin booting melalui CDROM, jadi kita letakkan CD-ROM ke Priority pertama.

5. Exit

Pada fitur Exit ini, kita dapat menentukan apakah kita akan keluar BIOS tanpa/dengan melakukan penyimpanan konfigurasi, Membaca (Load) konfigurasi yang pernah kita simpan.

39. fitur fitur yang ada di windows nt



FITUR FITUR YANG ADA PADA WINDOWS NT

 Fitur fitur yang ada diantaranya adalah :

- Dukungan penuh untuk prosesor 32-bit 

Windows NT adalah sistem Opersi baru yang tidak dibangun dari MS-DOS (tidak seperti Windows 95). Semua fitur yang di tawarkan oleh mikroprosesor Intel 386DX ke atas (prosesor 32-bit) dapat digunakan oleh Windows NT.

Dukungan aplikasiWindows NT merupakan sistem opersi 32-bit murni,dan aplikasi 32-bit murni jarang ditemukan di tahun pada saat diluncurkannya Windows NT. Untuk aplikasi, Windows NT bisa mendukung aplikasi dari sistem operasi terdahulu, seperti aplikasi MC_DOS (kecuali Game MC_DOS), Windows 16-bit,dan OS/2.

Dukungan sistem berkasWindows NT mendukung beberapa sistem berkas (file system) yang saat itu banyak digunakan, seperti FAT16 (sistem berkas milik MS-DOS), HPFS (sistem berkas milik IBM OS/2), dan NTFS (sistem berkas milik Windows NT sendiri).

 Desain full GUIWindows NT didesain dengan menggunakan GUI (Graphical User Interface atau antarmuka grafis) secara penuh. Hal ini dilakukan dengan mengintegrasikan komponen grafis (GDI, Graphics Device Interface) secara integral ke dalam komponen sistem operasi.
