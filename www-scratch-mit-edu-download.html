Pengertian WWW :

Www atau World Wide Web maupun Web saja adalah sebuah sistem yang saling terkait dalam sebuah dokumen berformat hypertext yang berisi beragam informasi, baik tulisan, gambar, suara, video, dan informasi multimedia lainnya dan dapat diakses melalui sebuah perangkat yang disebut web browser

Pengertian Hypertext Transfer Protocol (HTTP) :

adalah sebuah protokol jaringan lapisan aplikasi yang digunakan untuk sistem informasi terdistribusi, kolaboratif, dan menggunakan hipermedia. Penggunaannya banyak pada pengambilan sumber daya yang saling terhubung dengan tautan, yang disebut dengan dokumen hiperteks

HTML atau HyperText Markup Language adalah program mark-up yang digunakan untuk membuat sebuah halaman web, menampilkan berbagai informasi di dalam sebuah Penjelajahan web Internet dan formating hypertext sederhana yang ditulis kedalam berkas format ASCII agar dapat menghasilkan tampilan wujud yang terintegerasi.

Pengertian URL :

(Uniform Resource Locator) adalah rangkaian karakter menurut suatu format standar tertentu, yang digunakan untuk menunjukkan alamat suatu sumber seperti dokumen dan gambar di Internet

Tombol-tombol Motion:

Move ( ) steps -> Jika memasukkan sebuah angka kedalam kurung, sprite akan bergerak maju sesuai angka ke arah yang ditetapkan.

Turn left ( ) degrees -> Jika memasukkan sebuah angka kedalam kurung, sprite akan berputar melawan jarum jam sebanyak derajat yang ditetapkan.

Go to (⬇︎) - > Setelah memilih sebuah sprite dari kurung, sprite yang diberikan kode ini akan berada di posisi sprite yang dipilih.

Go to x:( ) y:( ) -> Setelah memberikan angka pada 2 kurung dalam kode, sprite akan berada di posisi x dan y yang diberikan.

Glide ( ) secs to (⬇︎) - > Setelah memberikan angka dalam kurung dan memilih sprite dalam kurung, sprite yang diberikan kode akan pergi ke posisi sprite yang dipilih selama detik yang diberikan.

Glide ( ) secs to x:( ) y:( ) -> Setelah memberikan angka dalam 3 kurung, sprite akan bergerak selama detik yang diberikan ke posisi x dan y yang ditetapkan.

Point in direction ( ) -> Sprite akan mengarah ke arah yang ditentukan.

Point towards (⬇︎) -> Sprite akan mengarah ke sprite yang ditentukan.

Change x by:( ) -> Posisi x sprite akan berubah sebanyak yang diberikan dalam kurung.

Set x to: ( ) -> Posisi x sprite akan diatur ke posisi yang diberikan dalam kurung.

Change y by:( ) -> Posisi y sprite akan berubah sebanyak yang diberikan dalam kurung.

Set y by:( ) -> Posisi y sprite akan diatur ke posisi yang diberikan dalam kurung.

If on edge, bounce -> Jika sprite menyentuh bagian ujung layar, sprite akan menjauh ke posisi lain secara acak.

Set rotation style (⬇︎) -> Style rotasi sprite akan ditetapkan setelah memilih style rotasi seperti:

All around: Sprite dapat mengarah 360 derajat.

Left-right: Sprite dapat mengarah hanya kiri dan kanan.

Don't rotate: Sprite hanya mengarah 90 derajat.

Tombol-tombol Control:

Wait ( ) seconds -> Sprite akan menunggu selama detik yang ditentukan sebelum melakukan kode selanjutnya.

Repeat ( ) -> Sprite akan mengulangi kode yang dimasukkan ke blok ini sampai jumlah waktu yang ditentukan.

Forever -> Kode sprite yang dimasukkan ke dalam blok ini akan diulangi berkali-kali/tanpa batas.

If < > then -> Kode yang dimasukkan dalam blok ini akan berjalan setelah kode Sensing/Operator dimasukkan dalam tempat di antara tulisan If dan then berjalan.

All: Kode semua sprite dalam projek yang diberikan kode ini akan berhenti.

Keliling

___________________

p = 48 m

l = 1/2p = 1/2 × 48 = 24 m

maka, keliling pekarangan:

K = 2(p + l)

= 2(48 + 24)

= 2(72)

= 144 m
