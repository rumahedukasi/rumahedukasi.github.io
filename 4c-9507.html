di daerah lereng gunung.suhu udara dapat mencapai suhu 4⁰C di bawah 0⁰C bilangan bulat yang menunjukan suhu itu adalah ?

Pembahasan

Suhu 4⁰C dibawah 0⁰C

karena dibawah 0 maka lambangnya adalah bilangan negatif sedangkan apabila di atas 0 maka adalah bilangan positif.

Lambang bilangan negatif = ( - )

Lambang bilangan positif = ( + ) ➱ Biasanya tidak dilambangkan.

➱ Maka lambang 4⁰C dibawah 0⁰C adalah =

- + 4 = - 4⁰C
