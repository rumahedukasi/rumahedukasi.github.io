Jawaban, D

Maksimumkan,
=> f(x,y) = 3x+5y
Dengan kendala,
=> x+y ≤ 50 ;
=> -x + 2y ≤ 40;
=> 0 ≤ x ≤ 30;
=> y ≥ 0

Kita gunakan gunakan metode titik pojok, dengan cara menggambar kan daerah yang memenuhi sistem pertidaksamaan. Kemudian mencari titik-titik pojok dari daerah solusi tersebut.

Titik potong antara x + y = 50 dan 2y = x+40,
2y - 40 + y = 50
3y = 90
y = 30
x = 50 - 30 = 20
Titiknya (20, 30)

Dari gambar, diperoleh titik pojok daerah solusi yaitu A(0, 0) B(0, 20), C(20, 30), D(30, 20), E(30, 0)
(lihat di gambar)

Uji semua titik pojok tersebut ke fungsi objektifnya,

A(0, 0)
=> f(0, 0) = 3(0) + 5(0) = 0

B(0, 20)
=> f(0, 0) = 3(0) + 5(20) = 100

C(20, 30)
=> f(0, 0) = 3(20) + 5(30) = 210

D(30, 20)
=> f(30, 20) = 3(30) + 5(20) = 190

E(30, 0)
=> f(30, 0) = 3(30) + 5(0) = 90

Nilai terbesar yaitu 210 ketika x = 20, y = 30

Akar akar persamaan kuadrat dari [tex]\rm x^{2}-20x+100=0 [/tex] adalah [tex]\bf =10 [/tex] (Opsi E)

PENDAHULUAN

Fungsi Kuadrat adalah suatu fungsi dimana pangkat variabel tertingginya adalah dua. Bentuk umum fungsi kuadrat adalah

[tex]\begin{gathered}\left\{\begin{matrix} \rm f(x)=ax^{2}+bx+c \\\\~\rm y=ax^{2} + bx + c \end{matrix}\right.\end{gathered} [/tex]

Dengan syarat a ≠ 0 dan a,b,c elemen Real

Menentukan akar akar persamaan kuadrat

Rumus ABC, yaitu jika ax² + bx + c = 0 maka dapat dicari x1 dan x2 dengan

[tex]\rm x_{1,2}=\dfrac{ -b\pm \sqrt{ b^{2}-4ac } }{2a } [/tex]

Pemfaktoran, yaitu jika ax² + bx + c = 0 maka dapat difaktorkan ke (x - x1)(x - x2) = 0

[tex]\rm x_{1}+x_{2}=-\dfrac{b }{ a } [/tex]

[tex]\rm x_{1}-x_{2}=\pm \dfrac{\sqrt{ D } }{ a } [/tex]

[tex]\rm x_{1}\times x_{2}=\dfrac{c }{ a } [/tex]

Rumus tersebut juga untuk mengetahui hubungan penjumlahan,pengurangan dan perkalian akar akarnya pada bentuk persamaan umumnya.

[tex] \\ [/tex]

PEMBAHASAN

Dengan metode pemfaktoran

[tex]\rm x^{2}-20x+100=0 [/tex]

[tex]\rm x^{2}-10x-10x+100=0 [/tex]

[tex]\rm x(x-10)-10(x-10)=0 [/tex]

[tex]\rm (x-10)(x-10)=0 [/tex]

[tex]\rm (x-10)^{2}=0 [/tex]

[tex]\rm x-10=0 [/tex]

[tex]\rm x=10 [/tex]

[tex]\rm x=c=10 [/tex]

[tex] \\ [/tex]

Kesimpulan :

Jadi, Akar akar persamaan kuadrat dari [tex]\rm x^{2}-20x+100=0 [/tex] adalah [tex]\rm =10 [/tex] (Opsi E)
