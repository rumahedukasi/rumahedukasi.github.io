10. sbuah mug berisi air seberat 400 gram diletekan di atas meja hitunglah tekan mug terhadap meja! tugas mtk​


Jawaban:

semangat semangat semangat

Jawab:

Diketahui: mug berisi air = 400 g = 0,4 kg

                 asumsikan percepatan gravitasi 9,8 m/s^2

                 bidang tekan adalah meja (asumsikan berbentuk pesegi panjang dengan rumus luas adalah p x l = pl m^2)

Ditanya: Besar tekanan mug terhadap meja (P)?

Penjelasan dengan langkah-langkah:

P= Tekanan (Pa)

F= Gaya --> m (massa) x g (percepatan gravitasi) --> (Newton)

A= Luas bidang tekan (dalam kasus ini adalah meja) (m^2)

Rumus: P = F/A

                = (m x g)/A

                = (0,4 kg x 9,8 m/s^2) / A

               = 0,04 Newton / A

misalnya panjang meja 4 m dengan lebar 2 m, maka luas meja adalah 8 m^2

               = 0,04 N / 8 m^2

               = 5 x 10^-3 Pa

✏Perpangkatan adalah sebuah bentuk operasi dalam ilmu matematika yang boleh digunakan apabila sebuah bilangan dikali dengan bilangan yang sama lebih dari satu kali. Pangkat sendiri merupakan banyaknya bilangan yang dikali dengan bilangan yang sama. Letak pangkat sendiri berada diatas bilangan yang dikalikan tersebut.

✏Bilangan berpangkat merupakan sebuah bilangan yang ditujukan untuk menyederhanakan penulisan bilangan apabila dikali dengan angka yang sama berulang kali.

▪ Contoh Perpangkatan 2:

2² = ( 2 x 2 ) = 4

3² = ( 3 x 3 ) = 9

4² = ( 4 x 4 ) = 16

5² = ( 5 x 5 ) = 25

▪ Contoh Perpangkatan 3:

2³ = ( 2 x 2 x 2 ) = 8

3³ = ( 3 x 3 x 3 ) = 27

4³ = ( 4 x 4 x 4 ) = 64

5³ = ( 5 x 5 x 5 ) = 125

▪ Contoh Perpangkatan 4:

2⁴ = ( 2 x 2 x 2 x 2 ) = 16

3⁴ = ( 3 x 3 x 3 x 3 ) = 91

4⁴ = ( 4 x 4 x 4 x 4 ) = 256

5⁴ = ( 5 x 5 x 5 x 5 ) = 625

▪ Contoh Perpangkatan 5:

2⁵ = ( 2 x 2 x 2 x 2 x 2 ) = 32

3⁵ = ( 3 x 3 x 3 x 3 x 3 ) = 273

4⁵ = ( 4 x 4 x 4 x 4 x 4 ) = 1.024

5⁵ = ( 5 x 5 x 5 x 5 x 5 ) = 3.125
