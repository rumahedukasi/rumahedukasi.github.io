11. A plan of a building is drawn on the scale of 1:200. a.) The width of the building is 15 m, how wide is the plan of the building? Give your answer in centimetres. b.) The length of the building drawn in the plan is 12 cm, how long the actual length oh the building? Give your answer in metres.


Jawaban:

a. 7½cm

b. 24m

Penjelasan dengan langkah-langkah:

The scale for this plan is 1 : 200.

This means that 1 cm on the plan represents 200 cm (or 2 m) in real life.

a. actual width = 15m = 1500cm

width on a plan =

[tex] {1500cm} \times ({1 \div 200} ) \\ = 1500cm \times \frac{1}{200} =7 \frac{1}{2} cm[/tex]

b. length on a plan = 12 cm = 0,12m

actual length =

[tex] {0.12m} \div ({1 \div 200} ) \\ = 0.12m \times {200} =24m[/tex]

30. A building is drawn with a scale of 1: 1000. If the length of the building is 64 m, then the length of the building in the map is ... cm.


Height of the building on the map

The formula = Scale x actual building height

           = [tex]\frac{1}{1000}[/tex]x 6400 cm

          = [tex]\frac{64}{10}[/tex] cm

          = 6.4 cm

This is the formula

Scale = Building height on map : Actual building height

Building height on map = Scale x actual building height

Actual building height = Building height on map : Scale

Penjelasan:

Terjemahan adalah sebuah hal yang memiliki makna yang sama dari bahasa tertentu. Biasanya terjemahan ini dari bahasa asing ke bahasa yang ia mengerti. Jika tidak bisa berfikir sendiri, mereka biasa menggunakan alat penerjemahan otomatis lewat internet itu sendiri.

Label => artinya tetap label, biasanya berisi yang menjelaskan nama barang itu sendiri.

Contents => artinya konten, namun untuk produk biasanya lebih ke arah isi dari produk itu sendiri.

Name of the product => artinya nama produk.

Name of the brain => seharusnya name of the brand artinya nama merek produk.

Description of the product => artinya deskripsi produk

Ingredients => artinya bahan-bahan.
