9. apakah yang dimaksud dengan web atau situs


Apakah yang dimaksud dengan web atau situs, website atau situs adalah suatu aplikasi yang berisikan informasi atau data lainnya yang dibuat menggunakan bahasa pemrograman tertentu dan menggunakan jaringan internet untuk mengaksesnya. Seiring dengan perkembangan jaman, website atau situs tidak hanya berfungsi untuk menampilkan informasi dan data saja, bahkan kemampuan website hampir setara dengan aplikasi yang desktop yang terinstall di komputer kita. Cotohnya banyak website yang sudah bisa melakukan pengeditan dokumen, bahkan pengeditan video.


Untuk membuat website dibutuhkan domain dan hosting. Domain berfungsi untuk mempermudah kita dalam mengakses suatu website. Domain yang terkenal dan sering dipakai adalah .com, .org, .co.id dan masih banyak lainnya. Selain domain, dalam pembuatan website kita memerlukan adanya hosting. Hosting inilah tempat kita meletakan file-file yang ada didalam website kita. Tanpa menguasai bahasa pemrograman kita dapat membuat website, yaitu menggunakan CMS. Proses penginstallan CMS ini sangat mudah, hanya dengan beberapa klik saja sudah selesai.


Demikian penjelasan mengenai apa itu website atau situs, website atau situs memerlukan jaringan internet, serta dalam membuat website kita memerlukan adanya hosting dan domain. Untuk membaca materi lainnya mengenai website atau situs dan cara membuat website dapat dibaca pada link berikut

16. Apa fungsi situs web


Jawaban:

1. Fungsi komunikasi

Situs web yang mempunyai fungsi komunikasi pada umumnya adalah situs web dinamis yang memberikan fungsi-fungsi komunikasi

2. Fungsi Informasi

Situs web yang memiliki fungsi informasi pada umumnya lebih menekankan pada kualitas bagian kontennya karena tujuan situs tersebut adalah menyampaikan isinya.

3. Fungsi Entertainment

Situs web juga dapat memiliki fungsi entertainment/liburan. Beberapa fasilitas yang memberikan fungsi hiburan adalah game online, film online, musik online, dan sebagainya.

4. Fungsi Transaksi

Situs web dapat dijadikan sarana transaksibisnis, baik barang, jasa, atau lainnya.
