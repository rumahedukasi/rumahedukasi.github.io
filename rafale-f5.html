32. jelaskan tombol di bawah inj dalam penggunaan microsoft power point:a. F5b. Shift+F5


a. F5 digunakan untuk memulai presentasi (slide show) dari awal slide (slide pertama)

b. Shift+F5 digunakan untuk memulai presentasi dari slide yang aktif.

# perhatikan gambar berikut (slide aktifnya adalah slide kedua):
bila ditekan tombol F5, maka presentasi akan dimulai dari slide pertama
bila ditekan tombol Shift+F5, maka presentasi akan dimulai dari slide kedua
F5 merupakan tombol jalur pintas bila ingin memulai tampilan penuh presentasi di halaman / slide pertama, sekalipun sedang membuka tampilan halaman / slide lainnya di file ppt itu.

Kalau sift+ f5 merupakan tombol jalur pintas bila ingin memulai tampilan penuh pada halaman / slide yang sedang dipilih di file ppt tersebut

38. tombol F5 berfungsi untuk​


Tombol F5 atau F5 berfungsi sebagai tombol refresh atau memutarnya (reload) halaman web. Tujuan dari tombol ini adalah untuk mengupdate hasil yang ada di layar web sesuai dengan isi yang terbaru dari server.

Jika Anda menekan tombol F5 ketika berada di halaman suatu situs web, maka halaman tersebut akan di-reload atau di-update dengan konten terbaru yang ada di server.

40. 29. Shortcut yang digunakan untuk debugging pada VB adalah .... a. F5 c. Shift + F5 b. F4 d. Ctrl + F4 ​


Jawaban:

Tombol shortcut pada keyboard yang berfungsi untuk menjalankan program yang sudah kita buat dalam program aplikasi visual basic adalah F5

Penjelasan:

Tombol F5 Pada visual basic, Visual studio dan visual studio code berfungsi sebagai debug, ( Menjalankan aplikasi )

Ctrl + F5 untuk menjalankan file tanpa melakukan debugging

Shift + F5 untuk menghentikan proses saat ini

Tombol F1 berfungsi untuk melakukan configurasi

Tombol F3 berfungsi untuk mencari kata-kata dalam suatu file ( Sama seperti Ctrl + F )

Tombol F6 tidak berfungsi untuk apapun

Tombol F10 juga tidak berfungsi untuk apapun

Meskipun banyak tombol shortcut pada keyboard yang tidak dipakai / kesulitan menggunakan shortcut yang sudah default.

Shortcut dapat diganti dengan

Memilih tools --> Option --> Environment --> Keyboard --> Pilih command untuk diganti --> Setting sesuai keinginan --> Pilih Ok.
