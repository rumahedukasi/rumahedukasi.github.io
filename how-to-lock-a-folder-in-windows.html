Cara Membuat Folder Baru Di Komputer Pada Windows Explorer

Folder dalam komputer biasanya juga sudah ada pada saat komputer dibeli. Tetapi hanya setingan default atau bawaan saja. Seiring dengan berjalanya waktu pasti akan banyak file yang mau tidak mau sengaja ataupun tidak akan masuk ke dalam komputer anda. 

Nah file tersebut dapat kita lihat pada Windows Explorer. Bagi yang kurang jelas, dapat anda lihat pada postingan sebelumnya di sini . Langsung saja kita belajar membuat folder baru. Tentunya kita sudah punya file yang akan kita masukan dalam folder. Misalkan file Audio. Kita akan membuat folder dengan nama “Musik Saya”. Untuk memperjelas, Lets Do It. 


.Buka Windows Explorer dan silahkan menuju lokasi file audio yang akan anda kumpulkan kedalam folderDisana kita lihat file audio berserakan. Di ruang kosong silahkan klik kanan dan pilih new kemudian pilih folder.

￼


Setelah anda klik atau pilih, maka akan ada folder baru dengan nama default “New Folder”  dan anda langsung bisa edit namanya.

￼


Ketik nama folder sesuai dengan “Musik Saya”. Folder baru sudah jadi dan anda tinggal memasukan file kedalamnya dengan perintah cut/paste.

￼

Cara Mengubah Nama Folder

Selanjutnya bagaimana apabila kita akan merubah nama sebuah folder yang telah kita buat, atau memang sudah ada saat kita membeli komputer tersebut. Ini juga bukan pekerjaan yang sulit, bahkan sangatlah mudah. 

Pertama anda klik pada folder yang ingin anda ubah namanya. Kemudian klik kanan dan pilih rename. Atau anda dapat juga melakukan ubah nama dengan cara klik dua kali dngan jeda beberapa detik pada folder yang ingin anda ubah namanya.


Sangat mudah bukan sobat. Nah sobat semua bisa praktekan langsung ilmu yang didapatkan hari ini dengan komputer sobat semua. Anda bisa melakukan sortir file-file yang sejenis dan anda masukan ke dalam satu folder dengan nama tertentu. Tentu akan memudahkan anda nantinya saat mencari beberapa file yang penting.

17. Mengapa ketika membuat folder dengan nama con tidak bisa di windows 7?


Tidak bisa membuat folder dengan nama CON
Keanehan ini pertama kali ditemukan oleh orang India, dia menemukan bahwa tidak ada seorang pun yang dapat membuat FOLDER di komputer dengan program Microsoft manapun dengan nama folder "CON". Ini merupakan hal nyata dan seluruh kru Microsoft sendiri tidak dapat menjawabnya kenapa hal ini bisa terjadi. Cobalah dengan membuat sebuah folder dengan nama CON , PRN , AUX , NUL , COM1 , LPT1. apa yang terjadi...??? gak bisa kan.... Coba cari hal lainnya yang berhubungan dengan system numerik-nya DOS.
Sebenarnya kru Microsoft bisa menjelaskan hal ini. Karena hal ini berhubungan dengan system numerik jaman dulu (DOS system). Jelas, kalo DOS merupakan sistem operasi dasar dari WINDOWS, sehingga sistem secara default menghindari penamaan dengan menggunakan kata-kata diatas, hal ini juga berlaku bukan hanya untuk folder, tetapi juga untuk penamaan file, coba jajal rename file (misal) contoh.doc menjadi PRN atau prn, kemudian enter. Hal serupa pun pasti akan terjadi.
Karena ini cuma berlaku buat OS turunan DOS, jadi ke OS selain itu (seperti hal nya LINUX) hal tersebut tidak akan berlaku.

22. Jelaskan kelebihan Dan kekurangan Dari Dari masing- masing aplikasi pengunci folder ! 1.Folder Lock Free 2.Easy file locker 3.Instant Lock Mohon bantuannya kak


Kelebihan dan kekurangan dari masing-masing aplikasi pengunci folder 

- Folder Lock Free

Kelebihan : Kunci enkripsi melindungi file dan folder. Backup online yang aman. Dapat mengunci file dan folder, dan membuat file dan folder tersebut tersembunyi. Fasilitas File Shredding, penghancuran file. 

Kekurangan : Password default dari aplikasi ini adalah nomor serial. Untuk backup online membutuhkan biaya langganan tambahan.

- Easy File Locker

Kelebihan : User dapat menentukan level proteksi dari file atau folder. 

Kekurangan : Perlindungan file dapat diakali dengan masuk ke safe mode.

- Instant Lock

Kelebihan : Bersifat portabel, dapat disimpan didalam flashdisk

Kekurangan : Karena sifat portabel-nya, aplikasi rawan hilang
