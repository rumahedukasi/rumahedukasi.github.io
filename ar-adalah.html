21. Dalam ring R, suatu elemen a ∈ R disebut nilpoten jika a^n = 0 untuk suatu bilangan asli n. Tunjukkan jika R komutatif maka P = {a ∈ R | a nilpoten} merupakan ideal dalam R!


Salah satu teorema pembuktian ring :

Misal, R ring, P ⊂ R. Maka, P merupakan ideal jika dan hanya jika

i) P ≠ ∅

ii) ∀a,b∈P berlaku a + (-b) ∈ P

iii) ∀a∈P dan b∈R, berlaku ab ∈ P dan ba ∈ P

Akan dibuktikan, P = {a ∈ R | a^n = 0, n ∈ℕ} adalah ideal dari R

i) pilih a = 0

Karena R ring, dan 0 ∈ R, serta 0^1 = 0 menyebabkan 0 nilpoten, maka 0 merupakan anggota P.

Artinya, ∃a = 0 э P ≠ ∅

ii) Ambil a, b ∈ P sebarang

artinya, a^n = 0 dan b^m = 0

Berdasarkan teorema binomial, didapat :

a^n - b^m = (a - b)^p = 0 untuk p ≥ n + m - 1

Karena n, m ∈ ℕ, maka n + m - 1 ∈ ℕ

Artinya, (a - b)^p = 0 untuk p ∈ ℕ

Artinya, (a - b) = a + (-b) = 0

Artinya, a + (-b) nilpoten

Jadi, ∀a, b ∈ P didapat a + (-b) ∈ P

iii)

Ambil a ∈ P, b ∈ R sebarang

Karena a nilpoten, maka a^n = 0

maka, (a^n)(b^n) = 0

(ab)^n = 0

ab = 0

serta

(b^n)(a^n) = 0

(ba)^n = 0

ba = 0

Jadi, ab, ba ∈ P

Artinya, ∀a∈P dan b∈R, didapat ab∈P dan ba∈P

Berdasarkan i-iii, terbukti bahwa P merupakan ideal dari R

b. Sifat komutatif

Contoh :

( -4 ) + 7 = 3 }

7 + ( -4 ) = 3 } [ ( -4 ) + 7 = 7 + ( -4 )

c. Sifat Assosiatif

[ 5 + (-7) ] + 8 = -2 + 8 = 6}
