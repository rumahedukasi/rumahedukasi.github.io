19. no. 13so hard, please jawabHiGh PoiNt


Kelas 5 Matematika
Bab Pengukuran

Luas sebenarnya
= luas pada denah x (1/skala)²
= 64 x (1/(1/50))²
= 64 x 2.500
= 160.000 cm²
= 16 m²Halo, selamat pagi kawan :)
___________________________

Mapel : Matematika
Tentang :
- Skala
- Bangun Datar
Pertanyaan yg diajukan : 1
Pembahasan :

Diketahui :
- Luas Pada Gambar : 64 cm²
- Skala (S) : 1 : 50

Ditanya :
- Luas Sebenarnya

Cara :

• Panjang Sisi Pada Gambar (PP) :

[tex]l = {s}^{2} \\ \\ 64 = {s}^{2} [/tex]

[tex]s = \sqrt{64} \\ \\ s = 8 \: cm[/tex]

• Panjang Sisi Sebenarnya (PS)

[tex]ps = \frac{pp}{s} \\ \\ ps = \frac{8}{ \frac{1}{50} } [/tex]

[tex]ps = 8 \times \frac{50}{1} \\ \\ ps = 400 \: cm[/tex]

[tex]ps = \frac{400}{100} \\ \\ ps = 4 \: m[/tex]

• Luas Sebenarnya :

L = s x s
L = 4 x 4
L = 16 m²

Jadi, Luas sebenarnya dari taman tersebut adalah 16 m². Jawabanmu yg D
