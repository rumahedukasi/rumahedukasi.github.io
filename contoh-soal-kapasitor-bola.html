2. Contoh soal kapasitansi kapasitor


Jawab:

Sebuah kapasitor dipasang pada tegangan 220 volt. Bila muatan yang tersimpan 110 μC, tentukan kapasitas kapasitornya!

Penjelasan dengan langkah-langkah:

Diketahui:

V = 220 volt

Q = 110 μC

Ditanya: C =…?

Pembahasan:

Gunakan persamaan berikut untuk mencari kapasitas kapasitor.

[tex]C=\frac{Q}{V} \\\ \\C=\frac{110}{220} \\\\= 0,5uf[/tex]

Jadi, kapasitas kapasitornya adalah 0,5 μF.

[tex]C = 4 \pi \times 8,85 \times 10^{-12} \times 3,3 \times 10^{-4}[/tex]

[tex]C = 1,1682\pi \times 10^{-14} ~ \texttt{Farad}[/tex]

[tex]C \approx 3,67 \times 10^{-14} ~ \texttt{Farad}[/tex]

Kapasitor bola terdiri dari dua bola sepusat yang radiusnya 1 mm dan 2 mm. Kapasitor silinder terdiri dari dua silinder sesumbu ( koaksial ) yang radiusnya 2 mm dan 4 mm, dengan panjang 1 cm, dan kapasitor lempeng terdiri dari dua lempeng sejajar luas masing-masing lempeng 1 cm2 dengan jarak I mm. Sistem diberi beda potensial 100 volt. Tentukan : a. Kapasitans masing-masing kapasitor b. Kapasitans kapasitor yang senilai dengan kapasitans system c. Beda potensial antara ujung-ujung masing-masing kapasitor.


a) Kapasitas kapasitor masing-masing kapasitor:

Kapasitas kapasitor bola adalah [tex]C_{1} = 0,22 pF[/tex]Kapasitas kapasitor silinder adalah [tex]C_{2} = 0,802 pF[/tex]Kapasitas kapasitor lempeng adalah  [tex]C_{3} = 0,885 pF[/tex]

b) Kapasitas kapasitornya adalah sebesar [tex]C_{s} = 0,16 pF[/tex]

c) Beda potensial masing-masing kapasitor:

Beda potensial kapasitor bola adalah 72,72 VBeda potensial kapasitor silinder adalah 19,95 VBeda potensial kapasitor lempeng adalah 18, 079 V

Penjelasan dengan langkah-langkah:

Kapasitor adalah komponen listrik yang berfungsi untuk menyimpan muatan listrik. Perbandingan antara muatan setiap keping dan beda potensial disebut dengan kapasitansi atau kapasitas kapasitor.

Kapasitansi dapat dirumuskan sebagai berikut.

[tex]C = \frac{q}{V}[/tex]

dimana:

C = kapasitas kapasitor (farad)

q = jumlah muatan tiap keping (coloumb)

V = beda potensial (volt)

1. Kapasitas Kapasitor Bola

Kapasitas kapasitor bola konduktor dapat dirumuskan sebgai berikut.

[tex]C = \frac{q}{V} \\C = \frac{R_{1} R_{2} }{k(R_{2}-R_{1}) }[/tex]

Untuk kapasitor bola terisolasi dirumuskan sebagai:

C = 4πε0R

C = [tex]C = \frac{R}{k}[/tex]

dimana:

C = kapasitas kapasitor bola (F)

R = jari-jari kapasitor (m)

k = tetapan Coloumb [tex](9 . 10^{9} Nm^{2}/C^{2} )[/tex]

2. Kapasitas Kapasitor Silinder

Dapat dirumuskan sebagai berikut.

C = (2πε0L) / (ln (Rl/Rd) )

dimana :

L = panjang kapasitor (m)

Rl = jari-jari luar (m)

Rd = jari-jari dalam (m)

3. Kapasitas Kapasitor Keping Sejajar / Lempeng

dapat dirumuskan sebagai berikut.

[tex]C = \frac{q}{V}[/tex]

C = Aε0 / d

dimana:

A = luas tepi keping pelat ([tex]m^{2}[/tex])

ε0 = permitivitas udara/vakum ( [tex]8,85.10^{-12} C^{2}/Nm^{2}[/tex])

d = jarak pisah kedua keping (m)

Rangkaian Seri Kapasitor

1) Muatan pada tiap-tiap kapasitor sama

[tex]q = q_{1} = q_{2} = q_{n}[/tex]

2) Beda potensial pada ujung-ujung kapasitor pengganti sama dengan pejumlahan beda potensial seluruh kapasitor

[tex]V = V_{1} + V_{2} + ... + V_{n}[/tex]

3)  Besar kapasitas pengganti dapat dihitung dengan persamaan:

[tex]\frac{1}{C_{s}} = \frac{1}{C_{1} } + \frac{1}{C_{2} } + ... + \frac{1}{C_{n} }[/tex]

---

Dari soal dapat diketahui bahwa:

Kapasitor Bola = [tex]C_{1}[/tex]

[tex]R_{C_{1} } = 1 mm = 1 . 10^{-3} m[/tex]

[tex]R_{C_{1} } = 2 mm = 2 . 10^{-3} m[/tex]

Kapasitor Silinder = [tex]C_{2}[/tex]

[tex]R_{d} = 2 mm = 2 . 10^{-3} m\\R_{l} = 4 mm = 4 . 10^{-3} m\\L = 1 mm = 1 . 10^{-2} m[/tex]

Kapasitor Lempeng = [tex]C_{3}[/tex]

[tex]A = 1 cm^{2} = 1. 10^{-4} m^{2}\\ d = 1 mm = 1.10^{-3} m[/tex]

Beda potensial = V = 100 V

Ditanya:

a) Kapasitas kapasitor masing-masing kapasitor

b) Kapasitas kapasitor

c) Beda potensial masing-masing kapasitor

Penyelesaian:

a) Kapasitas Kapasitor masing-masing:

- Kapasitor bola

 [tex]C_{1} = \frac{q}{V}\\C_{1} = \frac{R_{1} R_{2}}{k (R_{2} - R_{1})}[/tex]

[tex]\\C_{1} = \frac{2.10^{-3}m (1.10^{-3}m) }{9.10^{9} Nm^{2} (2.10^{-3}m - 1.10^{-3}m - 2. 10^{-3}m }[/tex]

[tex]C_{1} =\frac{2.10^{-3} }{9. 10^{6} }\\ C_{1} = 0,22 . 10^{-12} F\\ C_{1} = 0,22 pF[/tex]

- Kapasitor silinder

C2 = (2πε0L) / (ln (Rl/Rd) )

[tex]C_{2} = \frac{2(3,14) (8,85 . 10^{-12} C^{2}/Nm^{2}) (1.10^{-2} m) }{ln \frac{4.10^{-3} }{2.10^-3} } }[/tex]

[tex]C_{2} = \frac{5,57.10^{-13} }{0,693} \\C_{2} = 8,02 . 10^{-13} F\\C_{2} = 0,802 pF[/tex]

- Kapasitor lempeng

C3 = Aε0 / d

[tex]C_{3} = \frac{1.10^{-4}m^{2} (8,85 . 10^{-12} C^{2} / Nm^{2} }{1.10^{-3} } \\C_{3} = \frac{(8,85 . 10^{-16}) }{1.10^{-3}} \\C_{3} = (8,85 . 10^{-13})F\\C_{3} = 0,885 pF[/tex]

b) Kapasitas kapasitor

[tex]\frac{1}{C_{s}} = \frac{1}{C_{1} } + \frac{1}{C_{2} }+\frac{1}{C_{3} }\\\frac{1}{C_{s}} = \frac{1}{0,22pF} + \frac{1}{0,802pF} + \frac{1}{0,885pF}\\\frac{1}{C_{s}} = \frac{1}{0,22pF} + \frac{0,17644}{0,22pF} + \frac{0,1947}{0,22pF}\\\frac{1}{C_{s}} = \frac{1,371}{0,22pF}[/tex]

[tex]C_{s} = \frac{0,22}{1,371}\\C_{s} = 0,16 pF[/tex]

c) Beda potensial antara ujung-ujung masing-masing kapasitor

- Mencari muatan kapasitor

[tex]C = \frac{q}{V} \\q = C.V\\q = (0,16pF) (100V)\\q = 16 pC[/tex]

- Beda potensial masing-masing kapasitor:

- Kapasitor bola:

[tex]V = \frac{q}{C_{1} }\\V = \frac{16pC}{0,22 pF} \\V = 72,72 V[/tex]

- Kapasitor Silinder:

[tex]V = \frac{q}{C_{1} }\\\\V = \frac{16pC}{0,802 pF} \\V = 19,95 V[/tex]

- Kapasitor Lempeng:

[tex]V = \frac{q}{C_{1} }\\\\V = \frac{16pC}{0,885 pF} \\\\V = 18,079 V[/tex]

∴ Sehingga dapat disimpulkan bahwa:

a) Kapasitas kapasitor masing-masing kapasitor nya adalah:

Kapasitas kapasitor bola adalah [tex]C_{1} = 0,22 pF[/tex]Kapasitas kapasitor silinder adalah [tex]C_{2} = 0,802 pF[/tex]Kapasitas kapasitor lempeng adalah  [tex]C_{3} = 0,885 pF[/tex]

b) Kapasitas kapasitornya adalah sebesar [tex]C_{s} = 0,16 pF[/tex]

c) Beda potensial masing-masing kapasitor:

Beda potensial kapasitor bola adalah 72,72 VBeda potensial kapasitor silinder adalah 19,95 VBeda potensial kapasitor lempeng adalah 18, 079 V

10. dapatkah bola konduktor digunakan sebagai kapasitor?​


Jawaban:

Ya, bola konduktor dapat digunakan sebagai salah satu komponen dari kapasitor. Kapasitor adalah perangkat elektronik yang digunakan untuk menyimpan muatan listrik. Ini terdiri dari dua konduktor yang dipisahkan oleh bahan dielektrik atau ruang hampa udara.

Dalam kasus bola konduktor, biasanya satu bola digunakan sebagai salah satu konduktor, dan yang lainnya dapat berupa tanah (misalnya, melalui kabel yang menyentuh tanah) atau bola konduktor kedua. Ini menciptakan kapasitor sederhana dengan dua konduktor terpisah.

Untuk meningkatkan kapasitansi dari kapasitor bola konduktor ini, Anda dapat mengatur jarak antara bola-bola tersebut atau mengganti salah satu bola dengan bola yang lebih besar. Kapasitor bola konduktor sering digunakan dalam eksperimen pendidikan dan demonstrasi untuk mengilustrasikan prinsip-prinsip dasar kapasitor.

Namun, dalam aplikasi praktis di sirkuit listrik, kapasitor umumnya dibuat dalam bentuk yang lebih struktural seperti kipas, tabung, atau pelat datar untuk mengoptimalkan kapasitansinya dan memenuhi kebutuhan desain sirkuit tertentu.

13. Jelaskan apa yang dimaksud kapasitor fungsi kapasitor dan susunan kapasitor​


Jawaban :

Pengertian Kapasitor adalah sebuah komponen elektronika yang fungsi dasarnya untuk menyimpan muatan atau arus listrik didalam sebuah medan listrik untuk waktu yang terbatas sehingga secara fungsi mirip dengan baterai yaitu menyimpan arus.

Susunan kapasitor dapat berupa rangkaian seri maupun rangkaian paralel. Di pasaran banyak kita jumpai kapasitor yang nilai kapasitasnya bermacam-macam, dari yang kecil yaitu dalam ukuran piko farad (pF), nano farad (nF), dan mikro farad (μF). Akan tetapi kadang-kadang yang ada di pasaran tidak cocok dengan yang kita butuhkan, sehingga kita dapat menyusun kapasitor itu sedemikian rupa memiliki kapasitas yang kita butuhkan. Dalam rangkaian listrik ataupun rangkaian elektronika, kapasitor dapat disusun dalam dua cara, yaitu susunan seri dan paralel, tetapi dapat juga disusun gabungan susunan seri dan paralel.

Kondensator memiliki satuan yang disebut Farad dari nama Michael Faraday. Kondensator juga dikenal sebagai "kapasitor", namun kata "kondensator" masih dipakai hingga saat ini. Pertama disebut oleh Alessandro Volta seorang ilmuwan Italia pada tahun 1782 (dari bahasa Itali condensatore), berkenaan dengan kemampuan alat untuk menyimpan suatu muatan listrik yang tinggi dibanding komponen lainnya. **Penyelesaian:**

Kapasitor bola memiliki kapasitas sebagai berikut:

```

C = 4πε0r²

```

Di mana:

* C adalah kapasitansi (F)

* ε0 adalah permitivitas ruang hampa (8,854 × 10⁻¹² F/m)

* r adalah jari-jari kapasitor (m)

```

C = 4πε0r²

```

```

C = 4π * 8,854 × 10⁻¹² F/m * (18 cm)²

```

```

C = 3,14 × 8,854 × 10⁻¹² F/m * 324 cm²

```

```

C = 8,99 × 10⁻¹¹ F

```

Energi yang tersimpan dalam kapasitor bola adalah:

```

U = ½CV²

```

Di mana:

* U adalah energi (J)

* C adalah kapasitansi (F)

* V adalah tegangan (V)

Dengan demikian, energi yang tersimpan dalam kapasitor bola tersebut adalah:

```

U = ½CV²

```

```

U = ½ * 8,99 × 10⁻¹¹ F * 6 V²

```

```

U = 9,99 × 10⁻¹¹ J

```

**Jawaban:**

Energi yang tersimpan dalam kapasitor bola tersebut adalah **9,99 × 10⁻¹¹ J**.

Kapasitas kapasitor bola dapat dihitung dengan menggunakan rumus kapasitas kapasitor bola. Dengan menggunakan rumus tersebut, diperoleh kapasitas kapasitor bola tersebut adalah 8,99 × 10⁻¹¹ F.

**Catatan:**

Energi yang tersimpan dalam kapasitor bola adalah energi potensial listrik yang tersimpan pada kapasitor. Energi ini dapat diubah menjadi energi listrik, energi panas, atau energi mekanis.

38. tolong dijawab soal fisika tentang kapasitor ini


Pada Rangkaian Capasitansi.

Ketika beberapa capasitor di rangkai secara seri.
1/Cs = 1/C + 1/C +...
1/Cs = n 1/C
Cs = C/n

Ketika n capasitor dirangkai secara paralel.
Cp = C + C +...
Cp = nC

Kemudian rangkaian seri dan paralel di rangkai secara seri

1/C = 1/Cp + 1/Cs

1/C = 1/nC + n/C
1/C = (1/nC + n^2/nC)
1/C = (1+ n^2) / nC

C = nC / (1 + n^2)

Jadi, capasitor penggantinya adalah nC/(1+n^2)
