Jawaban:

User adalah nama lain dari pengguna nama ini umumnya di gunakan untuk seseorang yang memanfaatkan suatu layanan atau sistem tertentu, jika seseorang memanfaatkan sebuah layanan atau fasilitas yang di sediakan oleh sebuah perangkat (dalam hal ini mikrotik) maka seseorang tersebut dapat di kategorikan sebagai user(pengguna).

Sebagai contoh saat anda mencoba login ke dalam menu winbox yang ada pada mikrotik tentunya anda di haruskan untuk memasukan username dan password pada winbox tersebut, tapi dari pihak mikrotik sebelumnya sudah membuat user default yang memudahkan setiap user baru yang ingin mengakses mikrotik tersebut cukup memasukan username = admin dan password = kosong cara ini memang sangat mempermudah seorang user yang akan mengakses mikrotik apalagi saat baru pertama menggunakan mikrotik.

Namun hal yang fatal bisa terjadi jika ternyata administrator pada jaringan tersebut membiarkan user dan password default bawaan mikrotik masih aktif karna hal ini akan mengakibatkan setiap user asing dapat masuk dan mengakses setiap konfigurasi pada mikrotik tersebut jika user asing hanya melihat-lihat konfigurasinya saja mungkin itu bukan hal yang harus di khawatirkan akan tetapi bisa di bayangkan jika user asing tersebut mencoba merubah konfigurasi yang sudah dengan susah payah kita buat yang mengakibatkan sebagian jaringan eror atau bahkan mati total karna tabrakan IP atau di reset oleh user asing tersebut dan tentu hal ini tidak ingin anda alami bukan Untuk mencegah masuknya user asing pada mikrotik milik kita, pihak mikrotik sudah memberikan fasilitas user pada setiap mikrotik yang sangat membantu untuk mengatasi masuknya user asing pada mikrotik milik kita.

Jawaban panjang:

User adalah pengguna komputer. Disini user akan mengakses suatu halaman web di Internet. Karena itu yang user lakukan pertama adalah membuka browser di komputernya.

Browser adalah perangkat lunak yang digunakan untuk mengakses halaman internet melalui jaringan internet. Contoh dari browser adalah Chrome, Firefox, Opera dan Internet Explorer. Untuk menggunakan browser, seorang user akan mengetik alamat URL ke browser.

URL adalah alamat yang digunakan untuk mengakses suatu berkas atau data di jaringan komputer. Alamat internet adalah salah satu bentuk URL. Oleh browser, URL ini akan dikirimkan ke server untuk diproses. Contoh URL misalnya adalah http://www.goole.com. Bila user mengetik alamat ini maka dia akan meminta melalui browser di komputernya untuk menghubungkan diri dengan server Google dan menampilkan halaman utama dari situs Google.

Server adalah sistem komputer yang mengolah alamat internet dan menghantarkan halaman dari data yang disimpan di server kembali untuk ditampilkam di browser pada komputer user yang meminta halaman tersebut.

Halaman web yang dikirmkan bisa berupa halaman statis yang tetap. Misalnya halaman berupa file html atau gambar. Namun halaman web yang dikirimkan bisa juga berupa halaman dinamis yang dihasilkan dari pengolahan database dan dengan menggunakan querry atau pencarian.
