34. perbedaan antara jenis perangkat lunak public domain dan open source adalah?​


Perbedaan antara perangkat lunak public domain dan open source adalah sebagai berikut:

Public Domain: Perangkat lunak public domain adalah perangkat lunak yang tidak memiliki hak cipta, yang berarti dapat digunakan, dimodifikasi, dan didistribusikan secara bebas tanpa memerlukan persetujuan atau lisensi apapun. Perangkat lunak public domain tidak memiliki pembatasan atau kewajiban tertentu dan dapat digunakan untuk tujuan apapun.

Open Source: Perangkat lunak open source adalah perangkat lunak yang sumber kode terbuka dan dapat diakses oleh siapa saja. Pengguna dapat meninjau dan memodifikasi kode sumber sesuai kebutuhan, dan umumnya dapat mendistribusikan perangkat lunak tersebut ke publik dengan berbagai lisensi tergantung pada jenis perangkat lunak. Salah satu prinsip utama dari perangkat lunak open source adalah bahwa pengguna dapat berpartisipasi dalam pengembangan dan memperbaiki perangkat lunak tersebut.

Perbedaan lainnya adalah bahwa perangkat lunak public domain tidak terikat oleh lisensi atau aturan tertentu, sementara perangkat lunak open source mungkin memiliki beberapa keterbatasan atau persyaratan terkait lisensi yang dapat memengaruhi cara perangkat lunak tersebut dapat digunakan dan didistribusikan.

35. Sebutkan 3 jenis nama domain dan berikan 1 contoh


Jawaban:

-Top Level Domains

.com : merupakan singkatan dari commercial. Domain ini pada awalnya digunakan untuk website dengan tujuan komersial, namun saat ini sudah banyak pemilik web menggunakan .com karena mudah diingat dan cukup populer.

.org : digunakan untuk website organisasi no profit

.edu : domain untuk web institusi pendidikan

-Second Level Domains

Second Level Domains adalah nama domain yang Anda daftarkan.  Sebagai contoh, nike.com, di sini “nike” berperan sebagai second level domains. Anda dapat menggunakan nama merek, nama perusahaan, atau nama proyek untuk digunakan sebagai second level domains.

-Third Level Domain

Third Level Domain adalah bagian dari nama domain website yang terkait dengan SLD (Second Level Domains). Domain ini biasanya ditemukan di sebelah kiri SLD dan sering disebut sebagai subdomain.

Sebagai contoh, jika Anda ingin membuat Third Level Domain untuk blog perusahaan, maka nama domain Anda akan terlihat seperti “blog.contohwebsite.com”. Dalam domain tersebut, kata “blog” berperan sebagai Third Level Domain. Third Level Domain biasanya digunakan untuk merujuk ke server tertentu dalam suatu perusahaan.
