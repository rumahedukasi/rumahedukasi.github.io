[tex]dx= \frac{du}{4x^3} [/tex]

[tex]= \int\ {(x^3)(x^4-3)^{-6}} } \, dx[/tex]
[tex]= \int\ {(x^3)u^{-6}} } \, \frac{du}{4x^3} [/tex]
[tex]= \frac{1}{4} \int\ u^{-6}} } \, du[/tex]
[tex]=(\frac{1}{4}) (\frac{1}{-5})u^{-5}} } +c[/tex]
[tex]=-\frac{1}{20}u^{-5}} } +c[/tex]
[tex]=- \frac{1}{20}(x^4-3)^{-5}+c [/tex]
[tex]=- \frac{1}{20(x^4-3)^{5}}+c [/tex]

Untuk menjawab soal di atas, teman-teman bisa membuka kembali materi Pangkat. Penjelasan dan perhitungan selengkapnya dapat teman-teman baca pada pembahasan berikut.

Pembahasan

Perpangkatan suatu bilangan artinya perkalian berulang dengan bilangan yang sama. Secara umum dapat dituliskan sebagai berikut:

Untuk sebarang bilangan bulat p dan bilangan bulat positif berlaku

[tex]P^{n}[/tex] = {p × p × p × p × ….. × p} sebanyak n faktor

[tex]P^{n}[/tex] disebut bilangan berpangkat, p disebut bilangan pokok, dan n disebut pangkat (eksponen).

Sifat-sifat Operasi Bilangan Berpangkat

Untuk m, n  sembarang bilangan bulat dan p bilangan real, berlaku:

[tex]p^m[/tex] × [tex]p^{n}[/tex] = [tex]p^{m + n}[/tex][tex]p^m[/tex] ÷ [tex]p^{n}[/tex] = [tex]p^{m - n}[/tex][tex](p^{m} )^{n}[/tex] = [tex]p^{(m)(n)}[/tex][tex]p^{-m}[/tex] = [tex]\frac{1}{p^{m} }[/tex][tex]p^{0} =1[/tex]

Ditanyakan:

[tex]\frac{15p^{5} q^{-3} }{3p^{2} q}[/tex] [tex](q^{2}) ^{3}[/tex] dapat disederhanakan menjadi ...

Jawab:

Berdasarkan sifat-sifat operasi hitung bilangan berpangkat, soal di atas dapat kita sederhanakan menjadi:

[tex]\frac{15p^{5} q^{-3} }{3p^{2} q}[/tex] [tex](q^{2}) ^{3}[/tex] = [tex]\frac{15}{3}[/tex] [tex](p^{5-2})[/tex] [tex](q^{-3-1})[/tex] [tex](q^{(2)} ^{(3)} )[/tex] = 5[tex]p^{3}[/tex] [tex](q^{-4} )[/tex][tex](q^{6})[/tex] = 5[tex]p^{3}[/tex][tex](q^{(-4+6)})[/tex] = 5[tex]p^{3}[/tex][tex]q^{2}[/tex].

Sehingga dapat disimpulkan, bentuk sederhana dari [tex]\frac{15p^{5} q^{-3} }{3p^{2} q}[/tex] [tex](q^{2}) ^{3}[/tex] adalah 5[tex]p^{3}[/tex][tex]q^{2}[/tex].

37. conditioner rambut pakai di kulit rambut/seluruh rambut/ujung rambut aja? ​


ujung rambut sampai sepertiga atau setengah rambut dan jangan mengenai kulit kepala

semoga membantu!

Jawaban:

Conditioner rambut umumnya digunakan pada seluruh rambut setelah proses mencuci rambut dengan shampoo. Biasanya, langkah-langkah penggunaannya adalah sebagai berikut:

1. Cuci rambut dengan menggunakan shampoo untuk membersihkan kotoran dan minyak dari rambut.

2. Setelah membilas shampoo hingga bersih, aplikasikan conditioner rambut ke seluruh rambut, mulai dari akar hingga ujung rambut.

3. Pastikan rambut terlapisi dengan baik oleh conditioner. Jangan mengaplikasikan terlalu dekat dengan kulit kepala karena conditioner biasanya dirancang untuk merawat rambut dan tidak untuk kulit kepala.

4. Biarkan conditioner rambut meresap selama beberapa menit sesuai petunjuk pada kemasan (biasanya sekitar 1-3 menit).

5. Bilas rambut secara menyeluruh untuk menghilangkan sisa-sisa conditioner.

Dengan mengaplikasikan conditioner rambut ke seluruh rambut, Anda dapat memberikan nutrisi dan kelembapan pada rambut, sehingga rambut menjadi lebih lembut, mudah diatur, dan terlindungi dari kerusakan akibat paparan lingkungan dan penggunaan alat-alat styling.
