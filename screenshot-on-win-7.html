2. Jelaskan apa perbedaan Win XP dengan Win 7​


Jawaban:

ada banyak perbandingan, tapi akan saya kasih yang penting saja  

1. 32-bit dan 64-bit

sebenarnya Windows XP juga memiliki versi 64-bit, namun tidak banyak orang yang mengetahuinya. Saat Anda melakukan upgrade dari Windows XP ke Windows 7, kita harus memilih versi antara 32-bit atau 64-bit.

2. Aero Desktop

Aero Desktop sebenarnya hanya mempercantik tampilan Windows 7 saja, Fitur seperti Aero Snap dapat memudahkan Anda untuk mengatur jendela-jendela yang terbuka dan tranparansi yang ada dapat memudahkan Anda untuk melihat di balik jendela-jendela yang sedang terbuka.

3. Documents and Settings

Folder Documents and Settings yang merupakan tempat dari segala file dan folder personal telah digantikan dengan folder Users di windows 7.

4. Start Menu

Start Menu pada Windows 7 telah banyak berubah dan mendapat banyak kritik. Start menu tidak lagi menggunakan fly-outs dan scroll-outs

5. DirectX 11

Jika kamu seorang gamer, Anda pasti tahu bahwa Anda harus terus mengikuti perkembangan hardware dan software untuk dapat menjalankan game-game yang makin berat.

6. Touch Support

Windows 7 dahulu merupakan operating system pertama yang men-support touch atau sentuhan sebagai interface

25. Cara membagi partisi di win 7


Jawaban:

1.Pada tampilan desktop, klik tombol Start Menu dan pilih Control Panel.

2.Klik System and Security, jika tampilan “View by:” pada jendela Control Panel Anda adalah “Category”. Jika selain itu, Anda bisa langsung lanjut ke langkah selanjutnya.

3.Klik “Administrative Tools”. Tool ini berguna untuk beberapa hal. Salah satunya, adanya fungsi Defragment Harddisk.

4.Klik dua kali pada “Computer Management”. Jika Anda diminta untuk memasukkan password administrator atau melakukan konfirmasi, ketik passwordnya, atau klik Continue.

5.Klik “Disk Management” yang berada dalam sub menu “Storage”. Setelah diklik, akan muncul daftar partisi yang ada di komputer atau laptop Anda.

6.Klik kanan pada harddisk yang ingin dibagi partisinya, lalu klik “Shrink Volume”.

7.Tentukan berapa besar kapasitas harddisk yang akan dibagi. Jika sudah, klik Shrink.

8.Volume harddisk akan dikompresi dan ruang yang tidak teralokasi akan dibuat.

9.Lakukan kembali langkah nomor 6 pada ruang harddisk yang tidak teralokasi jika Anda hendak membuat lebih dari satu partisi.

33. Apa itu Screenshot? Bantu jawab!!


Jawaban:

Screenshot adalah menyimpan cuplikan isi layar baik pada Handphone, Laptop, atau Komputer.

Pembahasan

Kata "Screenshot" terbagi menjadi 2 yaitu "Screen" yang memiliki arti layar handphone, laptop, atau komputer. sedangkan "shot" yang memiliki arti tangkapan. Jadi, screenshot disebut juga sebagai tangkapan layar dalam bahasa Indonesia. Screenshot diberbagai perangkat handphone, laptop, atau komputer berbeda-beda. Cara screenshot di perangkat Handphone yaitu memencet tombol mengurangi volume dan daya Handphone. sedangkan pada laptop atau komputer yaitu ALT + PrtScn.
