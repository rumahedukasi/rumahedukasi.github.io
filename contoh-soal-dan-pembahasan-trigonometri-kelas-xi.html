PembahasanTrigonometri

Diberikan identitas trigonometri:

[tex]\large\text{$\begin{aligned}\tan^2\left ( \frac{\pi}{4}-\frac{1}{2}\alpha \right )=\frac{1-\sin\alpha}{1+\sin\alpha}\end{aligned}$}[/tex]

Pembuktian

[tex]\begin{aligned}&\textsf{Ruas kiri}\\&{=\ }\tan^2\left(\frac{\pi}{4}-\frac{1}{2}\alpha\right)\\&{=\ }\frac{\sin^2\left(\dfrac{\pi}{4}-\dfrac{1}{2}\alpha\right)}{\cos^2\left(\dfrac{\pi}{4}-\dfrac{1}{2}\alpha\right)}\end{aligned}[/tex]
[tex]\begin{aligned}&\ \left[\ \begin{aligned}&\cos2\theta=1-2\sin^2\theta\\&\Rightarrow \sin^2\theta=\frac{1-\cos2\theta}{2}\\&\cos2\theta=2\cos^2\theta-1\\&\Rightarrow \cos^2\theta=\frac{1+\cos2\theta}{2}\\&\therefore\ \frac{\sin^2\theta}{\cos^2\theta}=\frac{1-\cos2\theta}{1+\cos2\theta}\end{aligned}\ \right]\\&{=\ }\frac{1-\cos\left[2\left(\dfrac{\pi}{4}-\dfrac{1}{2}\alpha\right)\right]}{1+\cos\left[2\left(\dfrac{\pi}{4}-\dfrac{1}{2}\alpha\right)\right]}\end{aligned}[/tex]
[tex]\begin{aligned}&{=\ }\frac{1-\cos\left(\dfrac{\pi}{2}-\alpha\right)}{1+\cos\left(\dfrac{\pi}{2}-\alpha\right)}\\&\ \left[\ \begin{aligned}&\cos\left(\frac{\pi}{2}-\alpha\right)=\sin\alpha\\\end{aligned}\ \right]\\&{=\ }\boxed{\:\frac{1-\sin\alpha}{1+\sin\alpha}\:}=\textsf{Ruas kanan}\\&{\implies} \sf TERBUKTI\ \blacksquare\end{aligned}[/tex]

39. Tuliskan contoh soal identitas trigonometri, jawabannya dan pembahasannya.​


Diketahui :

Pembuktian suatu identitas trigonometri

Ditanya :

Contoh soal pembuktian identitas trigonometri ... ?

Jawab :

1. Soal : Buktikan (sin 2x)/sin x = (1 + cos 2x)/cos x

Penyelesaian :

Pembuktian dari kiri dan kanan langsung.

[tex]\frac{sin2x}{sinx} = \frac{1+cos2x}{cosx}\\\frac{2.sinx.cosx}{sinx} = \frac{(sin^2x+cos^2x)+(cos^2x - sin^2x)}{cosx}\\2.cosx = \frac{2.cos^2x}{cosx}\\2.cosx = 2cosx[/tex]

Terbukti bahwa (sin 2x)/sin x = (1 + cos 2x)/cos x adalah benar.

2. Soal : Buktikan (1 - cos 2x)/(1 - cos² x) = 2

Penyelesaian :

Pembuktian dari kiri.

[tex]\frac{1-cos2x}{1-cos^2x} = 2\\\frac{(sin^2x +cos^2x)-(cos^2x - sin^2x)}{sin^2x} = 2\\\frac{2sin^2x}{sin^2x} = 2\\2 = 2[/tex]

Terbukti bahwa (1 - cos 2x)/(1 - cos² x) = 2 adalah benar.

3. Soal : Buktikan cosec 2x = (1 + cot² x)/(2.cot x)

Penyelesaian :

Pembuktian dari kanan.

[tex]cosec2x = \frac{1+cot^2x}{2.cotx}\\cosec2x = \frac{\frac{sin^2x}{sin^2x}+\frac{cos^2x}{sin^2x}}{2.\frac{cosx}{sinx}}\\cosec2x = \frac{\frac{sin^2x+cos^2x}{sin^2x}}{\frac{2.cosx}{sinx}}\\cosec2x = \frac{\frac{1}{sin^2x}}{\frac{2.cosx}{sinx}}\\cosec2x = \frac{1}{sin^2x} . \frac{sinx}{2.cosx}\\cosec2x = \frac{1}{2.sinx.cosx}\\cosec2x = \frac{1}{sin2x}\\cosec2x = cosec2x[/tex]

Terbukti bahwa cosec 2x = (1 + cot² x)/(2.cot x) adalah benar.
