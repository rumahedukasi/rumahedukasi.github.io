Penjelasan dengan langkah-langkah:

Domain Fungsi

Diketahui :

fungsi [tex]f(x)=\sqrt{2x^2-9x-5}[/tex]

Ditanya :

domain dari fungsi f(x)

Penyelesaian :

fungsi berbentuk [tex]\sqrt{f(x)}[/tex] akan terdefinisi jika [tex]f(x)\geq 0[/tex]

maka [tex]f(x)=\sqrt{2x^2-9x-5}[/tex] akan terdefinisi jika

[tex]2x^2-9x-5\geq 0\\\\(2x+1)(x-5)\geq 0\\\\x\leq -\frac{1}{2}~~atau~~x\geq 5\\\\\\maka~domain~dari~f(x)~adalah~:[x|x\leq -\frac{1}{2}~~atau~~x\geq 5][/tex]

23. Interval nilai x yang memenuhi pertidaksamaan │7 – 4x│ ≤ 3 adalah … A. –5/2 ≤ x ≤ 1 B. –1 ≤ x ≤ 5/2 C. x ≤ –1 atau x ≥ 5/2 D. 1 ≤ x ≤ 5/2 E. x ≤ –5/2 atau x ≥ 1


Diketahui suatu pertidaksamaan nilai mutlak yaitu |7 – 4x| ≤ 3. Interval nilai x yang memenuhi adalah 1 ≤ x ≤ [tex]\frac{5}{2}[/tex] (D).

Penjelasan dengan langkah-langkah

Pertidaksamaan nilai mutlak.

Jika |f(x)| ≥ a maka f(x) ≤ –a atau f(x) ≥ aJika |f(x)| ≤ a maka –a ≤ f(x) ≤ a

Diketahui

|7 – 4x| ≤ 3

Ditanyakan

Tentukan nilai x yang memenuhi pertidaksamaan tersebut!

Jawab

Langkah 1

Berdasarkan rumus berikut:

Jika |f(x)| ≤ a maka –a ≤ f(x) ≤ a

Berarti diperoleh penyelesaian pertidaksamaan yaitu:

Jika |7 – 4x| ≤ 3 maka –3 ≤ 7 – 4x ≤ 3

Langkah 2

Berdasarkan langkah 1, diperoleh penyelesaian pertidaksamaannya yaitu:

–3 ≤ 7 – 4x ≤ 3

Pertidaksamaan 1.

–3 ≤ 7 – 4x

4x ≤ 7 + 3

4x ≤ 10

 x ≤ [tex]\frac{10}{4}[/tex]

 x ≤ [tex]\frac{5}{2}[/tex]

Pertidaksamaan 2.

     7 – 4x ≤ 3

         –4x ≤ 3 – 7

          –4x ≤ –4  

==> kedua ruas dibagi (–4) <==

–4x ÷ (–4) ≥ –4 ÷ (–4)

              x ≥ 1

Langkah 3

Irisan dari pertidaksamaan 1 (x ≤ [tex]\frac{5}{2}[/tex]) dan pertidaksamaan 2 (x ≥ 1) adalah:

1 ≤ x ≤ [tex]\frac{5}{2}[/tex]

Kesimpulan

Jadi interval nilai x yang memenuhi pertidaksamaan |7 – 4x| ≤ 3 adalah 1 ≤ x ≤ [tex]\frac{5}{2}[/tex].

Jawaban D
