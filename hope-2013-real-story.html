8. Banyak bilangan real x yang memenuhi x(pangkat)2016-x(pangkat)2014 = x(pangkat)2015-x(pangkat)2013 adalah...


x^2016 - x^2014 = x^2015 - x^2013

x^2014 (x^2 -1) = x^2013 (x^2 -1)

x^2014 (x^2 -1) - x^2013 (x^2 -1) = 0

x^2013 (x^2 -1) (x-1) = 0

x^2013 (x+1) (x-1)^2 = 0
x = 0,-1,1

Jadi ada 3 bil.real

13. banyak bilangan real x yang memenuhi x^2016-x^2014=x^2015-x^2013 minta bantuanya ya


x^2016 - x^2014 = x^2015 - x^2013
(x^2016 - x^2014) - (x^2015 - x^2013) = 0
x^2016 - x^2015 - x^2014 + x^2013 = 0
x^2013 (x^3 - x^2 - x + 1) = 0
x^2013 (x^2 - 1)(x - 1) = 0
x^2013 (x + 1)(x - 1)(x - 1) = 0

bilangan real x yang memenuhi :

x^2013 = 0
x = 0

x + 1 = 0
x = -1

x - 1 = 0
x = 1

Jadi, ada 3 bilangan yang memenuhi, yaitu -1, 0, dan 1
