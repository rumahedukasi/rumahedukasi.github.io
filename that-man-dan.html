Penjelasan:

Kalimat di atas mengunakan present continuous tense. Penanda dari tense tersebut adalah frase "look at that" yang menunjukkan suatu aktivitas yang sedang terjadi.

36. Does that man drink a lot of beer (+) (-)​


Jawaban:

(+) that man drinks a lot of beer

(-) that man does not drink a lot of beer

Jawaban:

Does that man drink a lot of beer?

(+) that man drinks a lot of beer

(-) that man does not drink a lot of beer

Pembahasan

Simple Present Tense

Tenses yg digunakan utk menyatakan Kegiatan berulang/rutin, kebiasaan dan kebenaran umum/fakta.

Rumus Present Tense

Ada dua macam

1. Verbal Sentence: kalimat yg terdapat kata kerja (Verb) di dalam kalimatnya.

(+) S + V1 (-s/es) + O

(-) S + do/does + not + V1 + O

(?) do/does + S + V1 + O?

2. Nominal Sentence: Kalimat yg tidak ada kata kerja (Verb) di dalam kalimatnya.

(+) S + Tobe (is/am/are) + O

(-) S + Tobe (is/am/are) + not + O

(?) Tobe (is/am/are) + S + O?

Kalimat soal bentuk Interrogative (?)

Auxiliary => doesSubject => That manVerb 1 => drinkObject => a lot of beer?

Berdasarkan Structure di atas,

terdapat verb (drink) maka gunakan Rumus Verbal Sentence:

(+) S + V1 (-s/es) + O

That man + drinks + a lot of beer

(-) S + do/does + not + V1 + O

That man + does + not + drink + a lot of beer

(?) do/does + S + V1 + O?

does + that man + drink + a lot of beer?
