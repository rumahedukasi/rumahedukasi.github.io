1. perbandingan tabungan nova dan gabby adalah 4:7 selisih uang mereka adalah Rp 120.000,00 berapa banyak uang gabby?​


Gabby =
[tex] \frac{7}{3} \times 120.000 = 280.000[/tex]
uang gabby =

[tex] \frac{4}{(7 - 4)} \times \: 120.000[/tex]

[tex] \frac{7}{3} \times 120.000[/tex]

[tex] \frac{840.000}{3} = 280.000[/tex]

2. Perbandingan umur lisa dan umur gabby lima tahun yang lalu adalah 4:5.jika lima tahun lalu umur lisa 12 tahun. berapalah umur gabby sekarang​


PEMBAHASAN

Lima tahun yang lalu

Lisa : Gabby => 4 : 5

Lisa = 12 tahun

Menentukan umur Gabby 5 tahun yang lalu:

[tex] \frac{5}{4} \times 12 \: tahun[/tex]

[tex]5 \times 3 \: tahun[/tex]

[tex]15 \: tahun[/tex]

Menentukan umur Gabby sekarang:

[tex]15 \: tahun + 5 \: tahun[/tex]

[tex]20 \: tahun[/tex]

Jadi, umur Gabby sekarang adalah 20 tahun

CONDITIONAL SENTENCE

Answer

a. If you had come with us, you would have seen an awesome movie.

b. If she studies harder, she will pass the exam.

c. I will come if it doesn't rain.

d. If I were you, I would slap his face.

Explanation

Conditional sentence structure consists -If clause and result clause ( main clause ).

Types of Conditional Sentences :

ZERO CONDITIONALThe Zero Conditional is used to describe the Phenomenon, habit, and General truth ( Fact ).

Pattern :

If+S+verb ¹ (s/es)+O , S+verb ¹ (s/es)+O

Example :

If you put the empty aerosol spray on a bonfire, it explodes immediately.

Discussion :

It's general truth. It is proven that the empty aerosol spray explodes when we put it on a bonfire.

FIRST CONDITIONALThe First Conditional is used to describe the truth in the present/future ( The prediction/planning will be probably done. However, the speaker will actualize his/her dream now, soon or later.

Pattern :

If +S+ verb¹ (s/es) + O, S+will+verb¹+O

Example :

If she finishes her study, she will go abroad.

Discussion :It uses simple future form in its main clause / result clause to describe the possible condition and state the prediction will be true. It means that she will probably go abroad.

SECOND CONDITIONALThe second Conditional type is used to describe the untruth in the present / future. It is contrary to the fact in the present or it could be in the future. The speaker has an intention ( dream ) but he/she can't afford his/her dream comes true in the present/future (improbable)We must use tobe "were" for both, singular and plural subjects.

Pattern:

If+S+verb²+O/C , S+would/could+verb¹+O/C

Example :

If I were you, I would slap his face.

Discussion :

It uses the simple past form to reflect the untruth condition in the present and the speaker's intention will not come true because it is contrary to the real condition for now or maybe future. It means that I'm not you, so I can't slap his face.

THIRD CONDITIONALThe Third Conditional is used to describe the untruth in the past. It is contrary to the fact in the past and It reflects a regret for something that the speaker couldn't afford to do in the past.

Pattern :

If+S+Had+Verb³+O , S+Would/could+have+Verb³+O

Example :

If you had come with us, you would have seen an awesome movie.

Discussion :It reflects the regret for something that couldn't be reached in the past. It means that you didn't come, so you didn't see an awesome movie.

A DISCUSSION BASED ON THE DIALOG ABOVE :

a. If you had come with us, you would have seen an awesome movie.

See the fact :

Shinta didn't come to drew's house Shinta had to study.

So, we must use the third conditional, to describe the event we couldn't reach in the past

b. If she studies harder, she will pass the exam.

See the fact :

Ghina is not stupid But, too lazy to read the lesson material.However, there is a chance to fix it.

So, we must use the first conditional, to describe the probability.

c. I will come if it doesn't rain.

See the fact :

Jane accepted Ann's request But not sure, whether she will come or notHowever, Jane will probably come.

So, we must use the first conditional, to describe the probability.

d. If I were you, I would slap his face.

See the fact :

It's just Gabby's illusion to calm Jessica's heart down.Gabby is not Jessica

So, we must use the second conditional, to describe the improbability.
