7. Mouse dom touchpad pada laptoptidak berfungsi bersamaan, (laptop Lenovo)langkah yangharus dilakukan ?​


Jawaban:

Aktifkan touchpad yang disable

Pada beberapa laptop biasanya sudah dilengkapi tombol khusus di keyboard untuk mengaktifkan dan menonaktifkan touchpad. Jadi bisa saja tanpa sengaja kamu menekan tombol tersebut sehingga membuat touchpad tidak berfungsi. Untuk mengatasinya, carilah tombol di keyboard yang memiliki ikon bergambar touchpad. Biasanya tombol tersebut berada di antara tombol F1 – F12. Jika berhasil ketemu, kamu hanya perlu melakukan tekan kombinasi Fn + tombol tersebut untuk mengaktifkan kursor yang freeze, misalnya Fn + F6 atau Fn + F9.

Cara lainnya, kamu bisa mengaktifkan touchpad dari Mouse Setting menggunakan mouse eksternal. Untuk Windows 7 atau 8, dari menu Start, lalu ketik Mouse. Setelah jendela Mouse Setting terbuka, lalu pilih opsi Enable Touchpad, klik Apply, dan OK. Sementara pada Windows 10, klik tombol Windows, lalu masuk ke Setting dan klik Device. Setelah itu, pilih Mouse & Touchpad untuk masuk ke dalam jendela pengaturan touchpad. Lihat opsi di bagian bawah dan klik Additional Mouse Options untuk mengaktifkan touchpad.

10. Cara membuat @ pada laptop Lenovo ini bagai mana mohon bantuan secepatnya?keyboard tidak memiliki tombol shift!​


Jawaban:

Jika keyboard pada laptop Lenovo Anda tidak memiliki tombol "Shift", tetapi Anda ingin membuat simbol "@" (@), Anda bisa mencoba cara-cara berikut:

1. Menggunakan Karakter Khusus (Character Map):

  - Buka "Character Map" di komputer Anda. Anda dapat mencarinya di kotak pencarian Windows.

  - Cari simbol "@" pada daftar karakter.

  - Klik simbol "@" untuk memilihnya, kemudian klik "Copy" atau "Copy to Clipboard".

  - Setelah itu, Anda bisa "Paste" (Tempel) simbol "@" di tempat yang diinginkan, misalnya di alamat email atau dokumen.

2. Menggunakan Kombinasi Tuts Keyboard (Shortcut):

  Beberapa laptop Lenovo memungkinkan Anda untuk mengakses simbol "@" dengan kombinasi tuts keyboard tertentu. Cobalah kombinasi tuts berikut ini:

  - Tekan dan tahan tombol "Alt Gr" (berada di sisi kanan spasi).

  - Sambil menahan tombol "Alt Gr", tekan tuts "2" (biasanya berada di atas huruf "Q").

  - Lepaskan kedua tombol secara bersamaan, dan simbol "@" akan muncul.

Jika kedua cara di atas tidak berhasil, mungkin ada masalah dengan layout keyboard pada laptop Anda atau tombol "Shift" rusak. Dalam kasus ini, Anda mungkin perlu mencari solusi teknis lebih lanjut atau memperbaiki keyboard laptop Anda.

37. cara mengatasi laptop yang black screen gimana ya? please bantu ya.


1Masuk ke Safe Mode

Solusi awal yang bisa dilakukan sebagai cara mendapatkan Windows yang baik kembali adalah dengan masuk ke mode aman sistem  atau safe mode windows 7. Langkah untuk masuk ke Safe mode Windows 7 sendiri adalah sbb :

Nyalakan laptop;

Sebelum masuk tampilan logo Windows 7, tekan tombol F8, beberapa kali, hingga masuk menu Boot Option Windows 7;

Dari opsi yang ada, pilih Safe Mode;

Setelah tampil safe mode, tunggu sekitar 5-10 menit lalu restart laptop dan nyalakn kembali laptop, langkah ini pernah diuji dan memang jika masalahnya tidak parah bisa menjadikan laptop menyala normal kembali;

Biasanya masalah seperti ini diakibatkan oleh service Windows yang bermasalah atau timing service yang delay.


2. Instal ulang graphic (display) adapter

Fungsi dari graphic adapter sendiri adalah menampilkan output hasil processing ke layar monitor, dan rendering, meski sekilas tak ada kaitan langsung antara blank screen dengan kerja graphic adapter, tetapi sebenarnya masalah pada driver graphic bisa mengakibatkan sistem menjadi unstable dan menjadi penyebab blue screen  misalnya. Adapun untuk instal ulang graphic adapter langkahnya adalah sbb :

Nyalakan laptop;

Masuk ke safe mode seperti langkah diatas;

Tekan tombol keyboard CTRL + ALT + DELETE untuk memunculkan Task Manager;

Pada jendela taskmanager, tekan CTRL + N atau klik menu File > New kemudian ketik explorer.exe dan tekan Enter;
