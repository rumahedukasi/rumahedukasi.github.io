Bima = B

a - b = 5 --> b = a - 5

b² = 4a + 1

(a - 5)² = 4a + 1

a² - 10a + 25 = 4a + 1

a² - 10a - 4a + 25 - 1 = 0

a² - 14a + 24 = 0

(a - 12) (a - 2) = 0

a = 12 tahun atau a = 2 tahun,

Jika a = 2 tahun, b = -3 tahun. TIDAK MUNGKIN

Jika a = 12 tahun, b = 7 tahun MUNGKIN

True of False!

1. Intan asks Audy to buy a hammer.  --> FALSE

Correct one: Intan asks Audy to use a hammer

2. Audy is able to use hammer.  --> TRUE

Jawaban terbukti dengan pernyataan Audy, "I think I can use it just fine"

3. Audy expresses his willingness to help Intan.  --> TRUE

Audy menunjukkan ketersediannya untuk membantu dengan mengucapkan "All right, I'll do it."

4. Intan asks Audy to hang the picture on the wall.  --> TRUE

Jawaban terbukti terdapat pada percakapan Intan, "Will you hang this picture for me?"

5. Audy needs some hammers and pliers to hang the picture.  --> FALSE

Audy needs a hammer and some nails to hang the picture.

6. Audy places the picture upside down at first.  -->  FALSE

Tidak ada pernyataan yang membenarkan pernyataan tersebut.

7. Intan thinks that the picture is in a straight position . --> TRUE

Jawaban dapat ditemukan ketika Intan menjawab, "Yes, it is" atas pertanyaan "Is it straight yet?".

8. Audy asks for Intan's willingness to pass the hammers and nails to him.  --> FALSE

Bentuk permintaan di mana diawali dengan kata kerja dasar merupakan sebuah command (perintah). Maka pernyataan tersebut bukanlah menyatakan willingness, tetapi perintah (order/command).

9. Audy asks Intan to buy him some more nails.  --> FALSE

The correct one is Audy asks Intan to hand her some nails.

10. Audy expresses his gratitude to Intan in the end of the dialogue. --> FALSE

It is otherwise that Intan expresses his gratitude to Audy by saying, "Thank you, Audy."
