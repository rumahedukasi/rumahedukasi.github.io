12. Jelaskan fungsi dan beri contohnya ( masing masing 1 )! A. Menu homeB. Menu insertC. Menu page layoutD. Menu formulasE. Menu dataF. Menu reviewG. Menu viewTOLONG DIBANTU PLISS​


Jawaban:

A. Menu Home: Fungsi dari Menu Home adalah memberikan akses cepat ke perintah-perintah dasar dalam pengolahan dokumen atau file. Contoh perintah yang dapat ditemukan di Menu Home adalah Save (Simpan), Copy (Salin), Paste (Tempel), Undo (Urungkan), dan Format (Pemformatan teks).

B. Menu Insert: Fungsi dari Menu Insert adalah untuk menyisipkan elemen atau objek ke dalam dokumen atau file. Contoh perintah yang dapat ditemukan di Menu Insert adalah Insert Picture (Sisipkan Gambar), Insert Table (Sisipkan Tabel), Insert Chart (Sisipkan Grafik), Insert Hyperlink (Sisipkan Hyperlink), dan Insert Header/Footer (Sisipkan Header/Footer).

C. Menu Page Layout: Fungsi dari Menu Page Layout adalah untuk mengatur tata letak dan penampilan halaman dokumen atau file. Contoh perintah yang dapat ditemukan di Menu Page Layout adalah Page Orientation (Orientasi Halaman), Margins (Batasan), Page Size (Ukuran Halaman), Columns (Kolom), dan Themes (Tema).

D. Menu Formulas: Fungsi dari Menu Formulas adalah untuk melakukan perhitungan dan pemrosesan data menggunakan rumus dan fungsi matematika. Contoh perintah yang dapat ditemukan di Menu Formulas adalah AutoSum (Jumlah Otomatis), Average (Rata-rata), Sum (Jumlah), Count (Jumlah Data), dan IF (Jika).

E. Menu Data: Fungsi dari Menu Data adalah untuk mengelola dan memanipulasi data dalam dokumen atau file. Contoh perintah yang dapat ditemukan di Menu Data adalah Sort (Urutkan), Filter (Saring), PivotTable (Tabel Pivot), Data Validation (Validasi Data), dan Text to Columns (Teks ke Kolom).

F. Menu Review: Fungsi dari Menu Review adalah untuk melakukan pemeriksaan, revisi, dan komentar pada dokumen atau file. Contoh perintah yang dapat ditemukan di Menu Review adalah Spell Check (Pemeriksaan Ejaan), Track Changes (Lacak Perubahan), Comments (Komentar), Protect Document (Lindungi Dokumen), dan Compare (Bandingkan).

G. Menu View: Fungsi dari Menu View adalah untuk mengatur tampilan dan navigasi dalam dokumen atau file. Contoh perintah yang dapat ditemukan di Menu View adalah Print Layout (Tata Letak Cetak), Full Screen (Layar Penuh), Zoom (Perbesar/Perkecil), Ruler (Penggaris), dan Gridlines (Garis Kisi).

Misalkan pada program Ms. Excel 2013:

A. menu home berfungsi untuk menampung ikon Bold

B. menu insert berfungsi untuk menampung ikon Picture

C. menu page layout berfungsi untuk menampung ikon Orientation

D. menu formulas berfungsi untuk menampung ikon Insert Function

E. menu data berfungsi untuk menampung ikon Sort

F. menu review berfungsi untuk menampung ikon Protect Sheet

G. menu view berfungsi untuk menampung ikon Zoom

Contoh penggunaan:

A. Misalkan pengguna program Ms. Excel ingin menebalkan teks, maka harus klik menu HOME terlebih dahulu lalu klik ikon Bold

B. Misalkan pengguna program Ms. Excel ingin menyisipkan gambar, maka harus klik menu INSERT terlebih dahulu lalu klik Picture

C. Misalkan pengguna program Ms. Excel ingin mengatur orientasi lembar kerja, maka harus klik menu PAGE LAYOUT terlebih dahulu lalu klik Orientation

D. Misalkan pengguna program Ms. Excel ingin memakai rumus tertentu, maka bisa klik menu FORMULAS terlebih dahulu lalu klik ikon Insert Function

E. Misalkan pengguna program Ms. Excel ingin mengurutkan data, maka bisa klik menu DATA terlebih dahulu lalu klik ikon Sort

F. Misalkan pengguna program Ms. Excel ingin memproteksi worksheet, maka klik menu  REVIEW dulu, lalu klik Protect Sheet

G. Misalkan pengguna program Ms. Excel ingin mengatur besar/ kecil tampilan, maka klik dahulu menu VIEW dan klik ikon Zoom

Untuk menu REVIEW bisa diamati contoh tampilannya pada lampiran:

- menu REVIEW ditunjukkan tanda panah hijau

- ikon Protect Sheet ditunjukkan tanda panah biru

- kotak dialog Protect Sheet ditunjukkan tanda panah hitam
