17. 5. Sebuah warung nasi padang berbentuk persegi panjang mempunyai panjang (x + 4)m dan lebar (x - 4)m. Jika luas warung nasi padang adalah 240cm², maka tentukan :A. Nilai xB. Keliling warung nasi padang​


Jawab:

a) nilai x

(x - 4)(x + 4) = 240

x² - 16 = 240

x² = 256

x= 16

b.  K = 2(p+l)

K = 2[ (x + 4) + (x -  4)]

K=  2x

K=  32

Materi : Fungsi dan Persamaan Kuadrat

Dik :

Sebuah Persegi Panjang

Panjang = ( x + 4 ) m

Lebar = ( x - 4 ) m

Dit :

Nilai x dan Keliling

Dijawab :

( 1 ). Nilai x

Panjang × Lebar = Luas

( x + 4 )( x - 4 ) = 240

x² + 4x - 4x - 16 = 240

x² - 16 = 240

x² = 240 + 16

x = ± √256

x = ± 16

Himpunan penyelesaian :

x = -16 [ Tidak mungkin ] ×

x = 16 [ Sudah Pasti Benar ✅ ]

( 2 ) Keliling

[ Result Nilai x = 16 ]

Panjang = ( x + 4 ) = 16 + 4 = 20 m

Lebar = ( x - 4 ) = 16 - 4 = 12 m

Maka langsung saja :

Keliling = 2( Panjang + Lebar )

Keliling = 2( 20 m + 12 m )

Keliling = 2( 32 m )

Keliling = 64 m

24. naha aya bedana antara warung nasi , restoran , jeung kafe ?


Kategori Soal : Bahasa Sunda - Wirausaha
Kelas : VI (6 SD)
Pembahasan :

Warung nasi nyaeta tempat atawa warung saderhana anu nyadiakeun jeung ngajual kadaharan sangu lengkep jeung deungeunna. Warung nasi tempatna biasana muat pikeun sakitar lima urang nepi ka sapuluh urang. Warung nasi biasana buka boh beurang boh peuting. Jalma anu ka warung nasi miboga tujuan utama pikeun dahar wungkul. Jalma anu datang ka warung nasi boh anu ngora boh anu kolot. Harga di warung nasi biasana relatif murah.

Restoran nyaeta tempat anu leuwih alus jeung leuwih gede ti warung nasi anu nyadiakeun jeung ngajual kadaharan sangu lengkep jeung deungeunna. Restoran tempatna biasana muat pikeun sakitar leuwih ti sapuluh urang nepi ka saratus urang atawa leuwih. Restoran biasana buka boh beurang boh peuting. Jalma anu ka restoran miboga tujuan utama pikeun dahar wungkul. Jalma anu datang ka restoran boh anu ngora boh anu kolot. Harga di restoran biasana leuwih mahal ti batan di warung nasi.

Sedengkeun kafe nyaeta tempat anu tadina mah nyadiakeun jeung ngajual inuman saperti kopi jeung sajabana, jeung kadaharangan ringan, tapi kafe oge ayeuna mah nyadiakeun kadaharan sangu jeung kadaharan lianna. Kafe tempatna biasana muat pikeun sakitar leuwih ti sapuluh urang nepi ka saratus urang atawa leuwih.

Kafe biasana buka sore nepi ka peuting, tapi aya oge kafe anu buka beurang jeung peuting. Jalma anu ka kafe miboga tujuan utama pikeun nginum, dahar jeung ngawangkong santai, sabab biasana suasanan di kafe mah cocog pikeun santai jeung ngawangkong. Jalma anu datang ka kafe lolobana barudak ngora. Biasana di kafe mah sok dibarengan ku musik. Harga di kafe biasana sarua jeung harga di restoran.
