2. Suatu kelompok Data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8 kelompok data B memiliki rata-rata dan median 8 jika data terkecil dari kelompok data A adalah P dan kelompok data B sama dengan kelompok data A tanpa P tentu nilai P ?


Diketahui:

- Kelompok Data A terdiri dari 5 data

- Rata-rata kelompok Data A adalah 66

- Median kelompok Data A adalah 8

- Kelompok Data B memiliki rata-rata dan median yang sama dengan kelompok Data A tanpa data terkecilnya (P)

Ditanya:

- Nilai P (data terkecil dari kelompok Data A)

Langkah Penyelesaian:

1. Untuk mencari nilai P kita perlu mengetahui nilai awal sebelum data terkecil dihapus dari kelompok Data A.

2. Rata-rata kelompok Data A adalah 66 jadi total dari 5 data tersebut adalah 5 x 66 = 33.

3. Median kelompok Data A adalah 8 artinya data tengah adalah 8.

4. Karena median adalah data tengah maka data ke-3 dan ke-4 harus memiliki nilai total yang sama dengan median atau 8.

5. Jadi total Data A sebelum data terkecil dihapus adalah 8 + 8 + x (data ke-3) + y (data ke-4) + z (data ke-5) = 33.

6. Diketahui bahwa x + y + z = 17 (rata-rata tanpa data terkecil) berdasarkan rata-rata kelompok Data A sebelum dihapus data terkecil.

7. Kita juga tahu bahwa x + y + z + P = 33 (rata-rata sebelum dihapus data terkecil).

8. Dengan mensubstitusikan z = 17 - x - y ke dalam persamaan x + y + z + P = 33 kita dapatkan x + y + (17 - x - y) + P = 33.

9. Ini dapat disederhanakan menjadi 17 + P = 33 sehingga P = 33 - 17 = 16.

Jadi nilai P (data terkecil dari kelompok Data A) adalah 16.

8. Sekelompok data memiliki nilai rata-rata 60. Jika koefisien variasinya 20% maka standar deviasi dari kelompok data tersebut adalah ….


Sekelompok data memiliki nilai rata - rata 60. Jika koefisien variasinya 20%, maka

standar deviasi dari kelompok data tersebut adalah 12.

Koefisien variasi adalah perbandingan antara simpangan standar dengan nilai rata - rata yang dinyatakan dalam bentuk persentase.

Fungsi dari koefisien variasi adalah untuk melihat sebaran data dari rata - rata hitungnya.

Besarnya koefisien variasi dihitung dengan rumus :

Koefisien variasi = (simpangan standar / rata - rata) × 100%

Dari rumus itu juga, kita dapat menentukan simpangan stardar dan rata - rata data dengan cara :

• Simpangan standar = koefisien variasi × rata - rata

• Rata - rata = simpangan standar ÷ koefisien variasi

Untuk lebih jelasnya, kita ulas soal di atas dalam pembahasan berikut.

PEMBAHASAN :

Sekelompok data memiliki nilai rata - rata 60. Jika koefisien variasinya 20%, maka standar deviasinya dapat dihitung dengan :

Simpangan standar = koefisien variasi × rata - rata

= 20/100 × 60

= 12

9. Suatu kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8. Kelompok data B memiliki rata-rata dan median 8. Jika data terkecil dari kelompok data A adalah p dan kelompok data B = kelompok data A tanpa p, tentukan nilai p!​


Jawaban:

Untuk menentukan nilai p, kita perlu memahami hubungan antara rata-rata, median, dan data dalam kelompok A dan B.

Dalam kelompok data A, dengan 5 data, median adalah 8. Median adalah nilai tengah saat data diurutkan dari terkecil hingga terbesar. Dalam hal ini, median adalah data ke-3 dari data yang diurutkan.

Jika median adalah 8 dan kita memiliki 5 data, maka data ke-3 adalah 8.

Jadi, kita dapat menulis data dalam kelompok A sebagai berikut:

p, ?, 8, ?, ?

Selanjutnya, kita diketahui bahwa rata-rata kelompok data A adalah 6,6. Rata-rata diperoleh dengan menjumlahkan semua data dan membaginya dengan jumlah data.

Jadi, kita dapat menulis persamaan berikut berdasarkan rata-rata:

(p + ? + 8 + ? + ?)/5 = 6,6

Selanjutnya, kita diketahui bahwa kelompok data B memiliki rata-rata dan median yang sama dengan 8. Jadi, dalam kelompok data B tanpa data p, kita akan memiliki data dengan median 8 dan rata-rata 8.

Dalam hal ini, karena hanya ada 4 data dalam kelompok data B, data ke-2 dan ke-3 (tanpa data p) harus menjadi 8.

Jadi, kita dapat menulis kelompok data B sebagai berikut:

?, 8, 8, ?

Sekarang, kita perlu menyelesaikan persamaan untuk nilai p dalam kelompok data A.

Dari persamaan rata-rata:

(p + ? + 8 + ? + ?)/5 = 6,6

Jumlahkan semua data yang diketahui:

p + 8 + ? + ? + ? = 6,6 × 5

p + 8 + ? + ? + ? = 33

Dalam kelompok data B tanpa data p, data ke-2 dan ke-3 harus menjadi 8, sehingga persamaan dapat ditulis sebagai berikut:

8 + 8 + ? + ? = 33

16 + 2? = 33

2? = 33 - 16

2? = 17

? = 17/2

? = 8,5

Jadi, nilai p adalah 8,5.


10. Terdapat dua buah kelompok data sebagai berikut: data kelompok pertama yaitu 2,,,3,4,6 mempunyai nilai rata-rata , data kelompok kedua 2,,,4,6,2,1 mempunyai rata-rata 2. Nilai adalah ….


Berikut adalah data dua buah kelompok yaitu:

Data kelompok pertama: 2, a, a, 3, 4, 6 mempunyai nilai rata-rata c.Data kelompok kedua: 2, c, c, 4, 6, 2, 1 mempunyai rata-rata 2a dan median c.

Data yang memiliki nilai median yang lebih besar adalah data kelompok kedua yaitu 3.

Soal lengkapnya dapat dilihat di lampiran.

Penjelasan dengan langkah-langkah

Rata-rata adalah perbandingan antara jumlah data dengan banyaknya data.

[tex]\bar{x} = \frac{jumlah \: data}{banyaknya \: data}[/tex]

Median adalah nilai tengah suatu data yang telah diurutkan dari terkecil ke terbesar.

Diketahui

Data kelompok pertama: 2, a, a, 3, 4, 6 mempunyai nilai rata-rata c.Data kelompok kedua: 2, c, c, 4, 6, 2, 1 mempunyai rata-rata 2a dan median c.

Ditanyakan

Tentukan manakah data yang memiliki median terbesar!

Jawab

Langkah 1

Data kelompok pertama: 2, a, a, 3, 4, 6 mempunyai nilai rata-rata c.

[tex]\bar{x} = \frac{jumlah \: data}{banyaknya \: data}[/tex]

[tex]c = \frac{2 \:+\:a \:+\: a \:+\: 3 \:+\: 4 \:+\: 6}{6}[/tex]

[tex]c = \frac{2a \:+\: 15}{6}[/tex]

6c = 2a + 15

6c – 2a = 15 ……. persamaan (1)

Langkah 2

Data kelompok kedua: 2, c, c, 4, 6, 2, 1 mempunyai rata-rata 2a dan median c.

[tex]\bar{x} = \frac{jumlah \: data}{banyaknya \: data}[/tex]

[tex]2a = \frac{2 \:+\:c \:+\: c \:+\: 4 \:+\: 6 \:+\: 2 \:+\: 1}{7}[/tex]

[tex]2a = \frac{2c \:+\: 15}{7}[/tex]

7(2a) = 2c + 15

14a – 2c = 15

–2c + 14a = 15 …… persamaan (2)

Langkah 3

Eliminasi persamaan (1) dan (2).

6c – 2a = 15   |×7| 42c – 14a = 105

–2c + 14a = 15 |×1| –2c + 14a = 15

                              --------------------- +

                              40c          = 120

                                           c = 3

Langkah 4

Substitusi c = 3 ke persamaan (1).

6c – 2a = 15

6(3) – 2a = 15

18 – 2a = 15

–2a = 15 – 18

–2a = –3

a = 1,5

Langkah 5

Substitusi a = 1,5 ke data kelompok pertama yaitu:

2, a, a, 3, 4, 6

2; 1,5; 1.5; 3; 4; 6

Kita urutkan dari terkecil ke terbesar yaitu:

1,5     1,5     2  |   3     4     6

Mediannya berada diantara 2 dan 3 yaitu:

Median = ½ (2 + 3)

             = ½ (5)

             = 2,5

Langkah 6

Median data kelompok pertama adalah 2,5.Median data kelompok kedua adalah c = 3.

Jadi nilai median yang paling besar adalah median pada data kelompok kedua.

12. Suatu kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8. Kelompok data B memiliki rata-rata dan median 8. Jika data terkecil dari kelompok data A adalah p dan kelompok data B = kelompok data A tanpa p, tentukan nilai p!​


Diketahui:

- Kelompok data A terdiri dari 5 data

- Kelompok data A memiliki nilai rata-rata 66

- Kelompok data A memiliki median 8

- Kelompok data B memiliki nilai rata-rata 8

- Kelompok data B memiliki median 8

Ditanya:

- Nilai p data terkecil dari kelompok data A

Langkah penyelesaian:

1. Nilai median adalah data tengah saat data diurutkan dari terkecil hingga terbesar.

2. Karena pada kelompok data A median adalah 8 dan kelompok data B juga memiliki median 8 berarti nilai ke-3 dan ke-4 dari kelompok data A juga 8.

3. Jumlah dari semua data kelompok data A adalah 5 * rata-rata = 5 * 66 = 33

4. Jumlah 4 data yang bernilai 8 adalah 8 + 8 + 8 + 8 = 32.

5. Jadi nilai data terkecil dari kelompok data A adalah 33 - 32 = 1.

Jadi nilai p adalah 1.

18. Suatu kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6.6 dan median 8 Kelompok data B memiliki rata-rata dan median 8. Jika data terkecil dari kelompok data A adalah p dan kelompok data B = kelompok data A tanpa p, tentukan nilai p​


Diberikan bahwa kelompok data A terdiri dari 5 data dengan rata-rata 6.6 dan median 8. Jadi, nilai p adalah 1.

Penjelasan dengan langkah-langkah:

Karena median kelompok data A adalah 8, berarti nilai tengah dari kelompok data A adalah 8. Oleh karena itu, salah satu kemungkinan distribusi data A adalah sebagai berikut:

p, x, 8, y, z

Karena median adalah 8, maka x, y, dan z harus lebih besar atau sama dengan 8. Jumlah data adalah 5, sehingga:

p + x + 8 + y + z = 5 * 6.6 ...(1) (rata-rata kelompok data A adalah 6.6)

Diketahui bahwa rata-rata kelompok data B juga adalah 8. Karena kelompok data B merupakan kelompok data A tanpa p, maka jumlah data kelompok data B adalah 4. Jadi persamaan untuk kelompok data B adalah:

x + 8 + y + z = 4 * 8 ...(2) (rata-rata kelompok data B adalah 8)

Dengan menggabungkan persamaan (1) dan (2), kita dapat mencari nilai p:

p + x + 8 + y + z = 33 ...(1)

x + 8 + y + z = 32 ...(2)

Substitusikan (x + 8 + y + z) pada persamaan (2) dengan 32 ke persamaan (1):

p + 32 = 33

p = 33 - 32

p = 1

Jumlah total data dalam kelompok data A adalah 5, sehingga nilai total seluruh data dalam kelompok tersebut adalah 5 * 6,6 = 33.

Median yang merupakan nilai tengah dalam urutan data adalah 8. Dalam hal ini, data ketiga dan data keempat dalam urutan data A memiliki nilai 8.

x2, 8, 8, x5

Nilai p adalah 1. Untuk mencari nilai p, kita perlu mengetahui data terkecil dari kelompok data A.

Penjelasan dengan langkah-langkah:

Diketahui:

Kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8.

Kelompok data B memiliki rata-rata dan median 8.

Jika data terkecil dari kelompok data A adalah p

kelompok data B = kelompok data A tanpa p,

Ditanya:

Tentukan nilai p!

Jawab:

Untuk mencari nilai p, kita perlu mengetahui data terkecil dari kelompok data A.

Kita tahu bahwa median dari kelompok data A adalah 8, yang berarti data ke-3 (diurutkan dari terkecil ke terbesar) adalah 8. Karena kelompok data A terdiri dari 5 data, maka data ke-3 adalah nilai tengah.

Jadi, kita dapat menuliskan kelompok data A sebagai berikut (diurutkan dari terkecil ke terbesar):

a, b, 8, c, d

Kita juga tahu bahwa rata-rata kelompok data A adalah 6,6. Kita dapat menghitung jumlah total dari kelompok data A dan mencari nilai dari a, b, c, dan d.

Jumlah total kelompok data A dapat dihitung dengan mengalikan rata-rata dengan jumlah data:

6,6 * 5 = 33

Jumlah total dari kelompok data A adalah 33. Dengan mengganti variabel dengan nilai yang diketahui, kita dapat menuliskan persamaan sebagai berikut:

a + b + 8 + c + d = 33

Kita tahu bahwa kelompok data B memiliki rata-rata dan median yang sama dengan kelompok data A, yaitu 8. Dengan menghilangkan p dari kelompok data A, kita dapat menuliskan kelompok data B sebagai berikut:

a, b, 8, c, d (tanpa p)

Karena kelompok data B memiliki 4 data, maka rata-ratanya dapat dihitung dengan mengalikan nilai median dengan jumlah data:

8 * 4 = 32

Jumlah total dari kelompok data B adalah 32. Dengan mengganti variabel dengan nilai yang diketahui, kita dapat menuliskan persamaan sebagai berikut:

a + b + 8 + c + d (tanpa p) = 32

Karena kita mencari nilai p, kita dapat mengurangi persamaan pertama dengan persamaan kedua untuk mencari nilainya:

(a + b + 8 + c + d) - (a + b + 8 + c + d (tanpa p)) = 33 - 32

Pada kedua sisi persamaan, a, b, 8, c, dan d akan saling menghilang. Oleh karena itu, persamaan tersebut dapat disederhanakan menjadi:

p = 33 - 32

p = 1

Jadi, nilai p adalah 1.

34. Suatu kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8. Kelompok data B memiliki rata-rata dan median 8 , jika data terkecil dari kelompok data A adalah p dan kelompok data B = kelompok data A tanpa p, tentukan nilai p!


Jawab:

Penjelasan dengan langkah-langkah:

Untuk menentukan nilai p, kita perlu menggunakan informasi tentang rata-rata dan median dari kedua kelompok data.

Diberikan bahwa kelompok data A terdiri dari 5 data dengan rata-rata 6,6 dan median 8.

Kita dapat menghitung total nilai pada kelompok data A dengan menggunakan rumus rata-rata: total nilai = rata-rata × jumlah data.

Total nilai pada kelompok data A = 6,6 × 5 = 33.

Diketahui median kelompok data A adalah 8, yang berarti data tengah adalah 8.

Karena data tengah adalah 8 dan ada 5 data secara keseluruhan, maka ada 2 data yang lebih kecil dari 8 dan 2 data yang lebih besar dari 8.

Jadi, jumlah kedua data yang lebih kecil dari 8 adalah 2.

Kita juga diberitahu bahwa kelompok data B memiliki rata-rata dan median 8, dengan kelompok data B sama dengan kelompok data A tanpa p.

Jika kelompok data B adalah kelompok data A tanpa p, maka jumlah data pada kelompok data B adalah 5 - 1 = 4.

Kita dapat menghitung total nilai pada kelompok data B dengan menggunakan rumus rata-rata: total nilai = rata-rata × jumlah data.

Total nilai pada kelompok data B = 8 × 4 = 32.

Karena total nilai pada kelompok data A adalah 33 dan total nilai pada kelompok data B adalah 32, maka nilai p adalah selisih antara total nilai kelompok data A dan kelompok data B.

p = Total nilai kelompok data A - Total nilai kelompok data B = 33 - 32 = 1.

Jadi, nilai p adalah 1.

40. Suatu kelompok data A yang terdiri dari 5 data memiliki nilai rata-rata 6,6 dan median 8. Kelompok data B memiliki rata-rata dan median 8. Jika data terkecil dari kelompok data A adalah p dan kelompok data B = kelompok data A tanpa p, tentukan nilai p!


Nilai p dari data tersebut adalah 1. Sedangkan nilai ∑x dari data A adalah 33.

Penjelasan dengan langkah-langkah

Diketahui:

x̄ data A = 6,6

Median data A = 8

n data A = 5

x̄ data B = 8

Median data B = 8

nilai terkecil data A = p

Data B = data A - p

Ditanyakan:

Berapakah nilai p.?

Jawaban

∑x data A = x̄ × n = 6,6 × 5 = 33

x̄ data B = 8

{(∑x data A) - p } / (n data A - 1) = 8

(33 - p ) / (5-1) = 8

(33 - p)/4 = 8

33 - p = 8 × 4

33 - p = 32

- p = 32 - 33

-p = -1

p = 1
