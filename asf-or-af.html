7. This is a..(g). It comes from..(Af). It has a long..(n) and two..(h). It lives in..(sav) or jungle. It eats..(g) or green..(l)


DESCRIPTIVE TEXT-RIDDLE

Answer :

This is a girrafe. It comes from Africa. It has a long neck and two horns. It lives in savana or jungle. It eats grass or green leaves.

Translation

Ini adalah jerapah. Dia berasal dari Afrika. Dia punya leher yg panjang dan dua tanduk. Dia tinggal di padang rumput atau hutan. Dia makan rumput atau dedaunan hijau.

EXPLANATIONDescriptive text is a text which describes people/place/animal and other things specifically.

GENERIC STRUCTURE  

1.  Identification  

Identification is the very beginning part of paragraph. It tells about the background, the introduction of who, what,  which will be described in the text.

2. Description

Description contains the quality which belongs to the things you are talking about.  

LANGUAGE FEATURES

1. It uses the specific participants, to identify the things.

Example : giraffe

2. It uses the adjective much more, to explain the noun in detail.

Example : green, long

3. It uses the simple present tense, it tells the fact or general truth

Example :

It comes from Africa

It has a long neck and two horns.

It lives in savana or jungle.

It eats grass or green leaves.

4. It uses the action verbs, to reveal the activity

Example : eat, live

18. jika AF = x dan AF = 1/3 AB,maka keliling bangun tersebut adalah


Keliling bangun tersebut adalah 10x. JAWABAN A. Simak penjelasan berikut!

Bangun ABCDEF pada gambar

AF = 1/3 AB

Ditanya

Keliling bangun tersebut

Penyelesaian

AF = 1/3 AB

x = 1/3 AB

AB = 3x

Keliling adalah jumlah seluruh garis tepi.

Maka, keliling bangun tersebut

K = x + x + 3x + x + x + 3x

K = 4x + 6x

K = 10x

Kesimpulan

Jadi, keliling bangun tersebut adalah 10x.
