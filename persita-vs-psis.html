16. Tabungan irima 30.000 lebih banyak dari sita. Amira mempunyai tabungan empat kali dari tabungan sita tabungan juli setengah dari tabungan sita. Sementara tabungan sita sekarang 80.000. Maka, jumlah tabungan keempat anak tersebut adalah?


Jawaban:

sita 80.000

irima 110.000

amira 320.000

Juli 40.000

Jawab:

terdapat 4 orang :
irima, sita, amira, juli

diumpamakan :
irima = a
sita = b
amira = c
juli = d

tabungan irima 30.000 lebih banyak dari sita
a = 30.000 + b

amira mempunyai tabungan 4 kali dari tabungan sita
c = 4b

tabungan juli setengah dari tabungan sita
d = 1/2b

jika tabungan sita = 80.000

tabungan sita = b = 80.000
d = 1/2b = 40.000

c = 4b = 4*80.000 = 320.000

a = 30.000 + b = 30.000 + 80.000 = 110.000

maka keempat tabungan tersebut adalah :
irima = 110.000
sita = 80.000
amira = 320.000
juli = 40.000

jumlah = 110.000 + 80.000 + 320.000 + 40.000 = 550.000

27. jumlah uang sita dan fauzia besarnya 100.000 jika perbandingan uang sita dan fauzia 3:2 selisih uang sita adalah​


Sita : Fauzia = 3 : 2

Sita + Fauzia = 100.000

Uang sita

= 3/(3 + 2) × 100.000

= 3/5 × 100.000

= 60.000

Uang Fauzia

= 2/(3 + 2) × 100.000

= 2/5 × 100.000

= 40.000

Selisih = 60.000 - 40.000 = 20.000
