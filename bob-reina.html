36. Ali :...... you do all the test' bob ? bob : Bob yes' I did.​


~ Answer:did

Ali: did you do all the test, Bob?

Bob: yes, I did.

.

~ Discussion:

Halo dan salam kenal. Agar dapat kita melengkapi percakapan Ali di atas, kita perlu memerhatikan dialognya ya.

Mari kita perhatikan pernyataan Bob sebagai petunjuk pertama.

》Bob: Yes, I did.

Adanya kata did merupakan auxiliary verb untuk menunjukkan pekerjaan yang dilakukan di waktu sebelumnya. Memiliki tense berupa simple past.

Ali mengajukan pertanyaan. Sehingga auxiliary verb terletak di awal kalimat. Agar sesuai dengan jawaban Bob, nyatakan dengan auxiliary untuk simple past yaitu did.

JAWABANNYA:

Did you do all the test, Bob?

2. Bob wrote some letters. Some letters are written by Bob.

3. Bob will write some letters. Some letters will be written by Bob.

4. Bob is writing some letters. Some letters are being written by Bob.

5. Bob was writing some letters. Some letters were being written by Bob.

6. Bob is going to write some letters. Some letters are going to be written by Bob.

7. Bob has written some letters. Some letters have been written by Bob.

9. Bob has to write some letters. Some letters have to be written by Bob.

10. Bob must write some letters. Some letters must be written by Bob.

11. Is Bob writing some letters? Are some letters being written by Bob?

12. Did Bob write some letters? Were some letters written by Bob?

Mengubah kata kerja yang ada menjadi verb 3 atau past participle. Menambahkan "be" di depan verb 3 tersebut. Ingat, kalimat pasif berarti subject berubah menjadi object.
