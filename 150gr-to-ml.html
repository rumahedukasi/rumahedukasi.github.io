4. 40gr air bersuhu 100°C dicampur dengan 150gr air bersuhu 10°c. Jika kalor sejenis air 4200j/kg°c hitunglah suhu campuran


Dik:
m₁ = 40 g = 0,04 kg
T₁ = 100°C
m₂ = 150 g = 0,15 kg
T₂ = 10°C
c = 4200 J / (Kg * °C)

Dit:
Suhu campuran, T = ?

Jawab
kalor lepas = kalor terima
Q lepas= Q terima
m₁ * c * ΔT₁ = m₂ * c * ΔT₂
karena pada kedua ruas memiliki c yang sama maka c dapat dihilangkan
m₁ * ΔT₁ = m₂ * ΔT₂
0,04 * (100 - T) = 0,15 * (T - 10)
4 - 0,04*T = 0,15T - 1,5
4 + 1,5 = 0,15*T + 0,04*T
5,5 = 0,19*T
T = 5,5 / 0,19 = 28,95°C

atau dengan cara cepat
T = (m₁ * T₁ + m₂ * T₂) / (m₁ + m₂)
T = (40 * 100 + 150 * 10) / (40 + 150) = 28,95°C

v2'-v1' / v1 - v2 = 1
v2' - v1' = v1 - v2
v2' - v1' = 20 - 0
v2' - v1' = 20
v2' = 20 + v1' ---> subtitusikan ke pers 1

0,15 v1' + 0,1 v2' = 3
0,15 v1' + 0,1 (20 +v1') = 3
0,15 v1' + 2 + 0,1 v1' = 3
0,25 v1' = 1
v1' = 4 m/s

v2' = 20 + v1'
v2' = 20 + 4
v2' = 24 m/s

14. 1. 4 buah partikel masing-masing masanya 100gr, 150gr, 200gr,250gr terletak pada masing-masing sudut bujur sangkar. sisi bujur sangkar 50 cm. tentukanlah momen inersial total di pusat bujur sangkar? mohon bantuannya ^_^


Perhatikan gambar pada lampiran.

Pusat bujur sangkar = titik O
Titik O = Titik temu kedua diagonal bidang

Diagonal bidang = AC dan BD
Diagonal bidang = s√2
Diagonal bidang = 50√2 cm

Jarak A, B, C dan D ke O = ½ Diagonal bidang
Jarak A, B, C dan D ke O = ½ . 50√2
Jarak A, B, C dan D ke O = 25√2 cm
Jarak A, B, C dan D ke O = 0.25√2 m

Massa partikel :
mA = 100 gr = 0.1 kg
mB = 150 gr = 0.15 kg
mC = 200 gr = 0.2 kg
mD = 250 gr = 0.25 kg

Momen inersia di titik O :
IO = IA + IB + IC + ID

IO = mA . rA² + mB . rB² + mC . rC² + mD . rD

IO = 0.1 . (0.25√2)² + 0.15 . (0.25√2)² + 0.2 . (0.25√2)² + 0.25 . (0.25√2)²

IO = (0.1 + 0.15 + 0.2 + 0.25) . (0.25√2)²

IO = 0.7 . 0.125

IO = 0.0875 kgm²

Jadi, besar momen inersia di pusat bujur sangkar adalah 0.0875 kgm²

17. diantara gas gas berikut,yang mempunyai volume4,48liter pada keadaan standar adalah......a.28gr N2 b.14,2gr Cl2 c.52 gr C2H2 d.120gr SO2 e.150gr No....mohon dijawab yah sama cara x..plisss​


diantara gas gas berikut,yang mempunyai volume4,48liter pada keadaan standar adalah

b. 14,2 gram Cl₂

Pembahasan

[tex]\boxed{\boxed{\bold{Stoikiometri}}}[/tex]  

Dalam bahasa Yunani, kata stoicheion berarti unsur sedangkan stoikiometri (stoichiometry) berarti mengukur unsur meliputi massa atom dan massa rumus, rumus kimia, persamaan kimia serta semua yang berhubungan dengan reaksi kimia. Dalam stoikiometri, satuan yang digunakan untuk menyatakan jumlah partikel dalam zat dinamakan mol.  

[tex]\mathbf{Cara~Menentukan~Jumlah~Mol~(n) }[/tex]  

1. Menentukan dengan jumlah partikel  

Standar yang digunakan untuk menyatakan 1 mol adalah jumlah partikel dalam isotop atom C-12 (karbon) bermassa 12 gram.  

12 gram C-12 mengandung [tex]6,02.10^{23}[/tex] partikel / atom / molekul / ion  

Jumlah partikel tersebut dikenal dengan bilangan Avogadro (L).  

bilangan Avogadro (L) = [tex]6,02.10^{23}[/tex] partikel / atom / molekul / ion  

[tex]\boxed{\boxed{\bold{n~=~\frac{Jumlah~Ion/Molekul/Atom/Partikel}{L} }}}[/tex]

2. Menentukan jumlah mol dengan volume  

a. Jika pengukuran dilakukan pada suhu 0oC dan tekanan 1 atm (STP (Standard Temperature and Pressure)), maka  

[tex]\boxed{\boxed{\bold{n~=~\frac{Vm}{22,4} }}}[/tex]

dengan, Vm = volume molar standar  

b. Menggunakan hipotesis Avogadro  

Jika diukur pada suhu dan tekanan yang sama maka gas-gas yang memiliki volume sama akan memiliki jumlah molekul yang sama pula.  

[tex]\boxed{\boxed{\bold{V_1~:~V_2~=~n_1~:~n_2}}}[/tex]  

c. Jika pengukuran dilakukukan tidak pada suhu 0oC dan tekanan 1 atm, menggunakan hukum gas ideal :  

PV = nRT  

[tex]\boxed{\boxed{\bold{n~=~\frac{P~x~V}{R~x~T} }}}[/tex]

dengan:  

P = tekanan (atmosfer)  

T = suhu mutlak (Kelvin), K = °C + 273  

V = volume (liter)  

n = jumlah mol (mol)  

R = tetapan gas ideal yang nilainya 0,082 L atm mol⁻¹ K⁻¹  

3. Menentukan dengan massa molekul relatif  

[tex]\boxed{\boxed{\bold{n~=~\frac{gram}{Mr} }}}[/tex]

dengan,  

n = jumlah mol (mol)  

gram = massa (gram)  

Mr = massa molekul relatif  

4. Menentukan dengan konsentrasi  

[tex]\boxed{\boxed{\bold{n~=~M~x~V}}}[/tex]  

dengan,  

M = konsentrasi (M)  

V = volume (Liter)  

[tex]\boxed{\boxed{\bold{Jawab}}}[/tex]  

Jika pengukuran dilakukan pada keadaan standar STP (Standard Temperature and Pressure)), maka  

[tex]\boxed{\boxed{\bold{n~=~\frac{Vm}{22,4} }}}[/tex]

dengan, Vm = volume molar standar  

maka,

n gas = [tex]\frac{V}{22,4}[/tex]

n gas = [tex]\frac{4,48}{22,4}[/tex]

n gas = 0,2 mol

a. 28 gram N₂

Ar N = 14

Mr N₂ = 2. Ar N = 2. 14 = 28

n N₂ = [tex]\frac{massa}{Mr}[/tex]

n N₂ = [tex]\frac{28}{28}[/tex]

n N₂ = 1 mol

b. 14,2 gram Cl₂

Ar Cl = 35,5

Mr Cl₂ = 2. Ar Cl = 2. 35,5 = 71

n Cl₂ = [tex]\frac{massa}{Mr}[/tex]

n Cl₂ = [tex]\frac{14,2}{71}[/tex]

n Cl₂ = 0,2 mol

BENAR

c. 52 gram C₂H₂

Ar C = 12

Ar H = 1

Mr C₂H₂ = 2. Ar C + 2. Ar H = 2. 12 + 2. 1 = 26

n C₂H₂ = [tex]\frac{massa}{Mr}[/tex]

n C₂H₂ = [tex]\frac{52}{26}[/tex]

n C₂H₂ = 2 mol

d. 120 gram SO₂

Ar S = 32

Ar O = 16

Mr SO₂ = Ar S + 2. Ar O = 32 + 2. 16 = 64

n SO₂ =  [tex]\frac{massa}{Mr}[/tex]

n SO₂ =  [tex]\frac{120}{64}[/tex]

n SO₂ = 1,875 mol

e. 150 gram NO

Ar N = 14

Ar O = 16

Mr NO = Ar N + Ar O = 14 + 16 = 30

n NO = [tex]\frac{massa}{Mr}[/tex]

n NO = [tex]\frac{150}{30}[/tex]

n NO = 5 mol

31. 1. 32gram suatu elektrolit biner yang berada dalam 150gr air ternyata mendidih pada suhu 100,18°C.Jika elektrolit ini terion 65% (Kb air = 0,5°C/m, Kf air = 1,8°C/m), tentukan: a. Mr elektrolitb. Titik beku larutanc. Tekanan osmotik larutan pada suhu 127°C dan massa jenis larutan 0,642 gr/mLd. Tekanan uap jika pada suhu 127 °C tekanan uap air adalah 80 mmHg​


Sifat koligatif merupakan sifat larutan yang bergantung pada jumlah zat terlarut dan tidak tergantung pada jenis zat. Zat terlarut hanya dikelompokkan antara zat terlarut elektrolit dan zat terlarut non elektrolit. Sifat koligatif larutan terdiri atas penurunan tekanan uap, penurunan titik beku, kenaikan titik didih, dan tekanan osmotik.

Pembahasan

Adanya zat terlarut dalam suatu pelarut cair mengakibatkan penurunan tekanan uap jenuh. Hal ini sesuai dengan Hukum Raoult yang menyatakan "besarnya tekanan uap larutan sebanding dengan fraksi mol pelarut dalam larutan". Semakin banyak zat terlarut dalam larutan, semakin sedikit bagian pelarutnya sehingga tekanan uap juga semakin rendah. Tekanan uap jwnuh larutan dirumuskan sebagai berikut :

P = Xp · P°

P = tekanan uap jenuh larutan (cmHg)

Xp = fraksi mol zat pelarut, dirumuskan dengan np / (nt + np)

P° = tekanan uap jenuh air murni

39. sebuah bola bermassa 150gr dilemparkan mendatar ke arah sumbu X positif dengan kelajuan 10m/s. lalu bola itu terpantul berbalik arah dengan kelajuan 16m/s hitung gaya rata rata yg dikerjakan alat pemantul tersebut jika bola menyentuh pemantyl dalam waktu 0.02 detik


m = 150 gr = 0,15 kg
v₁ = -10 m/s
v₂ = 16 m/s
Δt = 0,02 s
F = ............?

Penyelesaian :
F = ΔP/Δt
F = (P₂ - P₁)/0,02
F = m(v₂ - v₁)/0,02
F = 0,15 (16 - (-10))/0,02
F = 0,15 (16 + 10)/0,02
F = (0,15 x 26)/0,02
F = 195 N

Jadi,besar gaya rata ratanya adalah 195 N
