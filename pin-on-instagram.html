Jawaban:

Perlu diketahui bahwa Pin pada Windows 7 fungsinya untuk menempelkan sebuah program yang sering kita gunakan ke Taskbar atau ke Start Menu, tujuannya untuk memudahkan pengguna ketika ingin membuka program tersebut kembali cukup dengan mengklik icon Shortcut Program

Pin to start menu : untuk membuat shortcut di bagian start menu.

Penjelasan:

Contohnya ada shortcut di desktop, maka bisa diklik kanan mouse lalu pilih (klik) Pin to taskbar atau klik Pin to start menu.

Misalkan di Windows 10, bisa melihat contohnya di gambar berikut ini:

- Pin to Start menu ditunjukkan tanda panah merah

- Pin to taskbar ditunjukkan tanda panah biru

- Start menu bisa dilihat dg mengklik tombol Start (dalam lingkaran kuning)

- Taskbar bisa dilihat di bagian dalam kotak kuning

24. Jumlah pin Rendi dan Fajar 72 buah. Perbandingan pin Rendi dan Fajar 5 7. Selisih pin mereka adalah ....​


Jawaban:

jumlah pin Rendi adalah 5x dan jumlah pin Fajar adalah 7x. Jadi, kita memiliki persamaan:

5x + 7x = 72

Kombinasikan suku-suku yang sama:

12x = 72

Bagi kedua sisi dengan 12:

x = 6

Sekarang kita dapat mencari jumlah pin Rendi dan Fajar:

Jumlah pin Rendi = 5x = 5 * 6 = 30

Jumlah pin Fajar = 7x = 7 * 6 = 42

Selisih pin mereka adalah:

42 - 30 = 12

Jadi, selisih pin Rendi dan Fajar adalah 12 buah
