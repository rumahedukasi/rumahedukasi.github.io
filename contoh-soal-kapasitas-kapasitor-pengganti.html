1. dua kapasitor dengan kapasitas sama disusun seri. kapasitas penggantinya adalah


C₁ = C₂ = C

1/Cs = 1/C₁ + 1/C₂
Cs = C₁C₂ / (C₁ + C₂)
      = C·C / (C + C)
      = C² / (2C) = 1/2 C

Cara super cepat bila semuanya identik:
Cs = C / n = 1/2 C

2. Dua kapasitor dengan kapasitas sama disusun seri. kapasitas penggantinya adalah...


1/Cs = 1/C1 + 1/C2

1/Cs = 1/C + 1/C

1/Cs = 2/C

Cs = C/2

         = 2F + 3F + 6F

4. Tiga buah kapasitor dengan kapasitas 3μF dirangkai secara seri (lihat gambar). Besar kapasitas kapasitor pengganti adalah.. μF​


KAPASITOR

C₁ = C₂ = C₃ = C = 3 μF

n = 3

Cₜₒₜₐₗ = ... ?

Cₜₒₜₐₗ = C / n

Cₜₒₜₐₗ = 3 / 3

Cₜₒₜₐₗ = 1 μF ✔


5. Dua buah kapasitor masing masing memiliki kapasitas 2µF dan 4µF disusun seri. kapasitor penggantinya.....


resistor kalo di seri kan mudah ya, R total = R1 + R2

Kapasitor kebalikannya kalo di paralel C total = C1 + C2

kalo kapasitor di seri [tex]\frac{1}{C total} = \frac{1}{C1} + \frac{1}{C2} [/tex]


untuk soal yang ditanyakan, kasusnya kapasitor di paralel jadi

[tex]C total = C1 + C2 = 2uF+ 4uF = 6uF [/tex]

10. tentukan kapasitas pengganti untuk susunan kapasitor pada gambar berikut ​


Jawaban:

*Penyelesaian*

Hitung dulu yang paralel antara 2 μF dan 4 μF

Cp1 = (2 + 4) μF

Cp1 = 6 μF

Terus hitung 2 kapasitor yang seri di antara paralel

1/Cs1 = 1/3 + 1/Cp1

1/Cs1 = 1/3 + 1/6

1/Cs1 = 1/2

Cs1 = 2 μF

1/Cs2 = 1/6 + 1/12

1/Cs2 = 1/4

Cs2 = 4 μF

Setelah itu, hitung paralel antara Cs1 dan Cs2

Cp2 = Cs1 + Cs2

Cp2 = (2 + 4) μF

Cp2 = 6 μF

Hitung rangkaian kapasitor seri antara Cp2 dan 11 μF

1/Cs3 = 1/Cp2 + 1/11

1/Cs3 = 1/6 + 1/11

1/Cs3 = (11 + 6)/66

1/Cs3 = 17/66

Cs3 = 66/17

Jadi, kapasitas pengganti untuk susunan kapasitor pada gambar di atas adalah 66/17 μF
