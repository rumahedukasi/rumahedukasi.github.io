1. In a shop, there were 3 kinds of cakes: sponge cakes, butter cakes and chocolate cakes. 32 % of the cakes were sponge cakes. The ratio of butter cakes to chocolate cakes was 11: 6. There were 96 more sponge cakes than chocolate cakes. Find the total number of cakes. ​


Let x be the total number of cakes in the shop.

We know that:

32/100 * x = number of sponge cakes (1)

number of butter cakes / number of chocolate cakes = 11/6 (2)

number of sponge cakes - number of chocolate cakes = 96 (3)

From equation (1) we know that number of sponge cakes is 0.32x.

From equation (3) we know that number of sponge cakes - number of chocolate cakes = 96. So number of chocolate cakes = number of sponge cakes - 96.

We can substitute this value into equation (2)

number of butter cakes / (number of sponge cakes - 96) = 11/6

We can substitute (1) and (3) into this equation

(11/6) * (0.32x - 96) = 0.32x

multiply both sides by 6/11

0.32x - 96 = 0.32x * (6/11)

0.32x - 96 = 0.18x

0.14x = 96

x = 688

So there are 688 cakes in total in the shop.

29. Mrs Wong sold some cakes to Shops A, B and C. Shop A bought 20% of the cakes. Shop B bought 4 times as many cakes as Shop C. Shop B bought 88 more cakes than Shop A. How many cakes did Mrs Wong sell altogether​


Mrs. Wong sold 200 cakes altogether.

More Explanation :

Ratio is a relationship of two quantities, usually expressed as quotient of one divided by the other.

The ratio is usually expressed as a fraction.

To be known :

Shop A, Shop B, and Shop C

Shop A bought 20% of the cakes

Shop B bought 4 times as many cakes as Shop C

Shop B bought 88 more cakes than Shop A

To be asked :

Total cakes sold altogether = . . . . ?

To be answered :

Shop A = 20%

So that Shop B + Shop C = 60%

Shop B = 88 + Shop A

Shop B = 4 times Shop C, so that B = 4C

A = 20%

B + C = 80%

4C + C = 80%

5C = 80%

C = [tex]\tt \dfrac {80}{5}[/tex]

C = 16%

B = 4 times C

B = 4 × 16%

B = 64%

B = 88 + A

64% [tex]\tt x[/tex] = 88 + 20% [tex]\tt x[/tex]

88 = 64% [tex]\tt x[/tex] - 20% [tex]\tt x[/tex]

88 = 44% [tex]\tt x[/tex]

x = [tex]\tt \dfrac {88 \times 100}{44}[/tex]

x = 200

Conclusion :

Mrs. Wong sold 200 cakes altogether.
