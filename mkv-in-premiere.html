1. H.264/MPEG-4 AVC MKV

2. Xvid MKV

3. DivX MKV

4. VP9 MKV

5. H.265/HEVC MKV

6. AV1 MKV

7. Matroska 3D MKV

8. Matroska Audio MKV

Setiap jenis MKV memiliki karakteristik dan keunggulan tersendiri tergantung dari codec dan format yang digunakan.

3. pengertian dari video mkv adalah​


MKV (Matroska Video) adalah format file video yang digunakan untuk menyimpan video, audio, dan subtitle dalam satu file. Format ini dikembangkan sebagai alternatif untuk format file video lain seperti AVI, MP4, dan MOV, dan digunakan secara luas di seluruh dunia untuk menonton film, video musik, dan konten multimedia lainnya. Keuntungan dari format MKV adalah dapat menyimpan banyak jenis file audio dan subtitle dalam satu file video, serta memiliki kualitas video yang sangat baik dengan ukuran file yang relatif kecil.

7. sebutkan tujuan penggunaan mkv​


Jawaban:

Tujuan penggunaan MKV adalah untuk menyediakan solusi kontainer video yang lebih baik dan fleksibel daripada format file video tradisional seperti AVI, MP4, dan MPEG.

Jawaban:

Konsep MKV adalah open source yang membuatnya cukup berbeda dengan format file lain khususnya MP4 dan AVI. MKV juga mempunyai sejumlah fitur yang membuatnya unik dibanding format sejenis lainnya.

1. Mempunyai fitur pencarian cepat yang ada di dalam file.

2. Mempunyai chapter atau entri episode.

3. Memiliki dukungan ful metadata.

4. Bagian subtitle, video maupun audio bisa dipilih.

5. Dapat dikembangkan sendiri secara mudah.

6. Bisa dijalankan walaupun sebagian data mengalami kerusakan (recovery).

7. Dapat dijalankan secara streaming baik menggunakan jaringan internet lokal maupun wifi.

8. Memiliki menu seperti DVD.

Tujuan utamanya mungkin karena, mkv populer untuk pembuatan film yang mana dalam format tersebut dapat menyimpan sejumlah track audio dan subtitle hanya dalam satu bentuk file saja. Format ini juga bisa digunakan untuk semua jenis codec. jadi lebih mempermudah

9. bagaimana cara menginstal adobe premiere dari CD ?


1. Masukkan CD Installer Adobe Premiere Pro ke dalam drive CD‐ROM. Lalu masuk ke CD/DVD Drive.
2. Double klik SETUP.
3. Piliih bahasa yang akan digunakan dalam proses instalasi, pada pilihan select the Language, biarkan pilihan pada US English, kemudian tekan tombol OK.

4. Klik dialog End User License Agreement akan ditampilkan. Klik tombol Accept untuk melanjutkan proses instalasi.
5. Setelah akan munculkotak dialog pilihan instalasi. Klik Install Adobe Premier Pro untuk menginstalisasi adobe premier pro.

6. Selanjutnya akan tampil Adobe Premier Pro Setup, kotak dialog ini menginformasikan bahwa Adobe Premier Pro akan diinstal. Klik tombol Next. 
7. Setelah itu akan tampil kotak dialog Select Language. Pilih bahasa US English yang akan digunakan pada proses instalasi,kemudian klik tombol Next untuk melanjutkan.
8. Selanjutnya akan tampil kotak dialog Software License Agrement, klik tombol Yes.

9. Kotak dialog User Information akan ditampikan. Isi Title, First Name, Last (family) name, Company dan Serial Nunber. Setela terisisemua, klik tombol Next untuk melanjutkan. 
10. Kemudian tampil kotak dialog Choose DetinationLocation. Pilih tempat instalasi untuk file Adobe Premier Pro. Klik tombol Browse untuk memilih tempat instalasi. Apabila setuju dengan penyimpanan default, maka bisa langsung klik tombol Next untuk melanjutkan.

11. Setelah itu akan tampil kotak dialog Start copying File. Kemudian klik Next untuk melanjutkan.
12. Program instalasi akan di kopi seluruhnya. Proses ini ditandai degan progress bar. Tunggu hingga proses instalasi selesai.

13. Setelah selesai akan ditampilkan kotak dialog yangmemberikan 2 pilihan yaitu, Yes I want to restart my computer now dan No, I will restart my computer later. Pilih pilihan pertama dan klik tombol Finish unyuk mengakhiri proses instalasi. Komputer akan direstart secara otomatis.

26. sebutkan mekanisme mkv​


Jawaban:

MKV (Matroska Video) adalah format file multimedia yang digunakan untuk menyimpan video, audio, dan subtitle dalam satu file. Mekanisme MKV termasuk:

Penggabungan file: MKV memungkinkan penggabungan beberapa file menjadi satu file MKV, sehingga memudahkan pengguna untuk menyimpan dan memutar file multimedia yang terdiri dari beberapa file.Kompatibilitas: MKV didukung oleh banyak perangkat dan program pemutar multimedia, seperti VLC, Windows Media Player, dan banyak lainnya. Ini membuat MKV menjadi format file multimedia yang lebih fleksibel dan mudah diakses.Multiple audio dan subtitle tracks: MKV memungkinkan penyimpanan multiple audio tracks dan subtitle tracks dalam satu file, sehingga memudahkan pengguna untuk memilih bahasa yang diinginkan ketika memutar video.Compression: MKV menggunakan teknologi kompresi yang efisien untuk mengurangi ukuran file, sehingga memudahkan pengguna untuk mengunggah atau membagikan file multimedia yang besar.Kualitas video yang tinggi: MKV memungkinkan penyimpanan video dengan kualitas yang sangat tinggi, seperti video 4K, 8K, dan lain-lain.Menyimpan metadata: MKV juga dapat menyimpan metadata, seperti judul, sutradara, tahun rilis, dan lain-lain, yang memudahkan pengguna untuk mengatur dan mencari file multimedia.

28. jelaskan mengapa video mkv tanpa subtitle​


Jawaban:

Ada beberapa alasan mengapa sebuah video dalam format MKV tidak memiliki subtitle. Berikut adalah beberapa kemungkinan alasan:

Video tersebut memang tidak disertai subtitle: Bisa jadi video tersebut memang tidak disertai dengan subtitle ketika dibuat. Ini mungkin terjadi jika video tersebut dibuat oleh pengguna biasa dan tidak memasukkan subtitle ke dalamnya.

Subtitle terpisah dalam format yang berbeda: Beberapa kali subtitle dalam format MKV disimpan dalam file terpisah dalam format yang berbeda, seperti SRT atau ASS. Jika itu kasusnya, Anda mungkin perlu mencari file subtitle terpisah dan menambahkannya ke video menggunakan program atau aplikasi tertentu.

Masalah dengan pengaturan pemutar video: Ada kemungkinan bahwa masalah dengan pengaturan pemutar video Anda dapat menyebabkan subtitle tidak muncul pada video MKV. Pastikan bahwa pengaturan pemutar video Anda sudah benar dan menampilkan subtitle jika tersedia.

Tidak terdeteksi oleh pemutar video: Beberapa pemutar video mungkin tidak dapat mendeteksi subtitle yang disertakan dalam format MKV. Anda dapat mencoba menggunakan pemutar video yang berbeda untuk melihat apakah itu bisa menampilkan subtitle.

Ada masalah dengan file MKV itu sendiri: Terkadang file video dalam format MKV dapat rusak atau memiliki masalah lain yang menyebabkan subtitle tidak muncul. Jika itu terjadi, Anda mungkin perlu mencoba memutar file video yang sama di pemutar video yang berbeda atau mencoba memperbaiki file MKV menggunakan program atau aplikasi tertentu.

29. bagusan video format mp4 atau mkv?



 MP4 vs MKV, suport pemutar media

Dalam soal suport Mkv lebih unggul daripada Mp4. Mkv mempunyai konten yang lebih fleksibel dan serba bisa , sedangkan mp4 kontsinernya terbatas dan membutuhkan banyak modivikasi (modivikasi di software ecodingnya)



 MP4 vs MKV, subtitle 

dalam bidang subtitle Mp4 dan Mkv sama-sama unggul, keduanya memiliki suport yang baik pada soft  maupun hardsub pada sebuah subtitel, keduanya memiliki kekuatan yang seimbang dalam subtitle.



 MP4 vs MKV, lisensi

untuk ketentuan lisensi ke dua format vidio ini juga sama ungulnya, keduanya meggratiskan lisensi, hal ini membuat mereka sama kuatnya dibidang lisensi, namum pada  Mp4  tidak mampu menampung multiplay subtitle  audio dan vidio sedangkan pada mkv sebaliknya, mkv mampu menampung multiplay subtitle audio dan vidio


kesimpulan dari  MP4 vs MKV adalah
Mkv lebih baik sebagi format pilihan vidio dari pada mp4 karena Mkv memiliki suport yang lebih banyak dan juga  mampu menampung multiplay suptitele audio maupun vidio yang tidak bisa dilakukan oleh MP4.  jadi mkv sangat recoment bagi para downloder filem maupun anime meskipun kualitas vidionya tetap sama baiknya .
