8. 1. People can use this device easily. The passive form of that sentence is ... * a. this device can to be used easily b. this device can to be easily used c. this device can be used easily d. easily people can use this device e. easily this device can use


Jawaban:

1. People can use this device easily. The passive form of that sentence is ... *

a. this device can to be used easily

b. this device can to be easily used

c. this device can be used easily

d. easily people can use this device

e. easily this device can use

Penjelasan:

Jawaban dicetak tebal adalah poin c. this device can be used easily dan menggunakan bentuk Passive Voice dari kalimat yang menggunakan Modal Verb (Can). Pilihan lainnya salah.

Modal Verb (Can):

Passive Voice: (+) Subject + Can + Be + Verb-3 + Object

Semoga membantu ya.

c. This device can be used easily.

Pendahuluan Passive Voice adalah sebuah kalimat yang subjeknya sebelum predikat.

Passive Voice merupakan sebuah kalimat yang subjeknya dikenai perbuatan atau pun aktivitas.

Kalimat dari Passive Voice biasanya selalu berawalan oleh ter- atau di-.

Ciri - ciri Passive Voice, yaitu:Subjeknya sebagai penderita.Predikatnya berimbunan di- atau ter-.Predikatnya berupa predikat personal.

Penyelesaian

1. "People can use this device easily".

The passive form of that sentence is ...

a. this device can to be used easily

b. this device can to be easily used

c. this device can be used easily

d. easily people can use this device

e. easily this device can use

Passive Voice: This device can be used easily

Jadi, jawabannya adalah bagian C. This device can be used easily.

Pembahasan

Pertanyaan ini merupakan Passive Voice With Modals dengan rumus:

Modal + Be + V3

Kesimpulan

Passive Voice Modals merupakan sebuah kalimat Passive yang didalamnya terdapat berupa Modal.
