Cetak 1 foto: 1 x Harga cetak 1 foto

18. Di foto 1 ada contoh nya di foto yang ke dua adalah tugas nya


Terdapat empat persamaan kuadrat. Himpunan penyelesaian (HP) dari persamaan kuadrat:

x²-4x = 0 adalah {0,4}.x²-8x+16 = 0 adalah {4}.x²+6x-5 = 0 adalah {-3+√14,-3-√14}.2x²-3x-20 = 0 adalah {-2,5;4}.

Caranya dengan melengkapkan kuadrat sempurna dapat disimak pada penjelasan di bawah.

Penjelasan dengan langkah-langkah

Diketahui:

a. x²-4x = 0

b. x²-8x+16 = 0

c. x²+6x-5 = 0

d. 2x²-3x-20 = 0

Ditanya: penyelesaian dengan cara melengkapkan kuadrat sempurna

Jawab:

Untuk poin a:

Identifikasi persamaan kuadratKoefisien x²: a = 1Koefisien x: b = -4Konstanta: c = 0Nilai kuadrat dari setengahnya b (misalkan k)

k = (b/2)² = (-4/2)² = (-2)² = 4

Tambahkan k ke dua ruas persamaan

x²-4x+4 = 0+4

(x-2)² = 4

x-2 = ±√4

x-2 = ±2

x = ±2+2

x₁ = 2+2 = 4

x₂ = -2+2 = 0

HP

Tuliskan semua solusi ke dalam notasi himpunan. Jadi, HP persamaan kuadrat ini adalah {0,4}.

Untuk poin b:

Identifikasi persamaan kuadratKoefisien x²: a = 1Koefisien x: b = -8Konstanta: c = 16Nilai kuadrat dari setengahnya b (misalkan k)

k = (b/2)² = (-8/2)² = (-4)² = 16

Tambahkan k ke dua ruas persamaan

x²-8x+16+16 = 0+16

x²-8x+16 = 16-16

(x-4)² = 0

x-4 = ±√0

x-4 = 0

x = 4

HP

Tuliskan semua solusi ke dalam notasi himpunan. Jadi, HP persamaan kuadrat ini adalah {4}.

Untuk poin c:

Identifikasi persamaan kuadratKoefisien x²: a = 1Koefisien x: b = 6Konstanta: c = -5Nilai kuadrat dari setengahnya b (misalkan k)

k = (b/2)² =  (6/2)² = 3² = 9

Tambahkan k ke dua ruas persamaan

x²+6x-5+9 = 9

x²+6x+9 = 9+5

(x+3)² = 14

x+3 = ±√14

x = -3±√14

x₁ = -3+√14

x₂ = -3-√14

HP

Tuliskan semua solusi ke dalam notasi himpunan. Jadi, HP persamaan kuadrat ini adalah {-3+√14,-3-√14}.

Untuk poin d:

Identifikasi persamaan kuadrat

Pertama, ubah persamaan kuadrat agar koefisien x²-nya bernilai satu.

2x²-3x-20 = 0

2(x²-1,5x-10) = 0

x²-1,5x-10 = 0

Koefisien x²: a = 1Koefisien x: b = -1,5Konstanta: c = -10Nilai kuadrat dari setengahnya b (misalkan k)

k = (b/2)² = (-1,5/2)² = (-0,75)² = 0,5625

Tambahkan k ke dua ruas persamaan

x²-1,5x-10+0,5625 = 0+0,5625

x²-1,5x+0,5625 = 0,5625+10

(x-0,75)² = 10,5625

x-0,75 = ±√10,5625

x-0,75 = ±3,25

x = 0,75±3,25

x₁ = 0,75+3,25 = 4

x₂ = 0,75-3,25 = -2,5

HP

Tuliskan semua solusi ke dalam notasi himpunan. Jadi, HP persamaan kuadrat ini adalah {-2,5;4}.

28. Sebuah gedung yang tingginya 30 meter difoto dalam ukuran kecil dengan skala 1:500, kemudian foto tersebut diperbesar dengan skala 3:1, maka tinggi gedung di foto terakhir adalah


Skala adalah perbandingan antara ukuran pada denah (gambar) dengan ukuran sebenarnya. Skala dengan perbandingan 1 : k artinya setiap 1 cm pada gambar mewakili k cm ukuran sebenarnya.


Pembahasan

Diketahui

Tinggi gedung sebenarnya = 30 m = 3.000 cm

Gedung tersebut di foto dengan skala = 1 : 500

Ukuran Foto diperbesar dengan skala = 3 : 1


Ditanyakan

Tinggi gedung pada foto setelah diperbesar = ... ?


Jawab


Skala foto = 1 : 500


Misal

tinggi gedung pada foto = t

Tinggi gedung sebenarnya = T

Skala = [tex]\frac{t}{T}[/tex]

[tex]\frac{1}{500}[/tex] = [tex]\frac{t}{3.000\:cm}[/tex]

500 × t = 3.000 cm

t = (3.000 cm)/500
