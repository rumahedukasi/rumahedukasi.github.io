7. Diketahui 89- 28 V1o = √m - √n, Nilai m-n rm adalah​


Jawaban:

Untuk mencari nilai dari m - n, kita akan memulai dengan menyusun persamaan yang diberikan:

89 - 28 * V10 = √m - √n

Kemudian, kita akan mencoba untuk menyederhanakan persamaan ini dengan mengisolasi √m - √n:

√m - √n = 89 - 28 * V10

Sekarang, kita akan mengkuadratkan kedua sisi persamaan untuk menghilangkan akar kuadrat:

(√m - √n)² = (89 - 28 * V10)²

m - 2√mn + n = 89² - 2 * 89 * 28 * V10 + (28 * V10)²

Kita tahu bahwa m - n = (m - 2√mn + n) karena n akan dibatalkan. Jadi:

m - n = 89² - 2 * 89 * 28 * V10 + (28 * V10)²

Sekarang, mari kita hitung nilai dari persamaan di atas:

m - n = 89² - 2 * 89 * 28 * V10 + 28² * 10

m - n = 7921 - 49952√10 + 7840 * 10

m - n = 7921 - 49952√10 + 78400

Jadi, nilai dari m - n adalah 7921 - 49952√10 + 78400.

Penjelasan dengan langkah-langkah:

Untuk menyelesaikan persamaan ini, kita perlu melakukan beberapa langkah. Mari kita analisis satu per satu.

Langkah 1: Vereksa persamaan yang diberikan.

89 - 28 * V1o = √m - √n

Langkah 2: Kuadratkan kedua sisi persamaan untuk menghilangkan akar kuadrat.

(89 - 28 * V1o)^2 = (√m - √n)^2

Langkah 3: Perhatikan bahwa persamaan (√m - √n)^2 dapat digunakan dengan rumus a^2 - 2ab + b^2 = (a - b)^2.

Kita bisa menulisnya sebagai (√m - √n)^2 = m - 2√mn + n.

Langkah 4: Gantikan hasil perubahan persamaan di langkah 3 ke persamaan di langkah 2.

(89 - 28 * V1o)^2 = m - 2√mn + n

Langkah 5: Gantikan nilai m - n dengan x.

(89 - 28 * V1o)^2 = x

Langkah 6: Sederhanakan persamaan.

7921 - 4976V1o + 784V1o^2 = x

Kesimpulan:

Persamaan yang diberikan, 89 - 28 * V1o = √m - √n, telah berubah menjadi persamaan kuadrat 7921 - 4976V1o + 784V1o^2 = x, di mana x mewakili nilai m - n.
