3. Apa kelebihan dan kelemahan metode simpson?


Kelebihan Mode Simpson

1. Mode Simpson memiliki tingkat akurasi yang tinggi.

2. Mode Simpson dapat digunakan untuk menghitung integral yang kompleks.

3. Mode Simpson dapat digunakan untuk menghitung integral yang memiliki banyak titik.

4. Mode Simpson dapat digunakan untuk menghitung integral yang memiliki banyak titik dengan cepat.

Kekurangan Mode Simpson

1. Mode Simpson tidak dapat digunakan untuk menghitung integral yang memiliki titik yang berbeda.

2. Mode Simpson tidak dapat digunakan untuk menghitung integral yang memiliki titik yang berbeda dengan akurasi yang tinggi.

3. Mode Simpson tidak dapat digunakan untuk menghitung integral yang memiliki titik yang berbeda dengan cepat.

4. Mode Simpson tidak dapat digunakan untuk menghitung integral yang memiliki titik yang berbeda dengan tingkat akurasi yang tinggi.

17. Diketahui f(x)=x3(100-x4)1/2, 0 ≤ x ≤ 3 dan h =0,25. Hitunglah intergral berikut dan bandingkan dengan nilai integral secara analitik dan hitung galatnya! a. dengan metode simpson 1/3


Jawaban:

1.Diberikan fungsi f(x) = x^3 * (100 - x4)(1/2) pada interval [0, 3] dengan lebar h = 0.25. Kita perlu memisahkan interval ini menjadi subinterval dengan lebar h.

Pada kasus ini, interval [0, 3] terdiri dari 12 subinterval dengan lebar h = 0.25. Titik-titik seluruh subinterval adalah sebagai berikut:

x0 = 0, x1 = 0.25, x2 = 0.5, x3 = 0.75, x4 = 1, x5 = 1.25, x6 = 1.5, x7 = 1.75, x8 = 2, x9 = 2.25, x10 = 2.5, x11 = 2.75, x12 = 3.

2.Hitung nilai fungsi f(xi) pada setiap titik xi yang sudah ditentukan.

f(x0) = f(0) = (0^3 * (100 - 04))(1/2) = 0

f(x1) = f(0.25) = (0.25^3 * (100 - 0.254))(1/2) = 4.9902

f(x2) = f(0.5) = (0.5^3 * (100 - 0.54))(1/2) = 6.0299

…

f(x11) = f(2.75) = (2.75^3 * (100 - 2.754))(1/2) = 14.3301

f(x12) = f(3) = (3^3 * (100 - 34))(1/2) = 0

Hitung nilai fungsi f(xi) pada setiap titik xi untuk seluruh nilai xi yang sudah tercantum di atas.

3.Hitung nilai integral menggunakan rumusan Simpson 1/3.

Integral = (h/3) * [f(x0) + 4f(x1) + 2f(x2) + 4f(x3) + 2f(x4) + … + 2f(x11) + 4f(x12) + f(x13)]

Di sini, h = 0.25 adalah lebar subinterval, sedangkan f(xi) adalah nilai fungsi pada tiap titik xi yang sudah dihitung pada langkah sebelumnya. Dalam rumus ini, kita mengalikan dengan faktor 4 pada setiap subinterval yang berindeks genap, faktor 2 pada setiap subinterval yang berindeks ganjil, dan faktor 1 pada ujung interval.

Masukkan nilai-nilai tersebut ke dalam rumus dan hitung integral secara numerik.

Setelah proses ini selesai, kita dapat membandingkan nilai integral yang diperoleh dengan nilai integral secara analitik (jika diketahui) untuk mencari galatnya. Dalam kasus ini, kita perlu mengetahui persamaan dan metode yang tepat untuk menghitung nilai integral secara analitik dan menghitung galatnya.

40. Hitunglah luas daerah yang dibatasi dengan x = 4 dan x = 6 pada kurva Y= 2+ x2 dengan menggunakan integral biasa dan aturan simpson 1.


Untuk menghitung luas daerah yang dibatasi oleh kurva Y = 2 + x^2 antara x = 4 dan x = 6, kita dapat menggunakan integral biasa dan aturan Simpson 1.

1. Menggunakan Integral Biasa:

Untuk menghitung luas daerah dengan integral biasa, kita perlu mengintegrasikan fungsi Y = 2 + x^2 dari x = 4 hingga x = 6.

L = ∫[4 to 6] (2 + x^2) dx

L = [2x + (x^3 / 3)] |[4 to 6]

L = [(2 * 6 + (6^3 / 3)) - (2 * 4 + (4^3 / 3))]

L = (12 + 72/3) - (8 + 64/3)

L = 12 + 24 - 8 - 21.33

L = 6.67

Jadi, luas daerah yang dibatasi oleh kurva Y = 2 + x^2 antara x = 4 dan x = 6 adalah 6.67 satuan luas.

2. Menggunakan Aturan Simpson 1:

Aturan Simpson 1 menggunakan pendekatan polinomial orde 2 untuk menghitung luas daerah.

L = (h / 3) * [y_0 + 4 * y_1 + y_2]

Di sini, kita perlu membagi interval [4 to 6] menjadi beberapa subinterval dan menghitung nilai fungsi Y = 2 + x^2 pada titik-titik subinterval (y_0, y_1, y_2) dengan langkah h = (x_2 - x_0) / 2.

x_0 = 4, x_1 = 5, x_2 = 6

y_0 = 2 + (4^2) = 18

y_1 = 2 + (5^2) = 27

y_2 = 2 + (6^2) = 38

L = (2 / 3) * [(18) + 4 * (27) + (38)]

L = (2 / 3) * (18 + 108 + 38)

L = (2 / 3) * (164)

L = 218.67 / 3

L = 72.89

Jadi, luas daerah yang dibatasi oleh kurva Y = 2 + x^2 antara x = 4 dan x = 6 menggunakan aturan Simpson 1 adalah 72.89 satuan luas.
