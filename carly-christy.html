2x + Rp 180.000,00 = Rp 330.000,00
                           2x  = Rp 330.000,00 - Rp 180.000,00
                           2x  = Rp 150.000,00

22. Tiga orang pelajar Anna,Bobby dan christy menimbang berat badan secara berpasangan.Berat badan Anna dan Bobby 100Kg,dan Bobby dan Christy 120Kg,serta Anna dan Christy 110Kg. jika ketiga pelajar tersebut ditimbang secara bersamaan ,maka berat badan mereka adalah


Berat badan Anna dan Bobby 100 kg, berat Bobby dan Christy 120 kg dan berat badan Anna dan Christy 110 kg, maka berat badan ketiganya adalah 165 kg. Caranya adalah dengan menjumlahkan berat badan - berat badan tersebut, kemudian dibagi dengan dua. Permasalahan diatas merupakan penerapan sistem persamaan linear tiga variabel.


Pembahasan

Misal

A = berat badan Anna

B = berat badan Bobby

C = berat badan Christy


Berat badan Anna dan Bobby 100 kg

A + B = 100 ....... persamaan (1)

Berat badan Bobby dan Christy 120 kg

B + C = 120 ....... persamaan (2)

Berat badan Anna dan Christy 110 kg

A + C = 110 ....... persamaan (3)


Ketiga persamaan kita jumlahkan maka diperoleh

A + B = 100

B + C = 120

A + C = 110

------------------- +

2A + 2B + 2C = 330

-------------------------------- (kedua ruas bagi 2)

A + B + C = 165

Jadi berat badan mereka bertiga adalah 165 kg



Jika kita akan mencari masing-masing berat badannya, maka kita gunakan metode substitusi

Dari persamaan (1) diperoleh

A + B = 100

A = 100 - B

Dari persamaan (2) diperoleh

B + C = 120

C = 120 - B

Substitusikan A = 100 - B dan C = 120 - B ke persamaan (3)

A + C = 110

(100 - B) + (120 - B) = 110

220 - 2B = 110

-2B = -110

B = 55


A = 100 - B = 100 - 55 = 45

C = 120 - B = 120 - 55 = 65


Jadi

berat badan Anna = 45 kgberat badan Bobby = 55 kgberat badan Christy = 65 kg

Jumlah berat badan mereka bertiga

= 45 kg + 55 kg + 65 kg

= 165 kg
