7. Buktikan identitas :Sec x - Cosec x / sec x + cosec x = tan x - 1 / tanx + 1​


Jawab:

Pembuktian Identitas Trigonometri

Identitas trigonometri yang akan dibuktikan adalah:

[tex]\boxed{\ \dfrac{\sec x-\csc x}{\sec x+\csc x}=\dfrac{\tan x-1}{\tan x+1}\ }[/tex]

Pembuktian dari ruas kiri

[tex]\begin{aligned}\Biggl.&\textsf{Ruas kiri}=\frac{\sec x-\csc x}{\sec x+\csc x}\\&{=\ }\frac{\ \dfrac{1}{\cos x}-\dfrac{1}{\sin x}\ }{\dfrac{1}{\cos x}+\dfrac{1}{\sin x}}\times\frac{\sin x}{\sin x}\\\\&{=\ }\frac{\ \dfrac{\sin x}{\cos x}-\dfrac{\cancel{\sin x}}{\cancel{\sin x}}\ }{\dfrac{\sin x}{\cos x}+\dfrac{\cancel{\sin x}}{\cancel{\sin x}}}\\\\&{=\ }\frac{\tan x-1}{\tan x+1}=\textsf{Ruas kanan}\end{aligned}[/tex]

Pembuktian dari ruas kanan

[tex]\begin{aligned}\Biggl.&\textsf{Ruas kanan}=\frac{\tan x-1}{\tan x+1}\\&{=\ }\frac{\ \dfrac{\sin x}{\cos x}-1\ }{\dfrac{\sin x}{\cos x}+1}\times\frac{\dfrac{1}{\sin x}}{\dfrac{1}{\sin x}}\\\\&{=\ }\frac{\ \dfrac{\cancel{\sin x}}{\cancel{\sin x}\cos x}-\dfrac{1}{\sin x}\ }{\dfrac{\cancel{\sin x}}{\cancel{\sin x}\cos x}+\dfrac{1}{\sin x}}\\\\&{=\ }\frac{\ \dfrac{1}{\cos x}-\csc x\ }{\dfrac{1}{\sin x}+\csc x}\\\\&{=\ }\frac{\sec x-\csc x}{\sec x+\csc x}=\textsf{Ruas kiri}\end{aligned}[/tex]

Kesimpulan

∴  Telah ditunjukkan bahwa ruas kiri = ruas kanan. Oleh karena itu, identitas trigonometri di atas terbukti.

(Pakai salah satu dari cara pembuktian di atas sudah cukup.)

19. Bentuk yang senilai dengan (sec x-1) (sec x+1) adalah


Bentuk yang senilai dengan (sec x – 1) (sec x + 1) adalah tan² x. Hasil tersebut diperoleh dengan menggunakan identitas trigonometri. Berikut beberapa identitas trigonometri:

cosec A = [tex]\frac{1}{sin \: A}[/tex] sec A = [tex]\frac{1}{cos \: A}[/tex] cot A = [tex]\frac{1}{tan \: A}[/tex] tan A = [tex]\frac{sin \: A}{cos \: A}[/tex]

sin² A + cos² A = 1, maka

sin² A = 1 – cos² A cos² A = 1 – sin² A


Pembahasan

(sec x – 1) (sec x + 1)

= sec² x + sec x – sec x – 1

= sec² x – 1

= [tex]\frac{1}{cos^{2} \: x} - 1[/tex]

= [tex]\frac{1 - cos^{2} \: x}{cos^{2} \: x}[/tex]

= [tex]\frac{sin^{2} \: x}{cos^{2} \: x}[/tex]

= tan² x

36. Buktikan identitas dari persamaan tan² x / 1+sec x = sec x-1


# sin² x = 1 - cos² x

tan² x /(1 + sec x) = sec x - 1
tan² x = (sec x - 1)(sec x + 1)
tan² x = sec² x - 1
tan² x = 1/ cos² x - 1
tan² x = 1/ cos² x - (cos²x / cos² x)
tan² x = (1 - cos²x) / cos² x
tan² x = sin² x / cos² x
tan² x = tan² x ⇒ terbukti

37. 2-sec^x/sec^x= 1-2 sin^x buktikan

38. Buktikan sin² x = sec² x - 1 / sec² x.


sec²x - 1/ sec²x = tan²x . cos²x
sec²x - 1/ sec²x = sin²x/cos²x . cos²x
sec²x - 1/ sec²x = sin²x
