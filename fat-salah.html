3. 6. DICKY IS FAT. WANDA IS FAT. ANDRA IS FAT. HARIS IS FAT ->->-> Make a goodsentence by using Adjective Comparison​


Dicky is as fat as Wanda, Andra and Haris.

PREFACEQualitative Comparison:

Dibagi menjadi 3 :

1. Positive Degree:Menyatakan hal yang sebanding. (A = B)

Structure :

as + adjective + as

➡ Contoh :

Rani runs as fast as Dina. [Rani berlari secepat Dina. ]2. Comparative Degree:Menyatakan hal yang lebih besar atau lebih kecil pada dua benda. (A >/< B)

Structure :

adjective + er/ier + than

➡ Contoh :

Mila is taller than Tuti. [Mila lebih tinggi dari Tuti. ]3. Superlative Degree:Menyatakan hal yang paling diantara lebih dari dua benda. (A >>>)

Structure :

the + adjective + est/iest

➡ Contoh :

Rini is the cleverest student among us. [Rini adalah siswa terpandai di antara kita. ]EXPLICATION

Dicky is fat. Wanda is fat. Andra is fat. Haris is fat.

Karena tidak ada perbedaan di antaranya, berarti kalimat tersebut menggunakan positive degree. Sehingga tinggal disisipkan as + adjective + as dalam kalimatnya.

adjective : fatSehingga kalimatnya : Dicky is as fat as Wanda, Andra and Haris. Details:

Grade : VIII

Subject : English

Topic : Grammar

Category : 8.5

Keywords : Degree comparison, positive degree

7. efek dari virus fat​


FAT Merupakan singkatan dari File Allocation Table (FAT), jenis virus komputer satu ini bersifat merusak file pada penyimpanan tertentu. Apabila ada file kamu yang terkena virus ini, maka file akan disembunyikan oleh virus sehingga seolah-olah file tersebut menghilang. Jadi, apabila ada file kamu yang mendadak menghilang, bisa jadi penyebabnya adalah virus FAT ini.
