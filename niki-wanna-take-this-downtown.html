1. If they .... (live) downtown, we ....(visit) then this evening


Jawaban:

If they live downtown, we will visit then this evening

Penjelasan:

Conditional Sentence (If-clause di awal kalimat)

Rumus:

If + S + Verb 1, S + Will  + Verb 1

Semoga membantu

Jawaban:

If they live downtown, we will visit them this evening

Penjelasan:

Jawaban dicetak tebal dan kalimat di atas menggunakan bentuk kalimat Conditional Type-1.

Kalimat pengandaian tipe 1:

if+ subject + VI (Simple Present Tense) + Subject + will + Verb-1

atau

If+ subject + (be) present + adjective/noun + Subject + will be + adjective/noun

Contoh :

If I work hard, I will get a lot of money.

If I have a lot of money, I will buy an expensive car.

Semoga membantu ya.

26. uang nike dan niki sebesar Rp.45.000 uang nike 2/3 dari uang niki. berapa uang nike dan niki?


uang niki = x

uang nike = y

y = 2/3 x


x + y = 45.000

x + 2/3 x = 45.000

3/3 x + 2/3 x = 45.000

5/3 x = 45.000

x = 45.000 (3/5)

x = 27.000


y = 2/3 x

y = 2/3 (27.000)

y = 18.000


uang nike = y = Rp18.000

uang niki = x = Rp27.000
