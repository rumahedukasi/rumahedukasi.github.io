2. jelaskan cara mengecek ip komputer kita?​


cara mengecek ip menggunakan cmd :

1. Masuk ke command prompt pada Start Menu atau dengan cara menekan tombol Win + R, lalu ketikkan cmd kemudian tekan Enter.

2. Pada jendela command prompt, ketikkan perintah ipconfig untuk menampilkan informasi mengenai daftar koneksi yang terdapat pada komputer anda, kemudian tekan Enter.

3. Command prompt akan menampilkan daftar koneksi yang terdapat pada komputer anda beserta alamat IP (IPv4 address) nya

bearing adalah elemen mesin berfungsi untuk membatasi gerak relatif antara dua atau lebih komponen mesin. Bearing menjaga agar gerak relatif antara dua atau lebih komponen mesin tersebut selalu bergerak pada arah yang diinginkan.

15. Bagaimana caranya agar ip kita tidak sama dengan ip orang lain


Jawaban:

dengan mengubah Alamat IP secara Manual

Penjelasan:

Sebenarnya alamat IP kita ketika terhubung ke suatu jaringan akan automatis terisi (default) dan itu tidak mungkin sama dengan milik orang lain.

tapi kita bisa mengubah alamat IP kita akan tetapi memiliki resiko yaitu tabrakan IP yang terjadi karna ada 2 alamat IP dengan alamat yg sama dalam 1 jaringan.

cara mengubah alamat IP secara manual:

1. masuk ke Control Panel

2. klik Network And Internet

3. klik Open Network and Sharing Center, dan klik device atau modem pada bagian Connections (yg ada logo sinyalnya)

4. di wifi status klik Properties

5. kemudian cari internet Protocol versi 4 (IPv4) dan klik Properties

6. pada internet Protocol versi 4 (IPv4) Properties bagian general, ganti centang menjadi "use the following IP address" kemudian isi IP address, subnetmask, dan default gateway. cara mengetahui Alamat IP kita pada jaringan yaitu melalui CMD ketik ipconfig disitu keliatan Alamat IP, Subnetmask, dan default gateway kita.

19. cara menghitung ip host dari ip=60.253.117.12/16


cara menghitung first host address :
1. konversikan Ip address ke biner
2. pada host bit, jadikan semua bit = 0, kecuali yg terakhir = 1
3. konversikan kembali ke decimal

60 . 253 . 117 . 12 /16
00111100 . 11111101 . 01110101 . 00001100  ---> Ip address
                                   00000000 . 00000001  ---> host bit
sehingga = 60 . 253 . 0 . 1

cara menghitung last host address :
langkah ny sama, kecuali pada host bit, jadikan semua = 1, kecuali bit terakhir = 0

60 . 253 . 117 . 12 /16
00111100 . 11111101 . 01110101 . 00001100
                                    11111111 . 11111110
sehingga = 60 . 253 . 255 . 254

27. Bagaimana cara set ip client dan ip server?


Keduanya sama caranya.


1. Buka Control Panel.

2. Pilih Network and Sharing Center.

3. Pilih Koneksi yg digunakan sekarang (WiFi, LAN, dsb).

4. Klik kanan >> Properties.

5. Pilih Internet Protocol Version 4 (TCP/IPv4) (biarkan tercentang).

6. Pilih Properties.

7. Pilih Use the following IP address:.

8. Isi IP addressnya di kolom IP address:.

9. Subnet mask otomatis terisi, Default gateway biarkan saja.

DHCP (Dynamic Configuration Protocol) adalah layanan yang secara otomatis memberikan nomor IP berbeda dengan kita menuliskan alamat IP dengan cara manual.jadi klo pada contoh kasus diatas saya rasa tidak mungkin,dan kewenangan bisa menggunakan akses internet sperti kasus diatas adalah kewenangan admin melalui mikrotic.
kesimpulannya adalah sbb:
-jika yang diperoleh IP DHCP maka kita tidak bisa mengedit alamatnya
-untuk mengedit alamat IP bisa melakukan setting manual (addres,netmask,dan gateway untuk akses internetnya)
