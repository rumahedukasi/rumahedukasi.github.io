11. bilangan oksidasi Sn(OH)3 - toling dibantu


Bilangan oksidasi Sn dalam senyawa Sn(OH)₃ adalah +3.

Pembahasan :

Biloks Sn dalam senyawa Sn(OH)₃

Biloks H = +1  

aturan biloks :

Bilangan oksidasi hidrogen dalam senyawanya adalah +1 kecuali dalam Hidrida, atom Hidrogen biloksnya adalah -1. Contoh : Bilangan oksidasi H dalam H₂O, NH₃, dan HCl = +1, kecuali Bilangan oksidasi H dalam NaH dan CaH₂ = -1.

biloks O = -2

aturan biloks :  

Bilangan oksidasi dalam senyawa O adalah -2 kecuali dalam peroksida (-1) dan dalam senyawa biner dengan fluor (+2). Contoh : Bilangan oksidasi O dalam H₂O = -2 , kecuali bilangan oksidasi O dalam H₂O₂ dan BaO₂ = -1 , dan bilangan oksidasi O dalam OF = +2 .

biloks Sn(OH)₃ = 0

biloks Sn + (3 × biloks H) + (3 × biloks O) = 0

biloks Sn + (3 × (+1)) + (3 × (-2) = 0

biloks Sn + (+3) + (-6) = 0

biloks Sn + (-3) = 0

biloks Sn = +3

⇒ Kesimpulan, maka bilangan oksidasi Sn dalam senyawa Sn(OH)₃ adalah +3.

12. Zn(s) + MnO2(s) + H2O ï  Zn(OH)2(s)+ MnO(OH)(S) Tuliskan reaksi setaranya, biloksnya, oksidator, dan reduktornya...


Zn (s) + MnO₂ (s) + H₂O → Zn(OH)₂ (s) + Mn₂O₃ (s)

Reaksi setaranya, biloksnya, oksidator, dan reduktornya adalah

Reaksi setara :

Zn (s) + 2 MnO₂ (s) + H₂O → Zn(OH)₂ (s) + Mn₂O₃ (s)

Oksidator = mengalami reduksi = biloksnya turun = MnO₂

Reduktor = mengalami oksidasi = biloksnya naik = Zn

Pembahasan :

Zn (s) + 2 MnO₂ (s) + H₂O → Zn(OH)₂ (s) + Mn₂O₃ (s)

 |_____________________|  

 0                                           +2         (Reaksi Oksidasi/ Reduktor)  

Atom Zn pada reaksi diatas mengalami kenaikan bilangan oksidasi dari 0 berubah menjadi +2, sehingga mengalami reaksi oksidasi atau reduktor.  

Zn (s) + 2 MnO₂ (s) + H₂O → Zn(OH)₂ (s) + Mn₂O₃ (s)

                 |________________________|  

               +4                                                  +3    (Reaksi Reduksi/ Oksidator)

Atom Mn pada reaksi diatas mengalami penurunan bilangan oksidasi dari +4 berubah menjadi +3, sehingga mengalami reaksi reduksi atau oksidator.

Maka diperoleh :

Oksidator = mengalami reduksi = biloksnya turun = MnO₂

Reduktor = mengalami oksidasi = biloksnya naik = Zn

Pengertian Oksidasi dan Reduksi sebagai reaksi peningkatan dan penurunan bilangan oksidasi

Oksidasi adalah reaksi dengan peningkatan bilangan oksidasi.

Zat yang mengalami kenaikan bilangan oksidasi disebut reduktor.

Contoh :

K → K⁺ + e⁻

⇒ Unsur K mengalami kenaikan bilangan oksidasi yaitu berubah dari biloks 0 menjadi +1.

Reduksi adalah reaksi dengan penurunan bilangan oksidasi.

Zat yang mengalami penurunan bilangan oksidasi disebut oksidator.

Contoh :

Cl₂ + 2e⁻ → 2 Cl⁻  

⇒ Senyawa Cl₂ mengalami penurunan bilangan oksidasi yaitu berubah dari biloks 0 menjadi -1.

Aturan-aturan untuk menentukan Bilangan Oksidasi

• Unsur bebas mempunyai biloks =0 Contoh → H,N,Fe berturut-turut dalam H2, N2, dan Cl= 0

• Fluorin, unsur yang mempunyai biloks -1

• Golongan I (logam alkali ) biloks = +1 contoh: Li, Na, K, Rb, Cs

• Golongan II (Alkali Tanah) biloks = +2 contoh = Be, Mg, Ca, Sr, Ba

• biloks suatu unsur dalam suatu ion tunggal sama dengan muatannya Contoh: biloks Fe dalam ion Fe²+ = +2 S²- = 2-

• Biloks H umumnya= +1, kecuali dalam senyawa dengan logam maka biloks = -1 Contoh: NaH BaH2 = -1

• biloks O umunya = -2 Contoh: biloks O dalam H2O, MgO = -2 kecuali
-Dalam Fe2O = +2
-Dalam Peroksida, seperti H2O2, biloks O = -1 -Dalam Senyawa superoksida, seperti KO2 biloks O= -1/2
-Dalam senyawa terner, biloks O selalu = -2
-Dalam senyawa biner, biloks O belum tentu = +2

• biloks halogen, sperti F, Cl, I, Br, At = -1

• biloks O dalam unsur-unsur dalam suatu senyawa = 0 Contoh: H2SO4= 0

• bilols unsur-unsur dalam senyawa poliatom = muatannya contoh = SO4²- = -2

• biloks dalam senyawa ion = muatan kation dan anionya.

32. Setarakan persamaan reaksi redoks berikut : a. Fe^2+ + MnO4- ----> Fe^3+ + MnO2 b. CN- + MnO4- + H2O ----> MnO2 + CNO- + OH-


Mata pelajaran : kimia
Kelas : XII SMA
Kategori : penyetaraan reaksi Redoks
Kata kunci : reaksi Redoks, , metode biloks, metode ion-elektron (setengah reaksi)
Pembahasan :
Jawaban untuk persamaan reaksi a dan b bisa dilihat pada lampiran.

Penjelasan :
Reaksi redoks adalah reaksi yang terjadi akibat atom mengalami perubahan bilangan oksidasi. Jika bilangan oksidasi atom mengalami penurunan maka reaksinya disebut reduksi. Pada reaksi reduksi, atom melakukan penangkapan elektron. Sebaliknya, jika terjadi kenaikan bilangan oksidasi maka reaksinya disebut oksidasi. Pada reaksi oksidasi, atom melakukan pelepasan elektron.

Ada 2 cara untuk menyetarakan persamaan reaksi redoks yaitu dengan metode biloks (bilangan oksidasi) dan metode ion elektron (setengah reaksi). Penyetaraan reaksi redoks dengan menggunakan metode biloks dapat dilakukan dengan langkah - langkah sebagai berikut :
1. Menentukan biloks masing-masing atom
2. Menyamakan jumlah atom yang mengalami perubahan biloks.
3. Menentukan atom-atom yang mengalami perubahan biloks.
4. Menentukan jumlah elektron oksidasi dan reduksi lalu menyamakan elektronnya
5. Menghitung muatan kanan dan kiridan menyamakan muatan dengan menambahkan :
a. H+ di ruas yang jumlah muatannya kecil
b. OH- di ruas yang jumlah muatannya besar
6. Menambahkan H2O untuk menyamakan jumlah atom H dikedua ruas.

Tahapan pnyetaraan reaksi redoks dengan metode ion elektron atau setengah reaksi adalah :
1. Reaksi dibagi 2 terlebih dahulu menjadi setengah reduksi dan setengah oksidasi
2. Untuk reaksi reduksi, menambahkan elektron di ruas kiri sebanyak selisih muatan kanan dan kiri
Untuk reaksi oksidasi, menambahkan elektron di ruas kanan sebanyak selisih muatan kanan dan kiri
3. Menyamakan elektron oksidasi dan reduksi
4. Menambahkan H+ atau OH- untuk menyamakan muatan kanan dan kiri
5. Menambahkan H2O
a. Suasana asam, H2O ditambahkan di ruas yang kekurangan O
b. Suasana basa, H2O ditambahkan di ruas yang kelebihan O
6. Menggabungkan kedua reaksi menjadi seperti semulanya.
