6. Q.6⁴ × 4⁵ =nt: bantu like jwbn kimia sy, nanti like back sy like back​


♠ Pendahuluan

Perkalian adalah suatu menghitung biasanya digunakan dalam kehidupan sehari-hari dengan cara mengalikan suatu bilangan dengan bilangan lain,

Perkalian biasanya ditandai dengan => (×)

Penyelesaian ☁️

• 6⁴ × 4⁵

= ( 6×6×6×6 ) × ( 4×4×4×4×4 )

= ( 36×36 ) × ( 16×16×4 )

= 1.296 × ( 256×4 )

= 1.296 × 1.024

= 1.327.104
