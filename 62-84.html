17. 62, 73, 84, 95... Hitung pola ke n


Diketahui:

Barisan -> 62, 73, 84, 95

Ditanyakan:

Un

Jawab:

Dari barisan diketahui bahwa:

U1 = 62

U2 = 73

U3 = 84

U4 = 95

Karena U1 = 62, maka a = 62.

Hitung selisih tiap suku:

• U2-U1 = 73-62 = 11

• U3-U2 = 84-73 = 11

• U4-U3 = 95-84 = 11

Jadi, selisih tiap sukunya sama yaitu 11, maka b=11.

Un = a + (n-1)b

Un = 62 + (n-1)11

Un = 62 + 11n - 11

Un = 51 + 11n

28. Hasil dari 84/100 - 62/100 + 1/1.000


[tex] \frac{84}{100} - \frac{62}{100} + \frac{1}{1000} [/tex]

[tex] = \frac{84 - 62}{100} + \frac{1}{1000} [/tex]

[tex] = \frac{22}{100} + \frac{1}{1000} [/tex]

[tex] = \frac{11}{50} + \frac{1}{1000} [/tex]

[tex] = \frac{220 + 1}{1000} [/tex]

[tex] = \frac{221}{1000} [/tex]

[tex] = 0.221[/tex]

~Soal:

1. Tentukan hasil dari 84/100 - 62/100 + 1/1000 .

~Jawab:

= 84/100 - 62/100 + 1/1000

= (84 - 62)/100 + 1/1000

= 22/100 + 1/1000

= (22 × 10)/(100 × 10) + 1/1000

= 220/1000 + 1/1000

= (220 + 1)/1000

= 221/1000

≈ 0,221 ✓

~Kesimpulan:

∴ Jadi, hasil dari 84/100 - 62/100 + 1/1000 adalah 0,221.

29. Perhatikan soal berikut ini ! A. 84 B. 62 C. 52 D. 42


Jawaban:

1 Abad = 100 Tahun

1 Windu = 8 tahun

Umur Kakek = 3/5 Abad + 1/2 Windu

= (3/5 × 100) + (1/2 × 8)

= 60 + 4

= 64 Tahun

Umur Paman = 3 Windu - 2 tahun

= (3×8) -2

= 24 - 2

= 22 Tahun

Selisih umur = Umur Kakek - Umur Paman

= 64 - 22

= 42 Tahun

Jadi, selisih umur Kakek dan Paman adalah 42 Tahun
