Given that the melting ice temperature is 5⁰x and the boiling water temperature is 80⁰x, we can determine the temperature on thermometer x by setting up a proportion:

(40 - 0) / (100 - 0) = (T - 5) / (80 - 5)

Here, T represents the temperature on thermometer x that we want to find. By solving this proportion, we can determine the value of T.

Let's calculate:

(40 - 0) / (100 - 0) = (T - 5) / (80 - 5)

Now, let's solve for T:
