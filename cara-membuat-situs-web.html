14. Bagaimana cara halaman situs web dengan benar?


Mungkin yang kamu maksud adalah membuat, membuka atau menyimpan halaman situs web dengan benar. Jika salah satu dari ke 3 itu benar maka: cara halaman situs web dengan benar adalah dengan membuat struktur kode HTML (Hypertext Markup Language) yang sesuai dengan standar dari konsorsium W3C; cara membuka halaman situs web dengan benar bisa dilakukan dengan menyalin alamat situs lalu di buka pada address bar lalu klik enter, atau cukup dengan klik tautan hidup pada halaman situs; yang terakhir cara menyimpan halaman situs dengan benar bisa dilakukan dengan cara menekan tombol CTRL + S lalu pilih format yang diinginkan.

Pembahasan:

Cara membuat halaman situs web dengan benar

Meskipun pada dasarnya ada sebuah organisasi yang mengatur tentang standar website (W3C) yang ideal tapi pada prakteknya tidak ada patokan yang paling benar. Adapun standar yang harus diperhatikan adalah:

halaman web berformat HTML memiliki struktur yang sesuai/mendekati referesi dari W3Cdilengkapi CSS dan Javascript sebagai penunjang desain dan interaksi

Cara membuka halaman web dengan benar

Pada dasarnya kamu hanya perlu menyalin/copy alamat web yang hendak diakses sebelum alamat tersebut, adapun yang harus diperhatikan dari alamat web atau situsnya

nama domain. Contoh: brainly.co.iddirektori halaman yang terletak tepat setelah nama domain. Contoh, halaman ini memiliki alamat direktori /tugas/22545588

Cara menyimpan halaman situs web dengan benar

Menyimpan sebuah halaman situs web bisa dilakukan dengan 2 cara yaitu:

dengan menggunakan fitur "save as" atau CTRL+S di browser untuk megunduh html beserta file statis halaman tersebut sehingga bisa kita akses secara offlinedengan menggunakan fitur bookmark di browser atau peramban. Dengan cara tersebut kita bisa menyimpan/mengelola alamat dan halaman situs yang dianggap penting agar bisa diakses dengan mudah dilain waktu. Caranya buka halaman yang akan disimpan >> tekan tombol CTRL+D >> pilih direktori penyimpanan >> beri nama >> >> beri label/tag lalu save

19. Bagaimana cara halaman situs web dengan bener?

2. Kemudian lakukanlah perintah selanjutnya seperti mengetik alam web pada Address Bar

3. Tekanlah Enter pada keyboarad atau langsung klik tombol Go yang terdapat di sebelah address bar.

4. Tunggulah beberapa saat hingga keseluruhan tampilan selesai dan tertulis “Done” pada status bar

5. Jalankan Internet Explorer sesuai dengan fungsi menu yang pernah kita pelajari ketika mengakses internet.

6. Untuk mengakhiri Internet Explorer ada tiga cara yang dapat dilakukan, yaitu:

a. Melalui menu File, caranya : klik menu File pada jendela Internet Explorer, kemudian klik Close.

b. Klik kanan mouse pada blok judul Internet Explorer, kemudian pilih atau klik Close.

c. Klik langsung tanda silang pada balok judul Internet Explorer.
