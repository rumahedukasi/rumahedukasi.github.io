9. what is the languange feature of the text


║PROCEDURE TEXT║

Answer

The language features of the text are

1. Using the adverbial of sequence

Example : Firstly, secondly, thirdly, fourthly, after

2. Using the imperative sentence / command

Example : open the you tube......., add "ss" to the video......, choose the format......., Don't use you tube........

3. Using the simple present tense

Example :  There are many ways.........., Here are some easy ways........

4. Using the actions verb

Example : open, use, add, select, download

5. Using passive voice

Example : you will be directed .........., the video that you choose will be downloaded........

Explanation

PROCEDURE TEXTGenre teks ini menjelaskan bagaimana cara menggunakan / membuat sesuatu secara runtut.

SOCIAL FUNCTION / COMMUNICATIVE PURPOSE To explain how to make/to use something in sequential steps ( untuk menjelaskan langkah berurutan bagaimana cara membuat/menggunakan sesuatu )

GENERIC STRUCTUREGOAL

"Goal" berupa informasi judul yang ingin disampaikan kepada pembaca/penonton.

Example : How to make fried rice, how to operate the stand fan

INGREDIENTS / MATERIAL / UTENSIL

Dalam teks ini berisi informasi bahan-bahan, peralatan yang digunakan dalam mengerjakan sesuatu

STEPS

Dalam teks ini berisi informasi cara-cara ( langkah-langkah )  penggunaan/ pembuatan / pengoperasian sesuatu.

LANGUAGE FEATURE  { Unsur Kebahasaan }Penggunaan adverbial of sequence   ( terdapat identifikasi keterangan urutan )

Example : First, second, then.....

Penggunaan imperative command ( berisi kalimat perintah )

Example : cut the meat, slice the onion

Penggunaan adverb of detailed time  ( berisi informasi waktu pengolahan/penanganan )

Example : steam the potatoes for 30 minutes...

Penggunaan simple present tense  ( dikarenakan informasi yang disajikan sifatnya fakta, maka, penggunaan pola simple present tense pada kata kerja nya baik dalam kalimat verbal maupun nominal )Penggunaan action verb ( berisi kata kerja aksi )

Example : slice, put, blend

The characteristic of recount text:

Procedure Text

The characteristic of Procedure Text:

Using the Simple Present Tense (S + V1) Use imperative sentences Using the verbs action, for example: make, take, boil, cook, Using temporal conjunctions (or numbering to indicate sequence), for example: First, then, next, after that, lastUsing adverbial phrase

PEMBAHASAN:

Tujuan dari teks prosedur adalah untuk memberitahu pembaca bagaimana melakukan atau membuat sesuatu melalui urutan tindakan atau langkah-langkah.

“The purpose of procedure text is to tell the reader how to do or make something through a sequence of actions or steps.”

Generic Structure of Procedure Text  

Goal: memberikan informasi tentang maksud dan tujuan prosedur dan memprediksi suatu kesimpulan. (provides information about the purpose and objectives of the procedure and predicts a conclusion) Materials: berisi daftar materi atau bahan-bahan yang dibutuhkan untuk melakukan suatu prosedur atau langkah-langkah. (contains a list of materials or materials needed to carry out a procedure or steps) Steps: daftar urutan instruksi/aktivitas untuk mencapai tujuan dalam urutan langkah yang benar. (list the sequence of instructions / activities to reach the destination in the correct sequence of steps)

37. characteristic of the procedure text


A procedure is a text that explain how people make or so something step by step.
This text uses temporal conjunction, such as first, second, etc.
The structure of procedure text:
-aim/goal or tittle
-materials
-steps
Supporting features:
-imperative (ex. add two spoon of sugar into the glass)
-noun phrases (ex. thick layer)
-adverb of manner (ex. tightly)
