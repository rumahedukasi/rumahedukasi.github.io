```python

import numpy as np

import matplotlib.pyplot as plt

# Fungsi untuk menghitung gradient Richardson number

def richardson_number(T, z, u, v, g=9.81):

dz = np.gradient(z)

dT = np.gradient(T)

Ri = (g / T.mean()) * (dT / dz)

return Ri

# Data cuaca dan ketinggian

T = np.array([25, 24, 23, 22, 21, 20]) # Data temperatur (misalnya diambil pada beberapa tingkat di atmosfer)

z = np.array([0, 100, 200, 300, 400, 500]) # Ketinggian masing-masing tingkat

# Data kecepatan angin pada sumbu x (u) dan sumbu y (v)

u = np.array([5, 6, 7, 8, 9, 10])

v = np.array([2, 3, 4, 5, 6, 7])

# Hitung gradient Richardson number

Ri = richardson_number(T, z, u, v)

# Plot hasilnya dalam bentuk 2D

plt.plot(Ri, z)

plt.xlabel('Richardson Number')

plt.ylabel('Ketinggian (m)')

plt.title('Profil Richardson Number terhadap Ketinggian')

plt.grid(True)

plt.savefig('richardson_number_profile.png')

plt.show()
```

Planetary Boundary Layer (PBL) adalah lapisan atmosfer terdekat dengan permukaan bumi di mana gaya-gaya turbulensi bumi, seperti turbulensi dari tanah dan sirkulasi udara termal, dominan. PBL sangat penting dalam memahami polusi atmosfer dan perubahan iklim.

Untuk menentukan PBL, salah satu metode yang umum digunakan adalah analisis profil vertikal temperatur atau kelembaban. Kita dapat menggunakan data suhu atau kelembaban yang diukur di berbagai tinggi untuk memperoleh indeks atau parameter yang menunjukkan perubahan signifikan dalam profil atmosfer. Salah satu parameter yang sering digunakan adalah "Lapse Rate."

Berikut adalah contoh sederhana untuk menghitung Lapse Rate menggunakan data suhu dan tinggi di lokasi tertentu:

```python

import matplotlib.pyplot as plt

# Data contoh, Anda bisa mengganti data ini dengan data observasi Anda

altitude = [0, 100, 200, 300, 400, 500, 600, 700, 800, 900, 1000] # Tinggi dalam meter

temperature = [30, 29, 28, 26, 24, 22, 20, 18, 16, 14, 12] # Suhu dalam derajat Celsius

# Menghitung Lapse Rate

lapse_rate = [(temperature[i] - temperature[i+1]) / (altitude[i] - altitude[i+1]) for i in range(len(altitude)-1)]

# Plotting hasil dalam bentuk grafik 2D

plt.plot(lapse_rate, altitude[:-1])

plt.xlabel('Lapse Rate (°C/m)')

plt.ylabel('Altitude (m)')

plt.title('Planetary Boundary Layer Analysis')

plt.grid(True)

plt.show()

```
