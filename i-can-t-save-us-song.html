5. I can sing keroncong song Ubah ke kalimat pasif


Jawaban:

the keroncong song's can be sung by me

Kalimat Passive:

Keroncong song can be sung by me

PEMBAHASAN Active Voice

I can sing keroncong song

=> Saya bisa menyanyikan lagu keroncong

Structure Kalimat Aktif:Subject => IModal => CanVerb-1 => singObject => keroncong song

Passive Voice

Keroncong song can be sung by me

=> Lagu keroncong bisa dinyanyikan olehku

Structure kalimat Passive:Object => Keroncong songModal => canBe + Verb-3 => be + sungBy Subjek => By me

Penjelasan:

Soal ini merupakan materi Passive voice. Terdapat rumus Passive voice pada masing-masing Tenses.Soal nomor 5 menggunakan Tenses Simple Present, maka rumus passive voice nya :

Object + tobe (am, is, are) + Verb 3 + by/with + subject + adverb.

Maka..

A song is sung by us in your birthday.

Terjemahan:

Soal: kami menyanyikan sebuah lagu di ulang tahunmu.

Jawaban: sebuah lagu dinyanyikan oleh kami di ulang tahunmu.
