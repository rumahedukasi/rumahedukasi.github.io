39. Luas selimut tabung 5280 cm2 Dan tingginya 30 cm. Volume tabung tersebut dengan Q= 22/7 adalah


Bangun Ruang Sisi Lengkung

Diketahui:
L selimut tabung = 5280 cm²
t = 30 cm
π = 22/7

Ditanya:
V tabung = ...

Penyelesaian:
L selimut tabung = 2 x π x r x t
5280 = 2 x 22/7 x r x 30
5280 = 44/7 x r x 30
5280 x 7/44 = 30r
1680 = 30r
1680 : 30 = r
r = 56 cm

V tabung = π x r x r x t
= 22/7 x 56 x 56 x 30
= 295680 cm³
