9. engkap Penyelaraan Reaksi Redoks dengan Metode Bilangan Oksidasi Langkah-langkah penyetaraan reaksi redoks dengan metode bilangan oksidasi sebagai berikut Menghitung dan menuliskan bilangan oksidasi pada tiap-tiap unsur b Menentukan reaksi oksidasi dan reduksi dengan melihat unsur-unsur yang mengalami perubahan bilangan oksidasi. Menentukan ini si NO DATE 1 2 As2O3 + 2n AS H3 + 2002 2- (basa) 105 t Noz -) 12 + Noz (asam) Douras JJJJJJJ​


Jawaban:

Penyetaraan reaksi redoks dengan metode bilangan oksidasi yang dimaksud adalah:

As₂O₃ + 6Zn + 12OH⁻ → 2AsH₃ + 6ZnO₂²⁻ + 3H₂O2IO₃⁻ + 10NO₂ + 4H₂O → I₂ + 10NO₃⁻ + 8H⁺

Dasar penyetaraan redoks dengan metode biloks adalah perubahan biloks unsur yang terjadi.

Penjelasan:

Poin 1: As₂O₃ + Zn  → AsH₃ + ZnO₂²⁻ (Basa)

Pertama, tentukan atom yang mengalami perubahan biloks.

As₂O₃ + Zn  → AsH₃ + ZnO₂²⁻

+3           0       -3         +2

Kedua, setarakan atom yang mengalami perubahan biloks.

As₂O₃ + Zn  → 2AsH₃ + ZnO₂²⁻

Ketiga, tentukan naik dan turunnya perubahan biloks total, yaitu biloks dikalikan dengan jumlah unsur.

As₂O₃ + Zn  → 2AsH₃ + ZnO₂²⁻

+6           0       -6         +2

Biloks total As: +6 → -6, maka turun 12Biloks total N: 0 → +2, maka naik 2

Keempat, samakan naik dan turunnya dengan mencari KPK atau dapat dikalikan silang.

Turun: 12 × 1 = 12 Naik: 2 × 6 = 12

Reaksi menjadi: As₂O₃ + 6Zn  → 2AsH₃ + 6ZnO₂²⁻

Kelima, tentukan muatan di kiri dan kanan kemudian tentukan selisihnya.

Muatan kiri = 0Muatan kanan = 6× (-2) = -12.

Maka, selisih 12.

Keenam, tambahkan OH⁻ (basa) pada muatan besar sejumlah selisih.

Selisih 12, maka ditambahkan 12OH⁻ di ruas kiri.

Reaksi menjadi: As₂O₃ + 6Zn + 12OH⁻  → 2AsH₃ + 6ZnO₂²⁻

Terakhir, tambahkan H₂O untuk menyetarakan atom H.

Atom H ruas kiri = 12Atom H ruas kanan = 6

maka, tambahkan 3H₂O di ruas kanan.

Reaksi menjadi: As₂O₃ + 6Zn + 12OH⁻  → 2AsH₃ + 6ZnO₂²⁻ + 3H₂O.

Poin 2: IO₃⁻ + NO₂  → I₂ + NO₃⁻ (asam)

Selesaikan soal (b) dengan tahapan yang sama pada soal (a).

(i)  IO₃⁻ + NO₂  → I₂ + NO₃⁻

  +5       +4        0     +5

(ii) 2IO₃⁻ + NO₂  → I₂ + NO₃⁻

(iii) 2IO₃⁻ + NO₂  → I₂ + NO₃⁻

    +10        +4        0     +5

Biloks total I: +10 → 0, maka turun 10Biloks total N: +4 → +5, maka naik 1

(iv) Samakan naik dan turun

Turun: 10 × 1 = 10 Naik: 1 × 10 = 10

Reaksi menjadi: 2IO₃⁻ + 10NO₂  → I₂ + 10NO₃⁻

(v) muatan

Muatan kiri = 2 × (-1) = -2.Muatan kanan = 10× (-1) = -10.

Maka, selisih 8.

(vi) tambahkan H⁺ (asam) pada muatan kecil sejumlah selisih.

Selisih 8, maka ditambahkan 8H⁺ di ruas kanan.

Reaksi menjadi:  2IO₃⁻ + 10NO₂  → I₂ + 10NO₃⁻ + 8H⁺

(vii) tambahkan H₂O untuk menyetarakan atom H.

Atom H ruas kiri = 0Atom H ruas kanan = 8

maka, tambahkan 4H₂O di ruas kiri.

Reaksi menjadi: 2IO₃⁻ + 10NO₂  + 4H₂O  → I₂ + 10NO₃⁻ + 8H⁺

Jawaban:

Diketahui diameter lingkaran adalah 42cm tentukan keliling lingkaran

jawab:

Diketahui diameter (d) lingkaran adalah 42 cm. Maka keliling (K) lingkaran tersebut adalah 132 cm.

Penyelesaian Soal :

Diketahui : diameter (d) = 42 cm

Ditanya : keliling (K) ?

Jawab :

LANGKAH PERTAMA (I)

Hitung panjang jari jari (r) lingkaran dengan menggunakan cara sebagai berikut :

r = ¹/₂ × d  

Keterangan :  

r = jari - jari lingkaran  (cm)

d = diameter lingkaran  (cm)

Maka perhitungannya yaitu :

r = ¹/₂ × d

r = ¹/₂ × 42 cm

r = 21 cm

LANGKAH KEDUA (II)

Hitung keliling (K) lingkaran dengan menggunakan cara sebagai berikut :

K = 2πr  

keterangan :  

K = Keliling (cm)

π = ²²/₇ atau 3,14

r = jari - jari lingkaran (cm)

Maka perhitungannya yaitu :

K = 2πr  

K = 2 × ²²/₇ × 21 cm

K = 132 cm

∴ Kesimpulan

keliling (K) lingkaran tersebut adalah 132 cm.
