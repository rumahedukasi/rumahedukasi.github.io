Penjelasan:

Untuk menyelesaikan ekspresi ini, kita dapat menggunakan rumus trigonometri yang berhubungan dengan sudut-sudut khusus.

Pertama, kita tahu bahwa nilai dari tan 30° adalah √3/3. Kita juga tahu bahwa tan 60° adalah √3.

Kita dapat menggunakan rumus penjumlahan dan pengurangan sudut untuk menyelesaikan ekspresi ini.

Mari kita selesaikan langkah demi langkah:

Jan 50 - tan 20° = Jan 50 - tan (30° - 10°) = Jan 50 - (tan 30° * tan 10°) / (1 - tan 30° * tan 10°)

Kita tahu bahwa tan 30° = √3/3 dan tan 10° = √3 - 1/√3 + 1.

Jadi, kita dapat menggantikan nilai-nilai ini ke dalam ekspresi kita:

Jan 50 - tan 20° = Jan 50 - (√3/3 * (√3 - 1)/(√3 + 1)) / (1 - (√3/3 * (√3 - 1)/(√3 + 1)))

Kita dapat menyederhanakan ekspresi ini lebih lanjut:

Jan 50 - tan 20° = Jan 50 - (√3(√3 - 1))/((√3 + 1)(√3 + 1)) / (1 - (√3(√3 - 1))/((√3 + 1)(√3 + 1)))

Kita dapat mengalikan kedua sisi dengan (√3 + 1)(√3 + 1) untuk menyederhanakan ekspresi ini:

(√3 + 1)(√3 + 1)(Jan 50 - tan 20°) = (√3 + 1)(√3 + 1) - √3(√3 - 1)

Setelah menyederhanakan ekspresi ini, kita dapat mencari nilai dari (√3 + 1)(√3 + 1)(Jan 50 - tan 20°). Namun, ekspresi ini tidak dapat disederhanakan lebih lanjut tanpa nilai yang lebih spesifik untuk Jan 50.

26. tan 20° tan 40° tan 80° = √3​


Jawab:

Terbukti

Penjelasan dengan langkah-langkah:

[tex]\displaystyle \tan 20^\circ\tan 40^\circ\tan 80^\circ=\frac{\sin 20^\circ\sin 40^\circ\sin 80^\circ}{\cos 20^\circ\cos 40^\circ\cos 80^\circ}[/tex]

Penyebut

[tex]\begin{aligned}&\cos 20^\circ\cos 40^\circ\cos 80^\circ\\&=\frac{\sin 40^\circ}{2\sin 20^\circ}~\frac{\sin 80^\circ}{2\sin 40^\circ}~\frac{\sin 160^\circ}{2\sin 80^\circ}\\&=\frac{\sin 20^\circ}{8\sin 20^\circ}\\&=\frac{1}{8}\end{aligned}[/tex]

Pembilang

[tex]\begin{aligned}&\sin 20^\circ\sin 40^\circ\sin 80^\circ\\&=\frac{-2\sin 80^\circ\sin 40^\circ\sin 20^\circ}{-2}\\&=\frac{(\cos 120^\circ-\cos 40^\circ)\sin 20^\circ}{-2}\\&=\frac{-\frac{1}{2}\sin 20^\circ-\cos 40^\circ\sin 20^\circ}{-2}\\&=\frac{\sin 20^\circ+2\cos 40^\circ\sin 20^\circ}{4}\\&=\frac{\sin 20^\circ+\sin 60^\circ-\sin 20^\circ}{4}\\&=\frac{\sqrt{3}}{8}\end{aligned}[/tex]

Jadi

[tex]\begin{aligned}\tan 20^\circ\tan 40^\circ\tan 80^\circ&=\frac{\frac{\sqrt{3}}{8}}{\frac{1}{8}}\\&=\sqrt{3}\end{aligned}[/tex]
